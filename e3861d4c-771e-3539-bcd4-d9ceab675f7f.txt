　　第219章
<br/>　　屈九江被怼得鼻青脸肿，满面涨红，站在原地尴尬的笑了两声。
<br/>　　“此人才疏学浅，大概是花大钱从某位酸儒处购得一首诗，想着在今日哗众取宠。怕他也没想到会被我拆穿，于是便恼羞成怒，口出恶言，呵呵，实在是难登大雅之堂。”
<br/>　　曲九江干笑两声，发现无人应和，扭过头，见大家目光都看着他，一副『你不觉得自己是个傻叉么』的眼神。
<br/>　　他刚才的眼神明显不是口出恶言，而是想抽你呐。
<br/>　　百里胜试探着问：“你为何确定他这首诗不是自己做的？”
<br/>　　“诸多典故我闻所未闻，被我一问，便露出了破绽。”
<br/>　　“汝闻所未闻，见说未见，就证明这世上没有出现过么？”
<br/>　　“呃……”
<br/>　　“难道那黄须儿就不能是他家附近的一个壮汉，那贺兰山是云水关附近一座小山，云水关地处偏僻，我等自然闻所未闻，见所未见。”
<br/>　　“呃……”
<br/>　　曲九江再次怔住，猛然发现被当成傻叉的好像是自己。
<br/>　　……
<br/>　　程大雷、徐神机、徐灵儿快步离开王府，既没有回百里胜为他安排的宅子，也没有回地下城。
<br/>　　“军师，身后可有人跟踪？”
<br/>　　“没有。”徐神机道：“大当家，现在他们好像没有理由跟踪我们。”
<br/>　　“呃……好吧，你说得也有道理。”
<br/>　　程大雷来到城西一座偏僻的大宅子中，宅内没有仆人丫鬟，门口挂着两个字——程宅。
<br/>　　程大雷刚回来就把身上的官服和大胡子扯了，洗把脸换回本来模样。
<br/>　　“做事啦，做事啦，开始做事啦！”
<br/>　　程大雷在院子内点燃一支烟花，烟花直冲天际，在黑暗中炸开。
<br/>　　前门外一个推小车卖脂粉的小贩，正与两个姑娘讨价还价。忽然一怔，脸上的笑容消失，丢掉小车向这边赶来。
<br/>　　“喂，你这个人，东西不卖了？”买脂粉的姑娘喊道。
<br/>　　“不卖了，不卖了，东西送给你们了。”
<br/>　　茶楼内，一说书先生口若悬河，滔滔不绝，只唬得楼中茶客目不转睛，侧耳倾听。
<br/>　　看到窗外的烟花，他表情瞬间顿住，大步来到窗口，从二楼一跃而下。
<br/>　　“喂，喂，还没说完呢，怎么就走了，你……”
<br/>　　茶楼内的茶客愣住，没想到这个平日瘦瘦弱弱的说书先生，竟有如此了得的轻身功夫。
<br/>　　某街上睡倒的一个乞丐，青楼内喝花酒的花花公子，某家赌场看场子的壮汉……黑暗的城市中，仿若踩到一颗地雷，一条一直沉睡的神经苏醒。
<br/>　　南城，北城，东城这三地有分别有三个人，他们看着夜空中绽放的烟花，几乎是同时皱眉。
<br/>　　“西城那个家伙要干嘛。”
<br/>　　程宅。
<br/>　　程大雷搬了一张太师椅坐在屋檐下，着青衫，面无表情的歪在椅子上。
<br/>　　秦蛮是和阿喜一起到的，秦蛮道：“大当家，出事了？”
<br/>　　“那个人要来了。”
<br/>　　秦蛮一怔，瞬间明白程大雷话中意思，不动声色立在程大雷身后。
<br/>　　不多时，阿苦也到了。
<br/>　　“最近地下城一切都正常，并没有大事发生。”
<br/>　　“嗯，你先站在一边等着，等人齐了我有话和你们说。”
<br/>　　徐神机立在程大雷身后，只看到一个个人从外面进来，他们也不是都走大门，有人从墙外跳进来，有些是从狗洞爬进来。身上的衣服也各有不同，有人衣冠楚楚像个读书人，有人短衣打扮，像个做工的苦力，也有人一身褴褛，身上散发着臭味……
<br/>　　而这些人也并不是互相都认识，有些互相看到，心中发出一声惊呼。
<br/>　　“啊！原来你也是。”
<br/>　　“现在我点几个人，这些人可以离开了，这次用不着你们。”
<br/>　　程大雷开口一一数出几个名字，有些被叫到的人黯然离开，而有些则大吼着：“为啥用不着我，那个谁也在，我不比他强么。”
<br/>　　“滚。”程大雷惜字如金。
<br/>　　长久的人设发挥作用，这些人一一离开，最后院子里剩下的在八十人左右。
<br/>　　飞天蛤蟆白元飞，自幼被老猿长大，飞檐走壁，如履平地，擅长穿门入户，盗人家中财宝。
<br/>　　阿喜，曾经在军中任职，天生力大无穷，又性格暴躁，因打死长官的儿子被下了大狱，是程大雷将他从狱中救出。
<br/>　　阿苦，原先是账房先生，擅长心算，不论多复杂的账目看一眼便能得出最后数目。
<br/>　　哑巴郑三，是个杀手，除了程大雷外，无人知道他的来历，据听闻此人曾在土里被埋了三天三夜，只等目标经过时，发出必杀一剑。
<br/>　　六只耳邱鹤，包打听，此人在茶楼中说书，一方面收集各方面情报，据说长安城发生的大小事情没有他不知道的。
<br/>　　癞皮狗杜混，乞丐……
<br/>　　盗墓贼王大拿，赌徒乌鸦，花花公子钱若水……在程大雷面前立着的诸人每个人都各有本事，一直潜伏在城内，各自做各自的事，如今因程大雷一支烟花令，同时在这里汇聚。
<br/>　　“邱鹤，我要你在城内打听一个人，关于他的吃喝拉撒，我都要知道。”
<br/>　　“大哥要打听谁？”
<br/>　　“帝国战神杨龙停。”
<br/>　　“是。”
<br/>　　“飞天蛤蟆，我要你夜里潜入兵部，盗得他们的行事簿。”
<br/>　　杨龙停既然来，帝国肯定会操持着接待，在何处住宿，由什么人负责保护……这些若不是兵部负责，便是礼部负责。
<br/>　　程大雷一一发出号令，院子内各怀本事的江湖豪客有人得令离开，有些则仍留在院子中。
<br/>　　如今已可确定，杨龙停将会在三日后来到长安城，程大雷撒出人手去打听他的消息。各方行动开来，关于杨龙停的情报在第二天早晨便摆在程大雷的案前。
<br/>　　杨龙停这次随行带一千幽州兵，据说很多都是在战争中立下军功的，这次进京是为了等朝廷封赏。
<br/>　　一千人……程大雷顿了顿，看来在路上动手的机会也没有了，要动手，只能在杨龙停进入长安后。
<br/>　　三日后，杨龙停来到长安城，帝国明帝陛下亲自接待，同时，那一天也是武举开科的日子。
<br/>　　而程大雷一张大网已经铺开，只等着杨龙停现身京城时，面对面向他讨个公道。

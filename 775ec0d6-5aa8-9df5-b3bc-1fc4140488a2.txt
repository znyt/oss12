　　“先去b餐厅，通过餐厅出入口，可以快速到达主机房。”卡普兰一边摆弄电脑一边道。

<br/>　　“你确定这里是餐厅。”詹姆斯看着眼前硕大房间内的几个集装箱，扭头问卡普兰。

<br/>　　“嘿先生，你怎么了，有什么我可以帮你的吗？”雷恩走向对方。

<br/>　　锵！

<br/>　　“雷恩女士，请不要轻举妄动，不然我保证这根钢钉会插进他的脑袋里！”苏重藏在突击队员身后，挡住雷恩射击视线淡淡道。
<br/>　　“与其拿枪指着我，不如去指着你身后的活死人。”
<br/>　　雷恩察觉身后异常猛然回头，就看到刚才那个人，已经离她只有三米远。对方脸上腐烂的伤口，浑浊的眼球，滚滚恶臭让雷恩头皮发麻。
<br/>　　“嘿先生，请停下，立刻停下！”她觉出不对劲。
<br/>　　苏重脚尖一挑，突击队员掉在地上的突击步枪瞬间飞入苏重手心。不等雷恩反应，苏重就一枪打在那个踉跄男人心脏。
<br/>　　“你都干了什么！”雷恩大怒，转头就要对苏重开枪。
<br/>　　“闭嘴！回头看一看你在确定到枪口到底指向谁！”苏重一声冷喝！
<br/>　　雷恩下意识回头，就看到本应该倒地死去的家伙，不仅没死，反而张开腐烂的大嘴，露出漆黑舌头，快步向她冲来。好似迫不及待的要啃她一块肉一般！
<br/>　　砰！
<br/>　　一声枪响从身后传来，对方脑门上突兀开了个大洞，噗通一声倒在地上。近在咫尺，一股腐烂恶臭猛然传来。雷恩惊魂未定：“这到底是怎么回事？”
<br/>　　苏重放低枪口，推开身前突击队员，知道两人暂时不会攻击他。手里拿着枪，快步走到密码箱前。
<br/>　　卡普兰开锁的时候，苏重顺便记下了密码，输入密码快速打开箱子，迫不及待拿出一管绿色药剂。

<br/>　　不知道是不是错觉，他感觉身体轻松了很多，可能是血清在起作用。但暂时还不确定。

四目相对，夏夕绾看着陆寒霆，陆寒霆也看着夏夕绾。
<br/>
<br/>        夏夕绾穿着医院里的病号服，蓝白条的病号服将她纤柔的身体衬的很娇小，清纯的乌发随意的披散肩头，露出巴掌大的绝色小脸，因为额头受了伤，所以现在贴了一块创口贴，她比平日里多了几分柔弱，看着楚楚动人。
<br/>
<br/>        现在她一双澄亮而黑漉的翦瞳落在他的俊脸上，她的眸子里像是蓄了一汪盈盈的春水，让人看一眼就会沉溺下去。
<br/>
<br/>        陆寒霆脚步一顿，停了下来。
<br/>
<br/>        厉嫣然察觉到了陆寒霆的异常，她顺着陆寒霆的目光往前看去，也看到了夏夕绾。
<br/>
<br/>        厉嫣然拧了一下眉，对夏夕绾满满的厌恶，本来她被几个男人给带走了，她就是看戏的，她哪里知道陆寒霆竟然救了夏夕绾。
<br/>
<br/>        陆寒霆将油门踩到底，用劳斯莱斯幻影撞上了那辆面包车，他的右臂受伤太严重，几乎是废了。
<br/>
<br/>        陆寒霆为了救夏夕绾废了自己的一条胳膊！
<br/>
<br/>        厉嫣然从来没有将夏夕绾放在眼里过，她以为陆寒霆也就是被夏夕绾的美色给迷惑住了，但是直到这一刻她才有了深深的危机感，陆寒霆对夏夕绾的迷恋比她想象的深得多。
<br/>
<br/>        厉嫣然迅速勾起了红唇，她看着夏夕绾笑道，“夏小姐，你没事了吧，那一天你失踪真的把我们都吓坏了。”
<br/>
<br/>        夏夕绾从陆寒霆的俊脸上收回了自己的目光，然后看向厉嫣然，她淡淡道，“我很好，多谢关心。”
<br/>
<br/>        “那就好，不过最近外面谣言四起，都说你被那几个黑衣男人带上面包车发生了一些…不可描述的事情…”厉嫣然故意提起了这个话题。
<br/>
<br/>        “厉小姐！”这时蓝烟回来了，她迅速走到夏夕绾的身边，将夏夕绾护到自己的身后，“厉小姐，同为女孩子，你为什么故意在我家绾绾面前提起这么不愉快的话题，都说厉家的千金人美心善，可是我怎么觉得你嘴巴这么贱呢，这就是你们厉家的教养？”
<br/>
<br/>        蓝烟可是江湖女子，吵架都不带输的，现在她就像是一只护犊的母鸡，毫不留情面的啄了厉嫣然一口。
<br/>
<br/>        厉嫣然长这么大还没有被人骂过嘴贱，更何况是在陆寒霆的面前，她漂亮的小脸当即羞愤的通红，“你！”
<br/>
<br/>        “我怎么了？我们夏家可不怕你们厉家！”蓝烟叉腰道。
<br/>
<br/>        厉嫣然气的七窍冒烟，帝都的四大豪门和夏家之间的确是井水不犯河水，所以蓝烟很硬气，厉嫣然又碍于自己千金名媛的身份不能回骂过去，所以她抬眸，求助的看向身边的陆寒霆，“寒霆哥哥，她们都欺负我！”
<br/>
<br/>        蓝烟看向陆寒霆，其实她和陆寒霆毫无交集，直到他救了夏夕绾，将夏夕绾紧紧搂在怀里。
<br/>
<br/>        蓝烟现在还记得那一幕，所有人都在扒他的手指，想要将他和夏夕绾分开，但是他牢牢的抱着夏夕绾，力道大的好像已经将她嵌入了自己的骨血里。
<br/>
<br/>        蓝烟从来没想过陆家的这位嫡长子，现在帝都第一商界贵胄，竟然会跟她家绾绾发生什么故事。

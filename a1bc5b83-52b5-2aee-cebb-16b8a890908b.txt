　　“师尊，大师兄……我一定会为你们报仇的，七大宗门！天九域！”
<br/>　　楚风眠沉声道！
<br/>　　从今日之后，楚风眠又将多一个仇敌，天九域。
<br/>　　不管这天九域，到底是何方神圣，楚风眠也早晚一日，要毁了它。
<br/>　　“实力，我的实力还不够……”
<br/>　　楚风眠眼神之中露出几分血色，他已经踏入了古帝，就已经足够为师报仇了。
<br/>　　可现在，楚风眠才明白过来，也许在大罗天星域，楚风眠足以称得上强者了。
<br/>　　但是在真正的强者面前，在天九域的面前，楚风眠依然不算什么。
<br/>　　身为仙帝的剑道之主，都是死在了天九域跟七大宗门的手中，楚风眠这点实力，算得了什么。
<br/>　　至天境，仙帝！
<br/>　　楚风眠眼神之中露出几分坚毅，成就天人，的确是极端困难，但是楚风眠有自信，他定然会完全这一切。
<br/>　　超越剑道之主，为师门报仇。
<br/>　　楚风眠目光看向宫殿之内，那已经是失去了灵力的凌天神剑。
<br/>　　这是曾经剑凌的佩剑，也是剑凌唯一留下之物。
<br/>　　楚风眠看了一眼，转过头去，离开了宫殿，临走之时，在这宫殿之上，又是刻画了新的极道剑印。
<br/>　　这宫殿，也就随着剑凌一起，埋葬于此了。
<br/>　　除了楚风眠，再也无人可以打开，足以让剑凌安息了。
<br/>　　“先建立剑道门！然后去归墟天，寻找机会，踏入帝尊境！”
<br/>　　楚风眠心中下了决定。
<br/>　　楚风眠的身份，已经是暴露出去了，七大宗门的人，已经是知道楚风眠是剑道门的弟子。
<br/>　　按照剑凌的话，天九域跟七大宗门之间，有着联系。
<br/>　　如果天九域之中的强者真的到来，楚风眠的处境也会变得无比危险，提升实力，才是关键。
<br/>　　一旦是踏入帝尊境，楚风眠有自信可以媲美九劫古帝，甚至是达到接近半步天人境的地步。
<br/>　　到时候纵然是有着危险到来，楚风眠也有着自保之力。
<br/>　　楚风眠在地底之中，并没有离开，而是开始闭眼凝神，修炼起来。
<br/>　　之前吞噬了陈少主的精血，楚风眠也只是简单的吞噬了。
<br/>　　如今正好是需要时间，来巩固一下境界，所以楚风眠也没有着急离开，而是开始修炼起来。
<br/>　　一日的时间，楚风眠将陈少主血脉之中的吞天兽血脉，几乎是全部融入到了楚风眠的血脉之中。
<br/>　　随着得到这吞天兽血脉，最大的好处便是楚风眠施展吞天秘术的时候，威力足足是增强了六七倍。
<br/>　　原来吞天秘术，也是多半被楚风眠用来防止那些古帝逃走，吞噬精血之用的。
<br/>　　在对敌上，却是用处不大。
<br/>　　而现在楚风眠的吞天秘术威力增强了如此之多，纵然是八劫古帝，被吞天秘术笼罩，想要逃出生天，都无比困难。
<br/>　　而且先天神兽，几乎都是天下间最为强横的肉身，得到了这一缕血脉后，楚风眠感觉到他的肉身，也在不断的蜕变这。
<br/>　　已经是越发的向着不朽真身的方向发展，血肉晶体之上的血色，已经是快要淡到看不见了，彻底转变这。
<br/>　　但是现在，楚风眠的力量也彻底达到了瓶颈。
<br/>　　虽然楚风眠的肉身在蜕变，但是他的力量，却没有任何的变化，准帝的瓶颈。
<br/>　　只有踏入到帝尊境，楚风眠的实力才可以进一步增长。
<br/>　　“一日时间到了。”
<br/>　　楚风眠睁开眼，距离修炼以前，已经过了一日。
<br/>　　楚风眠将在这地底之中，一切的灵力，全部抹除之后，旋即化为了一道遁光，离开了地底。
<br/>　　当楚风眠回到北邙学院的时候，北邙学院已经是发生了翻天覆地一般的变化。
<br/>　　院长被杀，诸位老祖的死，几乎是让北邙学院震动。
<br/>　　尤其是北邙学院，即将改名的消息，也是传遍了整个北邙学院。
<br/>　　不知道是有多少弟子不愿意，甚至是许多北邙学院的强者，都打算反抗，但是在邓灵，武皇的威严之下，还是都被压了下去。
<br/>　　如今在北邙学院最深处的宫殿之中，几乎是北邙学院所有的强者，弟子之中的精英，都聚集于此。
<br/>　　在宫殿的正中心，邓灵跟武皇二人，正冰冷的看着下方。
<br/>　　突然之间，一道身影进入到了宫殿之中，旋即无数目光，望了过来。
<br/>　　“拜见门主。”
<br/>　　武皇跟邓灵，看到楚风眠，同时跪下，开口道。
<br/>　　现在楚风眠的身份，已经是剑道门的门主了，他们二人，自然如此称呼。
<br/>　　“这，就是那剑道门的门主？”
<br/>　　“据说我们北邙学院，要加入剑道门？”
<br/>　　“我们北邙学院，传承万年，在北域所向无敌，居然要加入这什么剑道门？”
<br/>　　“院长，诸位老祖都死了，这其中肯定有什么不对！”
<br/>　　无数北邙学院的强者，弟子，都是目光闪烁的看着楚风眠，一个个心中各有想法，甚至是议论纷纷。
<br/>　　这一幕让邓灵不由的皱了皱眉，楚风眠可是下命令让他处理好北邙学院的事的。

<br/>　　“这就是我们剑道门的门主，你们还不拜见门主？”
<br/>　　“拜见门主！”
<br/>　　大部分北邙学院的弟子，听到邓灵开口，虽然脸色有些不愿意，可还是跪在了地上，高声喊道。

<br/>　　这少年一脸女相，看起来无比诡异，声音也是无比尖锐。

<br/>　　听到他开口，不少北邙学院弟子都是跟随他喊了起来。
<br/>　　“我们北邙学院，不会加入什么剑道门！”
<br/>　　“不错！”
<br/>　　“这剑道门是什么东西，也配让我们北邙学院加入？”

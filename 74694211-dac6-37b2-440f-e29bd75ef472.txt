　　　　“轰隆隆！”
<br/>　　天空在泣血，乌云在暴动，血雨倾盆，狂风怒吼，雷霆爆闪。
<br/>　　面对妖族的屠戮，人族决定强势反击，向妖族发出战书，决意与妖族不死不休。这不仅震动了无数洪荒大神通者，更令妖族为之愤怒不已！
<br/>　　“嗡！”
<br/>　　妖帝帝俊、东皇太一一声令下，无数妖族高手齐齐出手，朝着人族祖地，首阳山追杀而来，煞气惊天，满身愤怒，他们的意图毫不掩饰，再明显不过了。
<br/>　　“人族，当灭！”
<br/>　　他们要让这个后天造化所生，敢于挑衅自己妖族的蝼蚁，这个卑微的种族，从洪荒之中消失。在帝俊、太一看来，妖族被这个后天种族挑衅，不仅丢了自己的面子，更是妖族的奇耻大辱，只有用鲜血与杀戮，才能宣泄心中的愤怒与杀机。
<br/>　　耻辱唯有用人族的鲜血来洗刷，他们已经顾及不到其他人了，此刻唯一的想法便是，灭了人族！
<br/>　　洪荒在颤栗，众生都在恐惧，全都感受到一种大难临头、机将灭世般的气息，无不颤栗。
<br/>　　血难来临，谁都敢变不了，圣人高高子在上，人族与妖族在他们眼中，都是一样的，谁生谁死，与他们无关。
<br/>　　无数人族在大哭，嘶吼声震动苍穹。
<br/>　　“人族的先贤们，请你们出手吧，这人间又将饱经磨难了，谁能救人族与水火？谁能免人族于血难？！呜呜……”
<br/>　　“人族已经是生灵涂炭了，无上的人族先圣，求你们出手吧！
<br/>　　“呜呜……武祖玄天何在？燧皇何在？巢皇何在？衣皇何在？我等愿为你祈祷，请你们出手吧，灭了眼前这些屠夫！”
<br/>　　众生的恸哭声惊天动地，似乎都传到了整个天地间，这是一种精神的升华感应。
<br/>　　人族祖地，首阳山。
<br/>　　所有老弱病残都被转移到安全的地方，有人族高手保护，而人祖殿方圆千万里的原野上，已有无数人族强者在守护，层层列阵，不让任何敌人靠近这里。
<br/>　　“轰隆隆！”
<br/>　　黑云压境，滚滚滚妖气携带着滔天之威，无数妖族妖兵、妖王、妖神、妖圣在妖帝帝俊与东皇太一的带领之下，来到首阳山。
<br/>　　帝俊太一身穿金黄色帝袍，高高在上，俯视着人族看守在外的高手，目光之中满是鄙夷、不屑，好似在看蝼蚁！
<br/>　　“哼！”
<br/>　　东皇太一冷哼道：“人族就这点高手，还想与妖族开战，真是笑话！”
<br/>　　“人族注定为妖族血食！”
<br/>　　在他们看来，这些人族中最厉害的就是洞天武尊境，相当于太乙金仙，在洪荒之中可能也算一个小高手了，但对于妖族来说，还远远不够看。
<br/>　　“……”
<br/>　　人族没有说话，他们眼中尽是刻骨的仇恨，恨不得将他们生生吞噬一般。
<br/>　　“白泽，你带领百万妖兵，将这些人族灭掉，要速战速决！”东皇太一眼中带着冷酷残忍的笑容，好似魔鬼，对着身边的妖神白泽淡淡道。
<br/>　　“是，陛下！”
<br/>　　白泽一声大吼，面上露出狰狞之色，对着身后的妖族大军大叫道：“给我杀！杀光这些卑微的人族！”
<br/>　　“吼！”
<br/>　　众妖响应，顿时一拥而上，与面前的人族无数大战起来，血流成河，无数人族武者死在妖族的先天神通下，同样，妖族也没有占到什么便宜，人族武者的凶狠程度超乎众人意料。无数妖族妖兵妖王，死在人族的疯狂之下。
<br/>　　人族的武道十分霸道，一样能够修炼出各种神通，更有洞天吞噬，无尽妖族被洞天的强大吞噬性吞噬一空，武道神通挥手即来，妖族亦损失惨重。同等级之下，妖族不是人族对手，若不是妖族仗着数量巨大，还可能落入下风。
<br/>　　“杀啊！你们这些妖孽，给我死！”
<br/>　　“老子砍死你！妖崽子！”
<br/>　　“杀啊！我跟你拼了！”
<br/>　　“杀杀杀！”
<br/>　　“……”
<br/>　　一个个人族武者双目赤红，这些年积累的无尽仇恨，在这一刻爆发出来，面目狰狞，发出野兽般的嘶吼声，带着一股滔天恨意，杀向妖族。即便战死，也要拖着妖族一起死。前一个倒下，后一个继续扑来，前仆后继，不畏生死！
<br/>　　你妖族再强又如何？我一样砍死你！
<br/>　　我什么都怕，你是不怕死！来啊，拼死，谁怕谁？
<br/>　　人族不可辱，你妖族算什么东西！
<br/>　　人生自古谁无死，我以我血荐轩辕！
<br/>　　生亦何欢，死亦何苦？我今天就要为人族战死，那有如何？我死得其所！
<br/>　　誓要掌灭群邪，剑荡群妖，杀尽敌寇，败尽妖邪，虽九死其犹未悔！
<br/>　　身可残，血可流，就是人族不能辱！
<br/>　　不死不休！
<br/>　　“轰隆隆！”
<br/>　　“杀杀杀杀杀杀！”
<br/>　　人妖两族大战，汇聚了无数种族大能的注视，看着眼前这一幕，许多大神通者们震惊了，这是多么强的一种意志啊，这种不畏生死，誓要守护人族的精神，震撼了无数修士。
<br/>　　看到这一幕，大多数修士都沉默了，扣心自问，他们面对如此，也做不到比这更好的。同时，许多人心中升起一股悲凉之意，今天是人族如此，谁知他日会不会轮到自己？
<br/>　　若是自己的种族面对如此情况，那又会如何？
<br/>　　这一刻，他们沉默了。
<br/>　　人族武者不断陨落，但妖族的妖兵陨落更多，比人族多了不止一倍、两倍，人族凶残，人族的武道，人族的霸道，震撼了所有人。
<br/>　　“可惜了，人族之中没有出现大罗金仙，人族要被灭了。”
<br/>　　在众人看来，人族大罗金仙都没有，妖族只要大罗金仙的妖王妖神一出手，人族还能有反抗之力？
<br/>　　虽然这次兵出人族，帝俊、太一并没有带太多的妖族高手，大多数强者都在天庭镇守，防备着巫族的偷袭，但也有数百大罗金仙妖王跟随，更有数十大罗圆满的妖神带队，最重要的是，有帝俊与太一这两位准圣后期的强者。
<br/>　　“人族，可惜了。哎！”
<br/>　　“人族潜力还是很大的，若是不得罪妖族，迟早会崛起，可惜……”
<br/>　　“是啊，人族被灭，已是定局。”
<br/>　　“……”
<br/>　　一些关注此战的修士，暗中传音，议论纷纷，他们心中充满感慨，在这洪荒之中，被灭族是常有之事，可是今天人族的表现，却是震惊了所有人。
<br/>　　让他们心中升起一股凄凉之感，即是对人族，也是对自己。
<br/>　　问天地间谁能永恒，谁能不朽？
<br/>　　今日是人族被灭，他日是其他种族被杀，天地间真的存在永立不倒的势力吗？
<br/>　　妖族行吗？巫族可以吗？这一刻，他们变得不确定了，因为，他们想起了远古时候的龙、凤、麒麟三族的结局。
<br/>　　今日这一幕，不就如同当年的那一幕吗？

　　眼前这鲜血魔尊的出现，也证实了楚风眠的猜测。
<br/>　　这其中的武者，果然就是曾经进入到归墟天中，结果没有离开的武者，他们都一直是留在了归墟天之中。
<br/>　　“没想到你居然还知道我的名声，正好，看你是北邙学院的弟子，北邙学院还敢跟我们九魔城作对？速速放了我，不然等我九魔城的强者来到，就是你灭亡的时候！”
<br/>　　鲜血魔尊大笑着开口道，语气狂妄到了极点。
<br/>　　“你们九魔城，还有多少弟子，在这归墟天中？还有这里，到底是一处什么地方？”
<br/>　　楚风眠并没有理会这鲜血魔尊的话，而是直接开口询问道。
<br/>　　“怎么，小子，你还想要审问我？做梦，这里是什么地方你都不知道，还敢踏足在这里？等着吧，我们九魔城的强者，很快便是就会到来了！”
<br/>　　鲜血魔尊冷哼道。
<br/>　　“哼！看来你是不死心，现在不想说？我会有办法让你说的！”
<br/>　　楚风眠眼神冰冷，从他的五根手指上，突然凝聚出了一道法阵，突然之间冲这鲜血魔尊的脑袋上，猛然打了过去。
<br/>　　在这鲜血魔尊的额头上，便是一座黑色转轮，显现而出。
<br/>　　这黑色转轮刻在这鲜血魔尊的额头上，他的脸色，似乎是有些不甘，在挣扎。
<br/>　　可是在楚风眠的灵力压制之下，这鲜血魔尊的任何挣扎，都没有任何意义，顿时下一刻，他的脸色便是化为了极为恭敬的神色。
<br/>　　诸天生死印！
<br/>　　如今这诸天生死印，已经是刻入到了鲜血魔尊的内心之中，令他彻底的臣服了楚风眠。
<br/>　　本来楚风眠是没有打算留着鲜血魔尊一命的，但是如今他来到了这处空间之中，还对其中，一无所知。
<br/>　　留下这鲜血魔尊一条命，正好可以了解一下，这空间，到底是一处什么地方。
<br/>　　“鲜血魔尊，你进入到这空间中多久了，这其中到底是什么地方，隐藏这什么秘密？”
<br/>　　楚风眠看着鲜血魔尊平静的开口道。
<br/>　　这空间中，到底是隐藏这什么秘密，足以是让这么多的武者一直是在其中等待。
<br/>　　楚风眠刚刚进入其中，只感觉这空间中，是遍地谜团，至于其他，却是一概不知，云里雾里的。
<br/>　　“禀告少主，这空间，名为天羽神域，是曾经云羽天宫的一处圣地，属下通过风雷谷，进入其中已经是长达八年的时间。”
<br/>　　鲜血魔尊恭敬的讲解道。
<br/>　　“至于这天羽神域，到底隐藏这什么秘密，属下也不清楚，只是这天空之上的云羽天宫，经常会落下一些奇遇，从天降下，我们在此，也都是在探索奇遇。”
<br/>　　“这八年时间，你都一直待在这里？”
<br/>　　听到这里，楚风眠有些吃惊的说道。
<br/>　　归墟天中，宝藏众多，既然是这云羽天宫的宝藏，难以得到，那应该早就要放弃的才对。
<br/>　　怎么会在这里，待了八年之久，可以说这鲜血魔尊的大半时间，都是待在了这空间之中。
<br/>　　鲜血魔尊开口回答道。
<br/>　　“是，其实属下并非是不想离开，而是根本无法离开，这风雷谷，就在天际之上，跟云羽天宫一样，根本无法踏足其中。”
<br/>　　“无法离开？”
<br/>　　楚风眠抬起头，这才注意到，这风雷谷，如今也是位于天空之上，凌空漂浮这。
<br/>　　在这处空间之中，根本无法催动遁光，甚至是飞行。
<br/>　　在外面看起来算不得是什么难度，但是在这里，想要在回到风雷谷中，却是难如登天。
<br/>　　如今这些武者，全部聚集在这了这里，竟然不是为的什么，而是根本无法离开？
<br/>　　不够在风雷谷中，楚风眠所碰到的那中年男子，又是如何离开的？
<br/>　　“难道是所有人，都没有办法从中离开？”
<br/>　　楚风眠出声追问道。
<br/>　　那中年男子，显然是回到了风雷谷中，既然如此，应该是有着可以离开的办法才对。
<br/>　　“有，在这天羽神域之中，有着曾经云羽天宫遗留下来的玉符，只要是催动玉符的力量，就可以打开空间玄门，前去风雷谷。”
<br/>　　鲜血魔尊开口道。
<br/>　　“不过像是这等玉符，在整个天羽神域中，都不超过十个，都是被强者把控，像是我们九魔城武者中，就有一人拥有。”
<br/>　　玉符，看来想要离开这里，必须要夺取其中一个玉符才行，不然的话，也会跟这鲜血魔尊一样，被困在这天羽神域之中来。
<br/>　　楚风眠心中暗自想到。
<br/>　　他可没有打算在这归墟天中待上好几年的时间，在这一个月后，他必须要离开。
<br/>　　如今看来最为重要的事情，便是夺取这其中一枚玉符来，也足以是让楚风眠，有着一条退路。
<br/>　　“九魔城弟子，有多少人，最为强大的人是谁？在这天羽神域之中，还有这天羽神域中，都有着那些势力的存在？有那些强者？”
<br/>　　楚风眠脸色露出几分凝重，开口道。
<br/>　　这才是楚风眠最为关心的问题，他也要急需弄清楚，这天羽神域之中的情况。
<br/>　　起码想要夺取云羽天宫的宝藏，这些人，是真正的竞争对手。
<br/>　　“我们九魔城，在这天羽神域之中，一共有着十七人，都是留在归墟天中的武者，其中最为强大的一位，名为锁魂魔尊，是三十年前，进入到归墟天的武者，他的实力，已经是只差一步，就足以超越生死，达到碎命之境。”
<br/>　　“除了我们九魔城外，剩下七大宗门，也都在天羽神域中，有着不少弟子的存在，大约跟我们九魔城伯仲之间，其中的最强者，都是只差一步，超越生死的人物。”
<br/>　　鲜血魔尊回答道。
<br/>　　“没有超越生死的人物？”
<br/>　　楚风眠追问道。
<br/>　　只差一步超越生死，跟真正的碎命之境，还是有着极大的差距的。
<br/>　　起码在强大的生死境武者，楚风眠都有着一战的资本，但是若是有着碎命之境的武者，楚风眠就必须要想好退路了。
<br/>　　“目前没有，不过天剑宗的剑祖荒，圣王宗的鹿战神跟荒古门的荒圣君，都只差一步，随时可能超越生死。”
<br/>　　鲜血魔尊开口道。
<br/>　　“在这天羽神域之中，如今便是以他们三人为最强，也占据这最好的一处区域。”

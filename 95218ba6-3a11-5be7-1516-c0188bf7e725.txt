　　这样一尊内门风云人物，突然出现，也令任务殿中的不少人，都是纷纷侧目过来。
<br/>　　“白萱怎么会出现在这任务殿。”
<br/>　　“听她的话，似乎是来帮这新人解围的？”
<br/>　　“怎么可能，白萱的性格，虽然不算冰冷，但是别说是一个小小的外门弟子，就算是内门中的地榜弟子，她都不会放在眼中，怎么会特意为这样一个新人，前来解围。”
<br/>　　一些外门弟子，更是挤了进来，也要目睹一眼这内门风云人物的容貌。
<br/>　　“这任务殿的事，我的确管不着，不过话不公道了，我自然要出来说两句。”
<br/>　　房间之中，白萱看着陆鸣长老，话中的意思，毫不退让。
<br/>　　白萱目光闪烁，冷冷的说道。
<br/>　　“陆鸣长老莫非是想要将今日的事，捅上去？利用职权，针对弟子，至少也要百年禁闭的惩罚。”
<br/>　　“你！”
<br/>　　听着白萱的话，陆鸣长老脸色青一阵红一阵，他敢随意欺压楚风眠四人。
<br/>　　便是因为楚风眠四人，只是外门弟子，甚至是新人，在北邙学院中无权无势。
<br/>　　就算是他欺压了，学院高层也不会知道，不会有人提楚风眠四人出头。
<br/>　　可是眼前这白萱却是不一样，白萱是内门弟子，位列地榜前十，绝对是风云人物，若是她将这件事捅出去，陆鸣长老肯定会受到处罚。
<br/>　　陆鸣长老目光闪烁，最终脸上的怒意，缓缓的散了去，不过他的眼神，却是变的更加深邃，似乎是冷静了许多。
<br/>　　“白萱，今日的事，可是上面交代下来的，你可要清楚，你真要管这个闲事吗？”
<br/>　　白萱脸色平静，冷冷的开口道。
<br/>　　“你们陆家的事，我管不着，不过今日的事，必须要按照规矩办。”
<br/>　　“好，你有种。”
<br/>　　陆鸣冷哼一声，看向楚风眠的眼神中，露出几分怨恨道。
<br/>　　“这次的任务，算你们完成了。”
<br/>　　得到满意的结果，白萱便是点了点头，然后转过头去，对着楚风眠说道。
<br/>　　“走吧。”
<br/>　　说着，白萱便是一步踏出，直接离开了任务殿，而楚风眠则是紧跟其后。
<br/>　　至于莽通三人，楚风眠则让他们三人自己离开了。
<br/>　　“这白萱，到底是要做什么？我跟她素未谋面，为何她来帮我解围？”
<br/>　　楚风眠跟随这白萱，心中却是露出几分疑惑。
<br/>　　这白萱，楚风眠可是第一次相见，跟她就连半句话都没有说过，今日白萱却是突然出现，帮他解围。
<br/>　　像是白萱这等人物，所做的任何事，必然有着其中的理由，楚风眠不相信她只是随意路过，出手相助的。
<br/>　　毕竟今日的事，可是彻底得罪了陆鸣长老，以后再任务殿中，肯定也会受到一些针对。
<br/>　　楚风眠思考一阵，便是平静开口道。
<br/>　　“多谢白萱师姐解围。”
<br/>　　“道谢倒是不必了，我帮你解围，也是有着我的目的。”
<br/>　　白萱听到楚风眠的话，缓缓的转过头来，打量了楚风眠一眼道。
<br/>　　“听说你是寒月漓师妹的朋友？我跟寒月漓师妹在内门之中，也有过一面之缘，今日偶然碰到，便是顺手帮你解围，不算什么恩情。”
<br/>　　“果然。”
<br/>　　听着白萱的话，楚风眠心中倒是并不惊讶。
<br/>　　他跟这白萱，没有任何交集，唯一可能的，便是寒月漓了。
<br/>　　看起来寒月漓在内门之中，如今的地位倒是水涨船高，起码不是一个小小新人，就连白萱这等风云人物，都跟她有所相识。
<br/>　　今日白萱突然出面，为楚风眠解围，果然也是因为寒月漓。
<br/>　　看着楚风眠脸色平静，白萱的脸上倒是露出了几分惊讶，看着楚风眠开口道。
<br/>　　“哦？你不生气？我救了你，不过也是为了利用你，只是想要卖一个人情给寒月漓师妹。”
<br/>　　楚风眠笑了笑道。
<br/>　　“生气？为何要生气？白萱师姐帮我解围，无论原因如何，都是帮我了一个大忙。”
<br/>　　“能跟寒月漓做朋友，你果然不是寻常人。”
<br/>　　白萱听着楚风眠的话，脸色露出了几分轻笑道。
<br/>　　“我本来只是想要利用你，卖个人情给寒月漓师妹的，现在看来，似乎这人情，还不如直接卖给你好。”
<br/>　　“走吧，我们先去其他地方再谈。”
<br/>　　说着，白萱直接化为一道遁光飞走，楚风眠便是紧跟其后。
<br/>　　两人一路飞行，直接便是飞到了一处山脉之中，这山脉，便是一座六等山脉，白萱带着楚风眠，一路来到了这处山脉之中。
<br/>　　白萱转过头，对着楚风眠开口道。
<br/>　　“我的洞府，是在五等山脉之中，不过在哪里，我却是不能带你进去，只能是来到这六等山脉了。”
<br/>　　楚风眠脸色平静，笑了笑的说道。
<br/>　　“无妨，这六等山脉，对我来说，也是高不可攀的存在，能来，也是沾了白萱师姐的光了。”
<br/>　　白萱听着楚风眠的话，反问一声道。
<br/>　　“是吗？我倒是觉得，以你的实力，来到这六等山脉，夺得一处洞府，应该不难吧。”
<br/>　　听到这话，楚风眠只是随意的笑了笑，并没有开口。
<br/>　　两人一路飞行，大摇大摆的，降落到了这六等山脉之中。
<br/>　　“哦？白萱怎么会来这里？她不是在五等山脉中修行吗？”
<br/>　　“你们看她身后跟随的那个人，居然只是一个御风境的新人。”
<br/>　　“御风境新人也敢来我们六等山脉之中？”
<br/>　　“有意思，一个不知死活的新人，难道是抱上了白萱这颗大树？想要来这六等山脉中修炼？”
<br/>　　“可笑，太可笑了。”
<br/>　　一声声冷笑议论声，在周围响起。
<br/>　　这六等山脉中修行的武者，都是内门弟子，外门弟子中就算是人榜前十的人物，也只能去七等山脉中修行。
<br/>　　看这楚风眠，不少武者都是露出了几分不怀好意的笑容。
<br/>　　毕竟楚风眠来到这里，简直便是羊入虎口一番。
<br/>　　在北邙学院中，强者为尊，没有实力，便是没有任何地位，被欺压，也是经常的事。
<br/>　　甚至一些内门弟子，都以欺压外门弟子为乐，夺取资源等。
<br/>　　物竞天择，在北邙学院中，被发挥的淋漓尽致。

        “这个女人长得有点像那个女明星苏漫雪诶？”
<br/>
<br/>        “真的耶！好像噢！”
<br/>
<br/>        有人在旁窃窃私语起来。
<br/>
<br/>        童筱颖今天穿着一件精致的白色无袖衬衫，衬衫领的两角上有手工钉上去的杏色花型宝石，下搭一条配套的杏色纱裙和一双白色低跟鞋。
<br/>
<br/>        她美得太过灵动，让路过的男士们频频侧目。
<br/>
<br/>        虽然她长得像那个女明星苏漫雪，但是她应该不是苏漫雪。
<br/>
<br/>        前台小姐听着周遭那些闲言碎语，故意咳了一声，清了清嗓子，语气里已是带了几分不屑：“要想抓住一个男人的心，就得先抓住那个男人的胃。小姐，您这追男人的手段用得不错。不过了，您没有预约，是见不到我们傅总的。”
<br/>
<br/>        感受到四周投来的各种目光，童筱颖有些不知所措，但又不甘心就这样回去，索性，她豁出去了，直接改口，一脸傲气地说道：“我是总裁夫人！”
<br/>
<br/>        几个“噗嗤”异口同声，一旁路过的男男女女，听到她说这句话后，一个个像看到傻子一样，脸上露出了嘲笑的神情。
<br/>
<br/>        “每天想着要嫁给我们傅总的女人，实在是太多了，自称是总裁夫人找上门来的也不止你一个。”前台小姐嗤之以鼻道。
<br/>
<br/>        童筱颖抿了抿唇，努力地想要强调自己的可信度：“我说的是真的！”
<br/>
<br/>        “这位小姐，我看你这身打扮，想必你应该是出身于名门贵族吧！既然是名门闺秀，你就应该要脸点。你若是再这样纠缠，还不离开的话，我只好叫保安过来请你出去了。”前台小姐有些不耐烦地说道，她垮着脸，准备拿起电话按内线请人过来。
<br/>
<br/>        “等等！”童筱颖连忙阻止她打电话，又拿着手机翻找刘婶的手机号码，结果，也一样。
<br/>
<br/>        现在她的这部手机里，和傅夜沉共同认识的人的联系号码，也就只有傅昼景一个人。
<br/>
<br/>        童筱颖一脸挫败地挠了挠额头。
<br/>
<br/>        就在这时候，电梯门“叮”的一声打开，几个女人有说有笑地从电梯里走了出来。
<br/>
<br/>        落以柔被身边的女同事簇拥在中间，一眼就看到了站在前台的童筱颖，听着周遭的碎碎念，她大致了解了此刻发生在童筱颖身上的来龙去脉。
<br/>
<br/>        童筱颖自称是“总裁夫人”，却没有人相信她所说的话。
<br/>
<br/>        落以柔暗自得意地笑了笑，她就知道童筱颖这个女人很有手段，居然又跑到公司里来自称是“总裁夫人”，怕是又想逼着大哥就范，不得不让大哥当众承认她童筱颖的身份。
<br/>
<br/>        虽然，大哥已经告知过她，童筱颖就是“总裁夫人”，但是，落以柔此刻却选择了视而不见，完全没想过要替童筱颖解围，直接谈笑风生地和身边的女同事们一起右拐去了员工餐厅。
<br/>
<br/>        “那你可以不以帮我给傅夜沉打一通电话？”童筱颖一动也不动地站在原地，试图说服这位前台小姐。
<br/>
<br/>        但是，前台小姐语气里满是嘲讽：“这位小姐，请自重。”

<br/>        童筱颖看得出来，这位前台小姐，包括周围路过的男男女女，全都不相信她是总裁夫人，不过没关系，她来这里的目的只是单纯地为了给傅夜沉送午饭。

<br/>        既然自己不能上去，那么她把午饭放下，拜托这位前台小姐去送不就行了。

<br/>        童筱颖放弃了这个念头，不得不抱着保温饭盒，朝大厅那边等候区的沙发走去。

<br/>        落以柔和几个女同事在员工餐厅里吃完饭，又给傅夜沉也打包了一份，回来路过大厅时，发现童筱颖竟然还坐在那里不肯离去。

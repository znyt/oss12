　　第124章
<br/>　　队伍稀稀落落散落在路上，像将粮食搬回蚁穴的蚂蚁。
<br/>　　莽莽苍苍的山林，被大雪覆盖，在天未亮之时，林梢像蒙着一层黑纱。
<br/>　　“毛子，毛子……”
<br/>　　丛林中突然窜出七八个人，名叫二毛的山贼扭过头去，见都是山贼熟悉的弟兄。
<br/>　　这些人是在十天前离开山寨，临走前没和任何人打招呼。如此，倒也没什么好指责的，毕竟大家都是为了一口饭而已，只要不是如于虬然那样临走前一把火烧掉粮仓，把事情做绝就好。
<br/>　　“虎子，你们还活着？”二毛有些惊喜。
<br/>　　“唉，别提了，这几天我都不知道是怎么活下来的。”虎子叹口气，注意到二毛几人后面拉着的雪橇：“这上面是什么？”
<br/>　　“粮食，都是大当家运回来的。”
<br/>　　“这么多粮食！”虎子盯着那一麻袋一麻袋堆在一起的稻米，流露出贪婪的目光，他忽然道：“二毛，我求你件事，这些粮食让我弄走吧，哥哥我已经有好几天没吃过饭了。”
<br/>　　“这怎么能行！”二毛道：“当初我让你别走，你是不听我的，现在说什么都晚了。”
<br/>　　“谁他妈知道他真能弄粮食回来。”虎子再次恳求道：“毛子，求你让我把粮食弄走吧，哪怕就让我弄走一袋呢。”
<br/>　　二毛目光闪烁，最后还是坚定的摇摇头：“虎子，你去求求大当家让你回来吧，大当家心肠软，肯定会原谅你的。”
<br/>　　“大当家，哪个大当家？”
<br/>　　“程寨主！”
<br/>　　“呵，我在断粮的时候走了，现在山寨有了粮食，我再想回来，他怎么可能答应。再说我和他说不上话，还是你抬抬手，救哥哥一命吧。这么多粮食，就算我搬走一袋两袋，也不会有人知道。”
<br/>　　“这是程寨主弄回来，大家都要靠这些熬过这个冬天，我一粒米也不能让你弄走。”
<br/>　　“你活着，我们就要饿死！好，你不给我，我就自己动手抢！”
<br/>　　虎子亮出刀，从一开始他手中的刀就没放下过。
<br/>　　二毛大惊失色，队伍稀稀落落，他们早已和大队伍脱节，此刻周围并没有其他人。而因为他们几个要运粮的缘故，身上也并没有带兵器。
<br/>　　虎子一刀照着二毛脑袋砍来，他踉跄躲过，想要扑过去，又怎么敌的过这些饿夫的刀锋。
<br/>　　二毛等人只能眼睁睁看着他们将粮车拉起，自己却无力阻挡。想到程大雷不远万里将这些粮食弄回，而快要到家门口时，自己却把粮食弄丢，大家有什么脸面去见程大雷。
<br/>　　嗖！
<br/>　　一支羽箭凭空而来，穿入一个人的背心。二毛一楞，扭头看去，只见程大雷赵子龙几个，此刻从前方奔过来。
<br/>　　“愣着干嘛，谁抢你你就杀他！”
<br/>　　程大雷冲着二毛几人吼了一声，拎着斧头就冲了过去。赵子龙、张肥几个跟在他后面。
<br/>　　虎子几个抢到粮食，还没来的及欢喜，程大雷几个人就冲了过来。虎子等人已经不知道饿了多久，此刻根本没有多少力气。一见程大雷心里就怯了，心里怯了，腿也就软了。
<br/>　　“杀了，一个都不能放走！”
<br/>　　程大雷挥起斧头将面前一人砍了，他们这帮人真如狼入羊群相仿，杀的虎子一帮人落荒而逃。
<br/>　　嘟，收到恐惧值881
<br/>　　嘟，收到恐惧之666
<br/>　　嘟，……
<br/>　　程大雷心中一动，自己的恐惧值快到10万了。杀死呼尔勒之后，程大雷的恐惧值就一直在涨，海上遇到海贼又涨了一波，如今已逼近10万，等到10万后自己就可以来个十一连抽。
<br/>　　虎子吓的慌不择路，一头跌进雪洞中，回过头，程大雷拎着斧头已经逼近自己眼前。
<br/>　　“程当家，我错了，求求您，饶我一条命，我再也不敢了。”
<br/>　　“饶你麻痹！”
<br/>　　在又收割一波恐惧值后，程大雷一斧头狠狠挥下。
<br/>　　“虎子……”二毛下意思喊了一声，却仍然阻止不了程大雷的斧头挥下。
<br/>　　程大雷几个把他们的兵器捡了，都丢在雪橇车上。
<br/>　　“愣着干什么，快走，就数你们几个最慢。”
<br/>　　程大雷是很想把大家聚在一起，但奈何这些人的体力参差不齐，速度也是有快有慢。
<br/>　　一旦和大部队脱节，拉着粮食走在这荒野中，感觉就像一丝不挂的美女走在土匪窝里。
<br/>　　程大雷几个在队伍中来回穿梭，努力把大家赶到一起。最后费了九牛二虎之力，总算是将粮食平安运回山寨。
<br/>　　粮仓内堆满粮食，米山是一种怎样到情形，大家今天算是都看到了。所有人都累的气喘吁吁，可在此刻，这些人仿佛不知道累一般，想要触摸堆积如山的粮食，脸上呈现一种异样的亢奋。
<br/>　　“好了，好了。”程大雷吼了一句：“还有完没完，谁也不许碰了，现在……做饭吧。”
<br/>　　这些人已经记不的有多久没吃过一顿饱饭，程大雷一声令下，所有人都忙碌起来。就在蛤蟆岭上埋锅造饭，没过多久，米香便飘荡开来。
<br/>　　山上传来阵阵笑声，这些人饿了好久，此时纵然的到许多粮食，却也不敢有丝毫浪费，便是一粒掉在地上的米也捡起来放进锅里。
<br/>　　只有饿过肚子的人，才更懂的珍惜粮食的道理。
<br/>　　望着这一幕，程大雷忽然收到一条系统提示。
<br/>　　嘟，正义的少年喔，恭喜你完成任务，奖励……
<br/>　　“程寨主……”
<br/>　　此时，高飞虎，小白狼，熊大熊二等请牛山的山贼头目来到程大雷面前。还未说话，便同时跪倒。
<br/>　　“你们这是做什么？”
<br/>　　“程寨主，我们几个都商量过了。”高飞虎道：“从此以后，整座青牛山当以程寨主未尊。”
<br/>　　“嗯，知道了，起来吧。”程大雷道。
<br/>　　噫！
<br/>　　事情怎么和自己想的不一样，其实五十多家山寨商量好这件事，还是好不容易克服了一下心理障碍的。可看程大雷的表情，好像根本不把这件事当回事。
<br/>　　“现在青牛山的弟兄，有一个算一个命都是程当家救得。我们的意思是，从今天起大家就跟着程当家混。”
<br/>　　“你们……是想都加入蛤蟆寨？”程大雷皱起了眉头。

　　叶少阳点头，犹豫了一下，说出了跟地藏菩萨见面的经过，还有“救世四子”的传闻。
<br/>　　老郭听罢，呆若木鸡，叼着鸡爪子半天不动。
<br/>　　“操，你要……对付无极鬼王？”
<br/>　　“差不多吧，鬼王不除，我根本就不得安生。”
<br/>　　老郭深吸一口气，吐掉没吃完的鸡爪，再没胃口吃什么了，望着落日，眼神中带着忧郁。
<br/>　　叶少阳也不好说什么。
<br/>　　过了一会，老郭回过头来，冲他笑了笑，“小师弟，你不是凡人，我不该用凡人的标准来要求你，什么也不说了，老哥我陪着你，怕个毛，不就是什么鬼王么，也未必就长两个卵子！”
<br/>　　叶少阳本来心情听沉重的，听了这话忍不住噗嗤一笑，道：“人家是鬼，可能一个卵子都没有。”
<br/>　　两人碰杯喝酒，气氛又活跃起来。
<br/>　　泰山的落日虽然没有日出出名，不过也值得一看，看完日落，天气越发冷了，人开始陆续下山，等人都走光了，叶少阳和老郭开始行动，拆开一个大旅行包，找出两把折叠扁铲，钻到草丛里，来到帝君庙前门外，两人商量了下，找好位置，然后轮流挖坑，差不多后，老郭去抱了个坛子过来，放进坑里，然后戴上一次性手套，对叶少阳道：“你继续挖坑去吧，这个我来。”
<br/>　　叶少阳知道他要干什么，站在一边，看着他打开一个塑料瓶，将准备好的鸡肠子、内脏之类的倒进去，闻着味道叶少阳就赶紧走了，真不是嫌弃，大凡开了光的法器，都怕见到秽物，否则就不灵了。
<br/>　　叶少阳进到院子里，从一片断壁颓垣中摸到正殿前，找了差不多的位置，又挖了个坑，一抬头看到正殿里的帝君像，在月光下反着光，正面就对着自己，两只眼睛好像在望着自己一样，叶少阳笑了笑，“帝君，得罪了啊。”
<br/>　　老郭埋好了坛子，来到院子里，叶少阳已经在不同地方挖了三个坑，两人一起分别都埋了坛子，然后老郭给转上秽物，埋好，再用一个准备好的薄木板盖上。
<br/>　　叶少阳敲了敲木板，道：“这么厚能行吗？”
<br/>　　“到时候你瞧好了。”老郭对自己的机关很是得意，看了看自己埋下的四个桶，点点头，满意说道：“四个只要有一个能中就行。”
<br/>　　搞定了这些，叶少阳本来还想布阵，仔细想了下，又怕弄巧成拙，黎山老母这群人都不是凡角，而且既然偷偷过来，疑心肯定很重，祭祀之前，未必不会对周围展开搜查，要是被发现埋伏，别的设计也都白费了，干脆什么也不弄了。
<br/>　　两人一路下山，到半山腰的时候，耳边突然想起一个蚊子般的声音：“老大好走……”
<br/>　　是秋莹。
<br/>　　叶少阳转头看了看，没有看到她，知道她隐形了，笑着说声辛苦，一路下山，回到客栈里。
<br/>　　大伙都在，围着碧清的平板电脑，津津有味地看剧，见叶少阳来了，都起来打招呼，叶少阳让他们继续，自己到阳台上坐着，思考这场战斗的部署，偶尔看一眼屋里，看到一群人围着电脑，七嘴八舌地评论剧情，在叶少阳看来，这场面说不出的温馨。
<br/>　　不管将来怎么样，有这么一帮兄弟姐妹，自己也是值了。
<br/>　　林三生是半夜来的，跟小九一起，见面之后，连招呼都没打，对叶少阳道：“他们后半夜上山！”
<br/>　　“这么快！”
<br/>　　叶少阳当时刚躺下，正要睡觉，听见这话立刻跳起来。
<br/>　　“对，我得到情报，他们已经凑好了祭品，今晚就要行动。”
<br/>　　“今晚……”叶少阳看看他，又看看小九，道：“那我们就在这等着吧，等秋莹的情报。”
<br/>　　“不上山埋伏？”小九问道。
<br/>　　“不上山！”叶少阳解释，今晚一个人都不去现场，免得被黎山老母察觉，等秋莹给了信号，他们再摸上山去，先在安全范围之内潜伏，等到祭司仪式举行到一半，再突然冲过去，破坏仪式……到时候大家都蒙着面，尽量不要被看到脸，虽然黎山老母事后一定会猜到是他们干的，但毕竟没证据，到时候咬死不承认……
<br/>　　林三生听罢，点头道：“全听你安排，我跟小九不方便出面，你们先动手，假如成功了，就不要我们出面了倘若真遇到危险，那我们再上。”
<br/>　　叶少阳答应下来，问林三生：“祭祀的过程是怎么样的，你知道吗？”
<br/>　　林三生摇头，“这事太隐秘，没法通过线报得到，不过他们据说他们准备了一个人当祭品。”
<br/>　　“人当祭品？”大伙都吃了一惊。
<br/>　　“对，活祭。”林三生道，“而且据说这人还有特殊的身份，为了找他，还为了找这个人，还费了挺大力气。”
<br/>　　活祭……这是华夏最原始的一种祭祀仪式，并非道门独有，后来因为太血腥残忍，改用猪狗牛羊这些，来代替人，但法术界之中，一直有活祭的传统，不光是活人，任何邪物、生灵都可以用来祭祀，只不过为正修门派所不取，但仍然有邪修的、尤其是南方巫教所秘密进行。
<br/>　　东岳大帝既然是远古之神，那八成是喜欢活祭的。
<br/>　　叶少阳沉吟一下，问道：“他们有几个人？”
<br/>　　“从空界来的，有黎山老母、摇光仙子，还有众阁派的秋明子，还带了一些他们各自的入室弟子，也都是好手。只是……他们之前就来了人间，或许又找了一些人。咱们这些人，应该没问题，况且我们目的是破坏仪式，不是要拼个你死我活。”
<br/>　　林三生环顾众人，道：“都是自己人，客气话我也不说了，总之大家一定小心，注意安危，别的都是次要。”
<br/>　　叶少阳也嘱咐了一番，然后一起研究战术，把能确定的都确定了，不过大家都明白，计划再好也赶不上变化，真是什么情况，只有赶上了才知道。
<br/>　　事情商量好了，大伙就干等着，只有老郭在外屋鼓捣他那些“暗器”，一边摇头叹道：“我一个人弄不了这么多啊，小师弟，你应该把雨晴他们叫来一起啊，哦对，雨晴去外地了，啥时候回来？”
<br/>(本章完)

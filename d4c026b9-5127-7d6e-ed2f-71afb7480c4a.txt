        “你今天不用去打工吗？”何明旭诧异地问道。
<br/>
<br/>        诺筱颖摇了摇头，高兴地回答道：“你不是说，我没时间陪你吗？所以，以后，每个周末，我都来看你。这样，我们就有时间在一起了呀！”
<br/>
<br/>        “也、也行！”何明旭呵呵地笑了笑。
<br/>
<br/>        “你还没吃早饭吧！我有买早餐过来。对了，我也给你的舍友也带了。与他们初次见面，还希望他们日后多多关照你！”童筱颖一边说着，一边放下自己的背包，从背包里把自己买的早餐全都拿了出来。
<br/>
<br/>        何明旭不得不领着诺筱颖进了屋，帮她将早餐放在自己床铺下的书桌上。
<br/>
<br/>        “哟！何明旭，这女生是你的谁啊？”留着一个球头，皮肤稍微黝黑，看起来痞里痞气的章海昌，手里拿着漱口杯，嘴里还含着牙刷，从洗漱间里走出来，口齿不清地问。
<br/>
<br/>        “我一看，就知道是你的妹妹吧！”何明旭的邻床舍友方浩博，从自己床上直接跳了下来，将诺筱颖吓了一跳。
<br/>
<br/>        “我是阿旭的女朋友！你们好，我叫诺筱颖。”诺筱颖落落大方地自我介绍道。
<br/>
<br/>        一瞬间，宿舍里便全都安静了。
<br/>
<br/>        而且，还安静得异常诡异。
<br/>
<br/>        “你们，怎么突然不说话呢？”诺筱颖见这三个男生，一个个目瞪口呆地看着她，不禁好奇地问。
<br/>
<br/>        马智杰拿起眼镜盒里的黑框眼镜戴上，与站在洗漱间门口的章海昌，四目相对。
<br/>
<br/>        章海昌向马智杰挤眉弄眼，示意他不要多嘴。
<br/>
<br/>        诺筱颖下意识地回头，看向了仍旧坐在上铺愣着不说话的马智杰。
<br/>
<br/>        比起刚刚他给她开门时，那打着赤膊吊儿郎当的模样，现在的他，倒是衣冠整洁，戴着黑框眼镜，看起来倒是文质彬彬的样子，完全判若两人。
<br/>
<br/>        马智杰的近视度数并不深，只不过他学的是IT，因为常年要接触电脑，所以戴眼镜也就戴习惯了。
<br/>
<br/>        何明旭的宿舍是个混合宿舍，他们四个都不是同一个班的学生。
<br/>
<br/>        “嗨，你好，我叫马智杰！”马智杰回过神来时，发现诺筱颖正看着自己，本能地抬起手来挥了挥，并向她自我介绍。
<br/>
<br/>        何明旭接下来将自己宿舍里的另外两个人方浩博和章海昌，都介绍给了诺筱颖认识。
<br/>
<br/>        因为，今天与林若琴还有约会，何明旭不得不随便找了个借口，先离开了宿舍。
<br/>
<br/>        而诺筱颖被留在了宿舍里，替他洗衣服、洗被子，晒衣服、晒被子，又见宿舍里脏乱无章，然后她又替他们把宿舍打扫得干干净净。
<br/>
<br/>        最后，诺筱颖还把何明旭的白色球鞋也给洗了。
<br/>
<br/>        三个男生，看着这么个勤快、体贴、温柔的女生，一个个都不忍心向她说出何明旭的实情。
<br/>
<br/>        何明旭其实刚一进大学，就和文学系的系花林若琴对上眼了。
<br/>
<br/>        两人不到一个星期，就发展成了男女朋友关系。
<br/>
<br/>        他们宿舍里，何明旭确实是长得最帅的，所以，何明旭第一个交上女朋友，他们也不足为奇。
<br/>
<br/>        只是，让他们大跌眼镜的是，人家不只是只交了一个女朋友，而且还有另外一个单纯的小女生，为他毫无怨言的忙前忙后。
<br/>
<br/>        诺筱颖在宿舍里忙完后，马智杰从外回来的时候，给她带回了一瓶饮料。
<br/>
<br/>        何明旭却在跟诺筱颖谎称，导师给他介绍了去给高中生补习功课的外快，他以后每个周末只有上午的时间能陪她，然后一到下午和晚上就和另一个女友林若琴约会去了。
<br/>
<br/>        说到何明旭是否有陪伴她诺筱颖，这一上午，他也只不过是看着她帮他洗衣服洗被子，顺便在其他三个舍友的起哄下，把全宿舍的衣服被子都洗了。
<br/>
<br/>        诺筱颖为了何明旭，真的毫无怨言地全做了，而她却只是单纯地希望，日后，他们三个能对何明旭多多关照。
<br/>
<br/>        天底下这么傻的女人，只怕除了她诺筱颖，就没有第二个了。
<br/>
<br/>        以前每个周末，马智杰都不会在宿舍里待着，后来，诺筱颖来了后，他每个周末一整天都留在了宿舍里写程序。
<br/>
<br/>        而且，宿舍里没有人发现他这异常行为。
<br/>
<br/>        诺筱颖来之前，他是什么样子，她并不知道，她来了以后，宿舍里就成了她和马智杰的独处时间。
<br/>
<br/>        每个周末的中午过后，何明旭吃过午饭后就会离开宿舍假装去做家教，章海昌和方浩博也会相继出去打篮球。
<br/>
<br/>        然后，被留在宿舍里的诺筱颖和马智杰，在这独处的时间里，她一直在洗东西，而他一直坐在电脑前码程序。

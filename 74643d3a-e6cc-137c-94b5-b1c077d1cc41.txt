　　就这样，原始饶有兴味地处理着各类因果报应之事，但没过多久，他就过了有趣的阶段。之后，略有修改，分出一道念头，让他来处理这等事务，自己只负责准圣以上者的七曰或者誓言就行。洪荒这么大，如若不然，自己非得忙死不可。
<br/>　　因果之道执掌天道因果报应，那么天罚之道，又该如何？
<br/>　　原始若有所思，天罚之道，顾名思义，应该就是代天降下各种灾劫惩罚之类的事情了。果然，因果之道于天罚之道结合，最适合自己不过了。
<br/>　　想到此处，原始嘴角露出一丝诡异的笑容，笑容之中，充满意味。之后，他对天大声宣道：
<br/>　　“吾为天罚圣主原始，而今勒令天下。”
<br/>　　“吾成圣后，天下修士，不论强弱，凡突破境界，不论大小，皆须经受相应天劫考验，度得过，则进，度不过，则陨！”
<br/>　　“吾成圣后，天下修士，不入大罗者，每五百年，皆须经受雷灾考验，明心见性者，则过，否则，则陨。”
<br/>　　“再五百年，须经受火灾考验，成则过，不成则陨。”
<br/>　　“再五百年，须经受风灾考验，成则过，不成则陨。”
<br/>　　“吾成圣后，天下修士，不入大罗者，每三百年，须经历难劫考验，难分九难，成则过，不成则陨！”
<br/>　　“吾成圣后，天下修士，不论轻强弱，立下誓言，皆受吾管。如不能遵守，吾必将降下天罚，惩戒违誓之人。”
<br/>　　圣音宏大，传遍洪荒宇宙，传到每一个洪荒生灵脑海之中。一时之间，洪荒生灵，具皆大惊失色。
<br/>　　别人怎么样，原始却是管不了。为修士降下三灾九难以及突破境界时的灾劫，本就是原始早就想好的事。以前的洪荒，天道不算完善，只有修士化形之时才会出现雷劫，平时的灾劫并不多。如今，他执掌天罚，是该做出改变了。
<br/>　　修行本就是行非常之道，夺天地之造化，侵日月之玄机。只有你从天地之中获得，却没有付出。天地之间，哪有那么好的事？
<br/>　　是故，为修士的修道之路上降下各种灾劫，本就是理所应当的事。这样不仅可以除去一些洪荒之中的败类、祸害，而且更能提高洪荒修士的道心与顶尖战力。
<br/>　　每一个能够从种种劫难之中活下来的修士，无一不是明心见性，道心坚定，战力顶天之辈。
<br/>　　昆仑山，玉虚宫中。
<br/>　　“大兄，你可真是……霸气侧漏啊！不如我们论道一番，如何？”通天眼睛大亮，对着原始道。
<br/>　　原始微微一笑，道：“此事不急，待吾先发一道圣旨，邀天下群雄前来一起论道，岂不美哉！”
<br/>　　“啊？”通天目瞪口呆。
<br/>　　接着，原始挥手便是一道圣旨，上面写道：
<br/>　　“百年之后，昆仑圣宫；
<br/>　　悟道茶会，共邀群雄；
<br/>　　喝茶悟道，共论圣道。”
<br/>　　顿时，二十四字金光闪闪，圣威凛凛，道韵流转，神威莫测。原始一挥手，这一道圣旨便一分为十，之后，十道圣旨向着洪荒八方飞去，转瞬间便没入虚空之中，消失不见。
<br/>　　老子：“……”
<br/>　　通天：“……”
<br/>　　……
<br/>　　天庭太阳神宫之中，妖帝帝俊正在极品先天灵宝参悟河图洛书的奥妙，与太古星辰连在一起，让他参悟出一座大阵的轮廓，可以接引太古星辰之力作战，威力极为强大。可惜，它还不够完善，还需要仔细斟酌参悟。
<br/>　　正在此时，虚空一阵波动，一道金光闪烁，顿时一道圣旨飞出，化作一道金色流光落入帝俊手中。
<br/>　　“嗯？”帝俊猛然惊醒，抓起圣旨，看了一眼，沉默片刻，默默道：“悟道茶会？哼！”
<br/>　　他的眼神望向昆仑方向，明灭不定，之后又望向火云洞方向，不知道在心些什么。
<br/>　　“圣人，哎——”
<br/>　　密室之中，传来一阵幽幽的叹息声，微不可闻。
<br/>　　……
<br/>　　洪荒之中，一处山脉之间，溪水潺潺流淌，鸟语花香，仙花遍地盛开，景色怡人。
<br/>　　一处高台之上，伏羲缓缓波动手指，弹出那令人心旷神怡的悦耳琴声，如一道清泉，流淌在这山谷之间，流入人的心田。在他的对面，他妹妹女娲却静静站在那里，闭上双眼，享受着这令人羡慕的时光。
<br/>　　突然，女娲睁开眼睛，看向伏羲，一会儿之后，默默道：“哥哥，你说这世间若是没有那么多争斗，那该多好啊。”
<br/>　　伏羲的琴音微顿，但他没有停下，而是继续弹着。良久，他才轻声道：“那是不可能的，洪荒之中，为了利益，争斗是不可避免的。”
<br/>　　“是啊，所以，我才想要快速成圣，那样，我便可以庇护我想要庇护的人啦。”女娲在心里默默道。
<br/>　　突然，女娲心有所感，抬头看向不远处的一片虚空。虚空一阵抖动，一道金光闪现，淡淡的圣威弥漫在空中，将空气压缩。一道金色圣旨化作流光落入女娲手中。
<br/>　　“悟道茶会，原始讲道？”女娲轻声呢喃道，之后，便不再多语，眼神慢慢变得无比坚定，不知道在想些什么。
<br/>　　……
<br/>　　西方灵山，接引与准提正在悟道，一道金光闪现，圣旨落入接引手中……
<br/>　　万寿山五庄观，镇元子正在祭炼地书，参悟大地之道，在他的对面，红云道人正在吃着人参果。突然，金光乍现，圣威突降……
<br/>　　幽冥血海深处，血光闪闪，阵光耀眼，突然，从血海之中走出一道身影，他与冥河道人一般无二，只是比起冥河道人来，气息稍微弱些。之后，无数道血光闪现，无数道冥河的身影出现。
<br/>　　“哈哈哈！血海大阵，来自血魔的传承，吾终于成功创出来了。四亿八千万血神子，虽然他们现在还很弱，但总有一天，他们会变得与本尊一样强大。哈哈哈！血海不枯，冥河不死！哈哈！”
<br/>　　血海深处，传出冥河道人的狂笑声。突然，一道金光闪现，圣威驾临……
<br/>　　北冥深处，一处岛屿之上，北冥宫宫主妖师鲲鹏正在参悟大道，突然，金光闪现……
<br/>　　太阴星上，月桂秘境之中，太阴之主望舒与她的徒弟嫦羲正坐在月桂树下谈法论道，两人的眼神有些飘忽不定，正在此时，虚空波动，金光乍现，圣旨落入月神望舒手中……
<br/>　　“这个混蛋！”
<br/>　　秘境之中，传来月神的怒骂声。
<br/>　　……
<br/>　　十张圣旨，全在此刻，落入十个不同的修士手中，而且这些修士无一不是洪荒之中的大能者。原始开悟道茶会，便是想将他们这些平时难得一见的修士聚在一起，探讨大道。他的元始大道，需要吸取众家之长，众道之力，才能不断进步。而洪荒之中，道行高深者，也就这几人。当然，隐藏起来的就不算了。
<br/>　　昆仑山，玉虚宫中。
<br/>　　“大兄，你可真是……不知道说你什么才好？”
<br/>　　通天很是无语道。悟道茶会都能整出来，还有什么他干不出来的。
<br/>　　“呵呵！”原始微微一笑，毫不在意。
<br/>　　“静等百年之后，悟道茶会开始吧。”

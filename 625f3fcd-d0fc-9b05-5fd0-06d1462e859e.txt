　　“这……现在？”美华不明白叶少阳为什么对花童偷偷溜掉一点不放在心上，不过得到的却是叶少阳肯定的答复，只好走过去，双手搭在他肩膀上，心不在焉地捏起来。
<br/>　　突然感觉自己腿上多了个东西，低头一看，是叶少阳把手放了上去。
<br/>　　“老大……”
<br/>　　叶少阳闭着眼睛，假装没听见。
<br/>　　美华羞红了脸，只好当作没这回事，继续给他捏肩膀……
<br/>　　过了一会，叶少阳突然勾住她的脖子，往身上拉过去。
<br/>　　老大什么时候变得这么色了？
<br/>　　美华先是害羞，随即立刻想到了这个问题。不对劲！
<br/>　　猛然间，美华想到了通灵佛子花最擅长的手段：幻觉！
<br/>　　再仔细一想，没错，这一定是幻觉！
<br/>　　知道是幻觉，就好办了。美华是鬼，不能像人一样咬舌头，但是她有别的办法，当下作法清心，眼前的一切也越来越模糊，最后变得清晰……
<br/>　　还是在房间里，一切都没有变化，转头看去，叶少阳靠着餐桌站着，用一种看热闹的眼神看着她，而那个花童，就坐在桌子上，又变回了人形，甩着两条腿，一脸坏相，冲她直眨眼睛。
<br/>　　“你……老大，我刚才做什么了？”想起之前的经历，美华有点面红耳赤。
<br/>　　“你什么也没做，就在那里站着，脸色绯红，你怎么回事，被他弄得进入幻觉了吗？干什么了？”
<br/>　　“我……”美华当然说不出口，狠狠瞪了花童一眼，“兔崽子，你无耻！”
<br/>　　花童吐了吐舌头，捧腹笑道：“开个玩笑啦，大姐姐，谁让你绑了我那么久，总得让我出一口气吧。好啦。还用再比吗？”
<br/>　　“你那算什么本事，有本事真正来打过！”
<br/>　　“幻象法怎么就不算本事啦，刚才我可是手下留情了哦，不然……”花童笑了笑，“不过你放心啦，这件事我不会跟主人说的。”
<br/>　　叶少阳一听就愣了，“怎么还跟我扯上关系了，你给她弄的什么幻象？”
<br/>　　“咳咳，就是普通的幻象啦，主人，我过关了吗。”
<br/>　　叶少阳望着他，沉吟道：“你这幻象，是不是对任何生灵都有用？”
<br/>　　“说起来是的，不过有两种生灵，我无法用幻象迷惑，一种是道心极为稳定的，我无法进入他的内心，自然也无法制造幻象。还有一种，就是无心人。”
<br/>　　“什么意思？”
<br/>　　“无欲无求，也就是接近证道的那种。不会被一切幻象迷惑。”
<br/>　　叶少阳似有所悟地点了点头。
<br/>　　“这么说，一般生灵都会中你的手段？”
<br/>　　“是的啦。不过道心稳定的，或者心思剔透的……像主人这样的，还是比较容易会堪破幻境的。”花童拉住叶少阳的手，哀求道：“主人，你就收下我做你的妖仆吧。”
<br/>　　收他做妖仆……
<br/>　　叶少阳犹豫起来。
<br/>　　按说，自己现在虽然恢复了实力，但是捉鬼联盟的小伙伴多数都不在，自己将来想要回去，难保不会经历一些危险的事，多一个帮手，当然就是多一分力量，而且这小鬼头的幻象法术的确也有过人之处，说不定将来就能帮自己一把。
<br/>　　其次，他虽然是通灵佛子花的孢子，但跟之前那一株其实没什么关系，现在属于“人之初”的阶段，不说性本善吧，至少也不是什么恶鬼厉妖，正是需要引导的时候，如果放纵它在人间，说不定将来会成长到什么样子。收下当妖仆，倒也合情合理……
<br/>　　动了这份心思之后，叶少阳开始琢磨这件事的可行性。“我收下你是可以的，不过你不能认主。”
<br/>　　“为什么？”花童怔住，随后有些失望，“主人，你还是不信任我吗？”
<br/>　　“不是这样，只是因为……你认主也是白认，因为我根本就不是这个时代的人。”
<br/>　　花童呆呆地看着他，道：“什么意思？”
<br/>　　叶少阳把自己的来历跟他说了一遍，花童听完之后，彻底懵了。
<br/>　　美华也上来作证，补充了一些。
<br/>　　这些事实，不由花童不信。
<br/>　　“那你呢，你也没有认主？”
<br/>　　“我不用认主。百年后的我，是他的门人。我还需要认主做什么，一心一意伺候他就是了。”
<br/>　　叶少阳听了这话，心中也很感动，果然是“本性难移”，百年后的美华，虽然很臭美，有点高冷范儿，不过对于自己，那也是绝对的忠诚，让做什么就做什么，绝对不带反驳的。
<br/>　　一日为主，终生为主。
<br/>　　这是她的性格使然。百年前的她，依然还是这种性格。
<br/>　　“好吧。你跟我不一样，我还是要认主的……”
<br/>　　叶少阳道：“我到这个时代之后，那些门人的魂印，在我手上都没了，你现在认主，等我回去，可能魂印一样也没了，但是你魂晶已经吐出，却不能跟我心意相通，也很是麻烦。”
<br/>　　花童歪着脑袋想了一会，说道：“这个好说啊，我先不吐出魂晶，等主人你回去的时候，我跟你一起走。等到了你的时代，我再认你为主人，这样不就行了吗？”
<br/>　　“跟我回我的时代……”叶少阳犹豫起来。
<br/>　　“对啊，”花童看了看他的脸色，道：“主人你担心什么？”
<br/>　　“你是这个时代的生灵，如果到我的时代……我总觉得有点不对劲，有点造成时空混乱的感觉。”
<br/>　　花童挠了挠头，仔细想了下他说的问题，道：“主人，这个我想不用担心，我们通灵佛子花是地藏王菩萨点化，为无量界的生灵，一向不在三界五行之中，生死簿上也没我的名字，我在这里也不认识任何人，没前生也没来世，不管去什么时代都是一样的，不会有任何影响的。”
<br/>　　叶少阳想了下，好像也真的是这样。禁不住他的软磨硬泡，总算是答应了。
<br/>　　“太好了，我有主人了！”花童兴奋地跳起来。叶少阳看得出，他是发自内心地高兴。
<br/>　　叶少阳扶着他坐下来，说道：“既然你是我门人了，那我得跟你说说规矩，就说我们这个组织吧，分成两个部分，在人间的叫捉鬼联盟，在阴司还有一个衙门，叫阴阳司。”

        季雅珊会自杀！
<br/>
<br/>        而且还是咬舌自尽！
<br/>
<br/>        连华生已经听说了，好在当时的审讯人员反应及时，才没让季雅珊的自杀得逞。
<br/>
<br/>        “雅珊熬不住的话，她会选择自杀！你们不了解雅珊，但是我了解她。她是我最好的朋友，难道，我恳求大哥放了我最好的朋友也有错吗？”童筱颖握紧了手中的勺子，声音哽咽着接着反问道。
<br/>
<br/>        连华生抿了抿唇，只能低着头，默默地喝粥。
<br/>
<br/>        “大哥跟我说，她不会管雅珊的事。而我现在，被你们禁足在这私宅里，哪里都去不了。大哥不回来见我，自然是因为我和他之间，被雅珊的那件事情膈应着。他不想面对，我也不想面对。他心里装着‘大局’，而我心里只想着‘小家’。连医生，你是大哥的好朋友。你若是见到他，帮我给他带句话。”童筱颖已经放下了手中的勺子，很显然，碗里的粥对她来说，如同嚼蜡。
<br/>
<br/>        连华生见她没心情再喝粥，还是好心地劝了一句：“你边吃边说，别浪费粮食。”
<br/>
<br/>        “告诉他，我很想他。”童筱颖微微垂下眼帘。
<br/>
<br/>        “就这么简单？”连华生诧异，“你就没有别的要说的吗？”
<br/>
<br/>        童筱颖摇了摇头。
<br/>
<br/>        人的感情其实很复杂，但是“爱”很简单，却就是讲不清也道不明白，剪不断理还乱的那种。
<br/>
<br/>        落以柔也要他连华生给傅夜沉带过话。
<br/>
<br/>        不过，落以柔的那些话，都是一些“好好照顾自己、要平安回来”的关问话。
<br/>
<br/>        落以柔爱上傅夜沉的原因，其实在连华生眼里看起来，也确实带着目的性。
<br/>
<br/>        一来，傅夜沉长得帅；二来，傅夜沉多金；三来，傅夜沉还救过她落以柔；四来，两人工作上算是朝夕相处还很默契。
<br/>
<br/>        更何况，落以柔还是个孤儿。
<br/>
<br/>        在面对傅夜沉时，她落以柔自然是没任何抵抗力。
<br/>
<br/>        所以，落以柔对傅夜沉的爱里还包含着其他因素，这件事情，连华生早就知道了。
<br/>
<br/>        只是那天，落以柔问起他，拿她自己跟童筱颖去相比时，他才对她说：“就算她落以柔是这世上最适合他傅夜沉的女人，但她终究不是童筱颖。”
<br/>
<br/>        过了许久，童筱颖才弱弱地问道：“大哥他……是不是又出什么任务去呢？你们上次围剿古堡，他‘四爷’的身份，有没有曝光？雅珊她现在还好吗？”
<br/>
<br/>        “你问题太多，我答不上来。”连华生傲娇道。
<br/>
<br/>        刚刚被她怼了好几次，他也得逞一下口舌上的威风。
<br/>
<br/>        童筱颖努了努嘴，不得不一个问题一个问题地来：“大哥他去哪呢？”
<br/>
<br/>        “出任务去了。”连华生敷衍道。
<br/>
<br/>        “那雅珊呢？她……”
<br/>
<br/>        “放心吧！她死不了！”连华生直接打断了童筱颖的话，“我们会盯着她，不会让她自杀的。虽然，她会尝些苦头吃。”
<br/>
<br/>        要不是为了一个季雅珊，阿沉至于落得个被全球通缉的下场吗？
<br/>
<br/>        她童筱颖怕是不知道，阿沉若是被抓回来，哪怕他没有出卖他们组织里的机密，但十有八九也是会被终身监禁。
<br/>
<br/>        要想傅夜沉被解除通缉令，除非傅夜沉把那个暗道上的冒险者集团给团灭了，立了大功回来。
<br/>
<br/>        不过，想想都觉得不大可能。
<br/>
<br/>        就算阿沉再怎么隐瞒身份，明暗两道，都不可能再用他了。
<br/>
<br/>        那个冒险者集团的高层能藏匿这么多年，光手下那十位爷都藏得那么深，肯定也在他们安防局里安插了卧底。
<br/>
<br/>        和夏阳枭一样，是一只狡猾透顶的老狐狸，只是夏阳枭代表着正义，而那个冒险者集团的领导人物代表的邪恶罢了。

<br/>        连华生的思绪到这里，忽然间意识到了一个问题。
<br/>
<br/>        傅夜沉向来不会去做没有把握的事情，就算他这次是真的为了童筱颖，突然决定只身去救季雅珊，但是以傅夜沉的性子，他不会这么唐突！
<br/>
<br/>        难道，阿沉他有自己的另一套他连华生都不知道的计划？！
<br/>
<br/>        阿沉心里到底在盘算着什么啊？
<br/>
<br/>        连华生匪夷所思起来。
<br/>
<br/>        “大哥什么时候会回来？”童筱颖又问道。

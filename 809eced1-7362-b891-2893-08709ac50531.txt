　　内门十堂，位于剑派中央位置。
<br/>　　亭台楼阁，雄伟宫殿，精致楼宇，应有尽有，居住着内门弟子与内门执事。
<br/>　　环境优雅不说，还被一大型聚灵阵覆盖着。
<br/>　　故那里的灵气远超外面的地方。
<br/>　　虽比不上寒冰池那般效果恐怖，但也是相当不错的修炼宝地。
<br/>　　而外门，则位于剑派前方。
<br/>　　灵气也仅比外面的地方高一些，聊胜于无。
<br/>　　占地广阔，却是万多名外门弟子以及外门执事的住所，简陋得多。
<br/>　　此时，外门后山之上，一孤傲少年，正盘膝打坐着，面朝太阳，俊朗而洒脱的面庞上，显得很是安宁。
<br/>　　少年自是萧逸。
<br/>　　距离上次的碑林悟道，已经过去了三天。
<br/>　　武道碑林，位于内门位置。
<br/>　　他离开后，便回到了外门的区域。
<br/>　　早在通过第二轮考核时，那三百人便已经是裂天剑派的正式弟子了。
<br/>　　第三轮考核，只是决定能否进入内门。
<br/>　　只不过，原本来外门区域的，只有280人，如今，却成了281人。
<br/>　　多了他萧逸。
<br/>　　这三天，他都在后山上独自修炼。
<br/>　　外门弟子，住所虽比不上内门弟子的优雅楼宇，但也不错。
<br/>　　只是，他更喜欢后山这安静的地方。
<br/>　　忽然，四道身影忽然快速袭来，最后停在了他面前。
<br/>　　正是林劲四人。
<br/>　　“萧逸。”林劲等人看着萧逸那孤傲的身影，一时间竟不知说些什么好。
<br/>　　“不好好在内门修炼，来外门区域做什么？”
<br/>　　最后还是萧逸率先打开了话匣子，淡淡地站了起来，笑着问道。
<br/>　　“我们...”林劲想说些什么。
<br/>　　但萧逸却发现他们脸色苍白，精神也似乎并不大好，皱眉打断道，“怎么回事，你们受了伤？”
<br/>　　“顾长空为难你们？”萧逸的双眸，一下子变得冰冷起来。
<br/>　　三天前，他知道顾长空此人，并非善类，且巧舌如簧，心思狠辣。
<br/>　　而自己乃是事件的中心，也是顾长风的眼中钉。
<br/>　　若自己不立刻将事情结束，顾长空将寻到更多办法，刁难林劲四人。
<br/>　　所以，他没有做任何辩解，放下顾长风的东西后，毫不犹豫地走了。
<br/>　　为的，就是堵住顾长空的口，也是防止顾长空有更多的借口对付林劲四人。
<br/>　　他很清楚，顾长空只是剑堂首席，就算有一定的权力，也不可能影响到其他内堂。
<br/>　　只要当时在武道碑林，顾长空没有机会和借口对付林劲四人。
<br/>　　一旦林劲四人顺利进入内门，顾长空便再也难以刁难他们。
<br/>　　萧逸一向心思缜密，心智过人，做什么事都有着自己的打算。
<br/>　　这时，林劲摇了摇头，讪笑道，“只是我们自己去找顾长空打了一架罢了。”
<br/>　　“但结果，你也看到了，并没能奈何得了他。”
<br/>　　“笨蛋。”萧逸骂道，“那顾长空，以我的估计，起码在洞玄七重以上的实力，你们怎么可能是他的对手。”
<br/>　　说着，萧逸拿出四粒生灵丹，给四人服下，道，“安心在内门修炼，我的事，我会自己解决。”
<br/>　　“抱歉。”林劲忽然呐呐地说了一句。
<br/>　　“抱歉什么？”萧逸笑骂一声。
<br/>　　萧逸很清楚林劲坚强的性格，他还是第一次见到林劲这副表情。
<br/>　　林劲认真地说道，“萧逸，你嘴上说着我们是笨蛋。但你很清楚，我们不是笨蛋。我们自然知道，三天前你为何毫不犹豫地离开。”
<br/>　　柳嫣然更是直视萧逸，道，“你的性子，我很清楚，若不是要顾及我们，你绝不会轻易离去。”
<br/>　　萧逸摆摆手，轻笑道，“与你们无关，不必多想。”
<br/>　　“不。”柳嫣然忽然走到萧逸面前，柔声道，“萧逸，我终于知道，当初在第一个考核时，你为什么不告诉我们你的武魂是什么。”
<br/>　　“一直以来，你实力过人，剑术超凡，我们怎么都想不到你的武魂竟然是控火兽。”
<br/>　　“我没有轻视你的意思，只是，你本就武魂资质低下，如今又不能享受内门的充足修炼资源，这对你来说，无疑是雪上加霜。”
<br/>　　柳嫣然说着，眼中尽是愧疚之色。
<br/>　　但下一秒，她的双眼，立马迸发出浓烈的杀意。
<br/>　　“裂天剑派，每五年都会举行一次全派比武，挑选出全派首席弟子。”
<br/>　　“全派首席，凌驾于内门十堂首席之上，必须力压内外门所有弟子方能胜任。”
<br/>　　“据我所知，那顾长空也没什么了不起的，在内门十堂中，实力不过排名第三。”
<br/>　　“算算日子，每五年的全派比武，正是两年之后举行。”
<br/>　　“两年后，我必要顾长空从裂天剑派除名。”
<br/>　　除名二字，她几乎是咬着牙说的。
<br/>　　柳嫣然，平日里一直是个温柔如水的女子，容貌绝色，性情恬静。
<br/>　　谈吐幽柔，如飘舞的柳絮，让人如沐春风，心情舒畅。
<br/>　　能让她如此杀机浓郁，代表着她心中已经无比愤怒，更是在心中将萧逸放到了一个极高的位置。
<br/>　　这时，林劲也认真地说道，“如今，我已入刀堂；铁牛和秦飞扬入了兽堂，嫣然则入了万木堂。”
<br/>　　“两年后的全派比武，以萧逸你的实力，就算得不到第一，只要做到大放异彩。”
<br/>　　“我们四人，必能让你重回内门。”
<br/>　　“额。”萧逸愣了愣，而后说道，“其实，对于我来说，在哪里修炼都是一样的。”
<br/>　　“和顾长空之事，我也会自己解决。不过，我还有更重要的事要处理，暂时不理他罢了。”
<br/>　　“更重要的事？”林劲面露疑惑。
<br/>　　“嗯。”萧逸点点头，道，“总之，你们不用担心我，安心在内门修炼即可。”
<br/>　　“好。”林劲答应道，“其实，今天过来，也是跟你说一声。”
<br/>　　“内门的修炼，很是严格。下一次，我们要突破洞玄境才能出来找你了，起码要数月之后。”
<br/>　　“没事。”萧逸耸耸肩。
<br/>　　“萧逸，我们不在的时间里，如果有人欺负你，便把他们记下来。等我出来，我会一一替你教训他们。”
<br/>　　说话的是秦飞扬，他一直没说过什么。
<br/>　　他平日里爱耍帅，爱出风头，可是，他极重义气。今日，竟是不知该说什么好，唯有即将离开时，才说了那么一句话。
<br/>　　“好。”萧逸虽然觉得没人能欺负他，但还是点了点头。
<br/>　　四人几个呼吸间，瞬间离去，想来在内门中是要努力修炼的，不得随便浪费时间。
<br/>　　萧逸也重新盘膝坐下修炼。
<br/>　　他之前没有说谎，对于他来说，在哪里修炼，确实都是一样的。
<br/>　　他来裂天剑派的最开始目的，根本不是为了进入内门，仅仅是一段缓一缓的时间。
<br/>　　从始至终，他最缺的只有时间。
<br/>　　没有人知道，他的天赋到底恐怖到什么地步。更没有人知道，他仅仅用了半年，就能从凡境一重跳到如今的先天九重。
<br/>　　有内门的修炼资源辅助，自然最好；没有，问题也不大。
<br/>　　至于他口中所谓的大事，自然是一年后要回萧家一趟。
<br/>　　当然，和顾长空的恩怨，他也不会忘记。
<br/>　　“呵。”萧逸忽然睁开了眼睛，冷冷地笑了笑。
<br/>　　“三十年河东，三十年河西。两年时间，对武者而言，弹指便过。顾长空，这笔账，届时我会与你好好算个清楚。”
<br/>(本章完)

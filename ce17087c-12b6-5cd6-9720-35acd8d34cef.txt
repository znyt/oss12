陆寒霆看着她通红的眼眶，他知道自己伤到她了，可是，他必须亲手将她给推开啊。
<br/>
<br/>        以前总是心存侥幸和贪婪，以为将她攥在手里就可以收获幸福，原来不是这样的，幸福与他无缘。
<br/>
<br/>        “陆太太，你现在跟我拿膜说事？取悦男人的方式有很多种，我怎么知道你有没有像取悦我那样取悦过陆子羡，再说了现在医学很发达，谁知道你那道膜是不是补的，对于你这个学医的来说，补个膜应该手到擒来吧…”
<br/>
<br/>        话还没有说完，夏夕绾抬手，用力的给了他一耳光。
<br/>
<br/>        啪，一声，陆寒霆被打偏了整张俊脸。
<br/>
<br/>        他所有薄情的话语都被打断，整个房间陷入了死一般的沉寂。
<br/>
<br/>        指间的香烟因为太长时间没有抽，现在猩红的火苗灼到了他的指腹，但他一点反应都没有，一点都不觉得疼。
<br/>
<br/>        这时夏夕绾抬起小手，抚上了他被打的俊脸，“有没有将你打疼，对不起，我不是故意要打你的…”
<br/>
<br/>        陆寒霆转过脸看着她，她澄亮的眼眶里已经覆上了一层晶莹的水雾，但是她倔强的没有将眼泪掉下来。
<br/>
<br/>        伸出小手抱住了他精硕的腰身，夏夕绾将苍白的小脸和冰冷的身体都贴在他怀里，“陆先生，我好冷，抱抱我，抱抱好不好？”
<br/>
<br/>        指尖的香烟掉了下来，两只大手垂在身侧，他已经感觉到她冷的像冰块的一样的体温，还有她的脸色很不好，她刚刚才二次试毒过，现在身体一定很难过，她嗓音软糯的求他抱抱她。
<br/>
<br/>        陆寒霆抬起手，但是半空中又僵住了，不敢给她一点点的回应和温柔。
<br/>
<br/>        “陆先生，你今天说够多了，不要再说了，你说离婚我就当你在开玩笑了，我不会当真的，我怎么会跟一个生气的人计较呢，我头有点晕，你抱抱我。”夏夕绾紧紧的抱着他。
<br/>
<br/>        但是她没有得到回应，男人僵着身体给她抱着，没有说话，
<br/>
<br/>        夏夕绾抬眸，仰着水汪汪黑漉漉的翦瞳望着他，“我可以向你发誓，如果除了你以外我有其他男人的话，就让我明天出门被车撞…”
<br/>
<br/>        这声“死”还没有说出口，陆寒霆已经一把将她推开了，他直接将她推出了自己的怀抱。
<br/>
<br/>        夏夕绾没站稳，一下子就摔在了松软的地毯上。
<br/>
<br/>        摔下去的瞬间她觉得两眼一黑，视线里什么都看不到了，整个世界都归于黑暗。
<br/>
<br/>        她是失明了么？
<br/>
<br/>        失明的恐惧让她瞳仁骤缩，僵在地毯上都忘记了反应。
<br/>
<br/>        陆寒霆看着她，狭长的眼梢布满了红血色，心里疼的睚眦欲裂，垂在身侧的大手捏紧了，再松开，他反复喘动着胸膛，用尽了全身的力气才忍着没有去抱起她。
<br/>
<br/>        薄唇掀动，他听到自己残忍薄情的低哑嗓音，“我早就说过了，我不缺陆太太，给自己保留一点体面和尊严，别纠缠我，早点签字离婚。”

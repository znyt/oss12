　　“对、对不起。”
<br/>　　佟千雪对着孙涛深深鞠了一躬，说道：“我也不知道会发生这些。”
<br/>　　“对不起？”孙涛抬脚对着旁边的树干，踹了上去，说道：“对不起要是有用，还要警察和法律干什么！？”
<br/>　　“孙涛，你差不多点行了啊！”
<br/>　　慕容飞这个时候走上前，指着孙涛说道：“我当你是兄弟，上了战场我可以帮你挡子弹眼睛都不眨一下，可是这并不意味着你可以责骂我老婆！”
<br/>　　“你们都安静点！”
<br/>　　齐凡上前把两个人推开，瞪了两个人一眼，说道：“你们还嫌事情不够麻烦？还要自己内讧！？都是还有两三年就强制退出作战部队的老鸟了，说话做事能冷静点吗？”
<br/>　　“无常。”齐凡对慕容飞说道：“出林子以后，你跟特勤小队那边联系，先护送佟医生回国，后面的营救行动你暂时不用参加了。”
<br/>　　“你也信不过我！？”慕容飞显然是对齐凡的这个安排有些不满意。
<br/>　　“不是信不过你，而是做出合理的安排。”齐凡拍了拍慕容飞的肩膀，说道：“我们后面的营救行动必然会很危险，佟医生跟在我们身边显然不合适，而且我们现在没有多余的人员去保护她，所以把她送回去是最好的选择。”
<br/>　　慕容飞深吸一口气，把自己那支M200狙击步枪和剩余的弹匣交给了齐凡，说道：“这把枪的弹道偏右，而且没有辅助计算机帮助，使用之前最好重新校准一下。”
<br/>　　齐凡接过慕容飞递过来的枪，对慕容飞叮嘱道：“路上小心！”
<br/>　　慕容飞点了点头，带着佟千雪转身离开。
<br/>　　目送着慕容飞和佟千雪离开，齐凡把A组和C组剩余的队员都集中到了自己身边。A组这边，除了齐凡和离开的慕容飞以外，还剩下陈应天、周立和孙涛三人；C组还是梅庆伟、张子航、刘子锦三个人。
<br/>　　“现在我们还有七个人，加上布谷鸟、天蛇、玄龟和毒液，一共是十一个人。”齐凡看了眼自己身边的战术中队队员，说道：“我们现在目前最大的困难就是弹药的问题。”
<br/>　　“这个我们可以回安全屋，可以去那里获得补给。”梅庆伟提议道。
<br/>　　齐凡点了点头，拿出那部罗昊交给他的手机，说道：“这是狂龙交给我的，他认为这其中很有可能存在布克尔特手中那支新型P病毒的线索，如果这其中真的有新型P病毒的线索，我们就必须要把布克尔特藏起那支P病毒给找出来。”
<br/>　　除此之外，齐凡还吩咐所有人切断跟特勤小队的联系，有美国人的介入他们的通讯很有可能会被监控，如果被监控到跟特勤小队的联系，进而挖出隐藏的特勤小队，这样造成的损失会很大，特勤小队在这里的情报点和安全屋就会全部暴露。
<br/>　　“现在我们要怎么跟布谷鸟他们联系？”刘子锦问道。
<br/>　　“狂龙在行动前，准备了一个接头地点。”梅庆伟说道：“他一开始就做了预案。这件事情一开始我们都不清楚，是行动正式开始之前，狂龙才告诉我们的，布谷鸟现在应该已经赶到那里去了。”
<br/>　　“狂龙就是狂龙，这么有先见之明，什么事情都已经想好了啊。”周立感叹道。
<br/>　　“所以他是‘龙刺’，而你不是。”陈应天拍了拍周立的肩膀，说道：“既然狂龙都已经有了安排，我们就先去跟布谷鸟他们接头，再商量后面的事情。”
<br/>　　除了齐凡和梅庆伟，出乎所有人意料的，罗昊选择的接头地点就是之前塔尼特跟她父亲见面的那所被废弃的学校。
<br/>　　“嗬，真没有想到狂龙居然会选择这里做为接头的地点。”陈应天环顾了一下这所破损严重的学校，说道：“居然用比洛多跟他女儿的接头地点直接拿来用，高，的确是高。”
<br/>　　“正因为你们都想不到，所以比洛多也不会想到。”齐凡解释道：“我哥在行动前跟我、宿鸟和毒液都谈过，如果这次行动中会发生意外情况，那么这个意外情况大概率会发生在比洛多身上，因为这是一个不可控因素。这里是比洛多和他女儿约定好的安全屋，只有他跟塔尼特知道，所以他不会在第一时间就考虑这里，因为这里在比洛多的脑子中就是一处秘密的安全屋，这个印象很深，短时间内他不会转变过来。”
<br/>　　进入学校的时候，齐凡发现了布置在教学楼楼梯口的诡雷，上次他们来这里的时候根本没有诡雷，而且从痕迹上看是新布置的，应该是布谷他们做的。
<br/>　　齐凡没有动手去拆掉这个诡雷，直接抬脚跨了过去，身后背着慕容飞那支M200狙击步枪，手中握着一支紧凑型HK416自动步枪，沿着楼梯往上走去。
<br/>　　约定的教室中，齐凡、梅庆伟等人跟布谷他们四个人顺利碰了头。
<br/>　　“到底是怎么回事？”莱昂对齐凡问道：“行动为什么会产生纰漏，还有狂龙他们人呢？”
<br/>　　“西伦科特把一支新型P病毒给了布克尔特，布克尔特用这支新型P病毒跟美国军方达成了交易，他们介入了进来。”齐凡把情况进行简单的说明，“而随着美国方面的介入，比洛多放弃了跟我们的合作，转而跟美国军方合作，狂龙和B组掩护我们撤退被抓了。”
<br/>　　“我就知道那个比洛多不可信，靠！”布谷抬脚踢了一脚旁边的桌子，对齐凡问道：“现在我们要怎么办？去干掉比洛多？”
<br/>　　“我们在布克尔特身上找到了一个手机，狂龙在被抓前把手机给了我。”齐凡把手机拿了出来，说道：“狂龙认为，布克尔特随身带着这个手机，里面有可能藏着跟那支P病毒有关的信息。”
<br/>　　这是一部很普通的智能手机，解锁手机，电量还剩下百分之五十左右，手机里面没有安装太多的应用程序，仅有的几个应用程序全都被放在了桌面上。

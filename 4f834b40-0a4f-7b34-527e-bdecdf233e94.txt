        良久后，季思妍才弱弱地开口：“三年前，是我求阿景给我的一个孩子。”
<br/>
<br/>        “我问的不是这个！”童筱颖疾言厉色道。
<br/>
<br/>        季思妍身心一怔，缓缓地垂下眼帘，低声下气地回答：“我和阿景上过床，但我真的只是单纯地想要个孩子。我不想与你争阿景，我也不想插足你们的婚姻。我现在，只想救我的儿子。希望我的儿子好好地活着。所以，筱颖，我求你接受我的儿子，好吗？”
<br/>
<br/>        这三年来，季思妍想通了。
<br/>
<br/>        如果傅昼景还爱她，他就不会抛弃她，和童筱颖远走高飞。
<br/>
<br/>        三年前，她其实是打算用孩子留住傅昼景。
<br/>
<br/>        但是，在这三年的时间里，傅昼景都不曾主动联系过她，就连一个通讯地址都不愿意透露给她，几乎跟她把关系断得一干二净。
<br/>
<br/>        如果不是儿子季小景患病，她是真的不再对傅昼景抱任何希望了。
<br/>
<br/>        就连她现在找到这里，还是带着儿子小景去见了冷爷爷，跪在冷爷爷面前，冷爷爷派人给小景做了亲子鉴定，确定是阿景的亲生儿子后，才给了她阿景在这里的住址。
<br/>
<br/>        季思妍甚至看得出，就连冷爷爷也在嫌弃小景的出身。
<br/>
<br/>        从那一瞬间，季思妍这才明白，自己得替自己三年前为爱冲昏头脑的愚蠢行为买单。
<br/>
<br/>        她其实算计过她童筱颖，但是每一次都被她童筱颖侥幸逃脱，而这一次，她不打算再去算计她什么了。
<br/>
<br/>        甚至，她都不想跟她童筱颖去争阿景了，只想好好地照顾着自己的儿子就足矣。
<br/>
<br/>        但在童筱颖听来，她季思妍的这些话，简直就是在跟她说冷笑话，甚至对她童筱颖来说，就是一种嘲讽。
<br/>
<br/>        小三带着自己丈夫的儿子，登堂入室，还在她面前扬言，不是来插足他们的婚姻，还要她接受他们的儿子。
<br/>
<br/>        如今的小三，都不再嚣张，反而卖起了苦楚了吗？
<br/>
<br/>        童筱颖微微仰起头来，不想让自己在季思妍这个小三面前掉眼泪。
<br/>
<br/>        两人面对面沉默了许久，而季思妍的那个才两岁多的儿子，竟然就那么安安静静地一个人坐在客厅里不吵也不闹。
<br/>
<br/>        童筱颖透过厨房大门，可以直接看到那边客厅里的情形，看到那个孩子一动不动、端端正正地坐在沙发上，有着与同龄的孩子不相符的性格，心里开始百感交集。
<br/>
<br/>        她恨这个孩子的同时，又对这个孩子产生了怜悯之情。
<br/>
<br/>        人的感情为什么要弄得这么复杂？
<br/>
<br/>        又有谁能告诉她，她现在该怎么做这个抉择？
<br/>
<br/>        童筱颖越是不说话，季思妍越是惶恐不安。
<br/>
<br/>        如果，她不能从童筱颖这里找突破口，那么她就只能赖在这里不走，等傅昼景回来解决这件事情。
<br/>
<br/>        “筱颖，我知道，你现在一定难以接受这个事实。你要怨我，怪我，甚至打我都可以。可是，我的孩子是无辜的。他还这么小，需要我们这当大人的去呵护，我的孩子真的等着阿景的第二个孩子的脐带血去救命！筱颖，冷爷爷不肯告诉我阿景的联系方式，我希望你能告诉我。或者，你可否让我在这里等阿景回来。你放心，我不会再跟阿景发生任何关系。我可以接受人工……”
<br/>
<br/>        “够了！”季思妍的话还未说完，童筱颖忽然间冷冷地呵斥了一声。
<br/>
<br/>        她到底要在她面前喋喋不休到什么时候？
<br/>
<br/>        童筱颖对季思妍产生了一种难以言喻的反感。
<br/>
<br/>        季思妍看着童筱颖的双眼里布满了血丝，立即识趣地闭上了嘴。
<br/>
<br/>        童筱颖随后二话不说地离开了厨房，拿来了自己的手机后，将里面傅昼景的联系方式点开，举到了季思妍的眼皮子底下。
<br/>
<br/>        “这是你和昼景两个人之间的事情，所以，你们俩个人之间自己去解决。不要告诉他，是我告诉你的他的号码，更不要跟他说，我已经知道了你两的事情。你就当我什么都不知道，更不要跟他提起，你来过这里。”童筱颖一口气说到底，见季思妍存好了傅昼景的号码后，立即就对季思妍下逐客令。
<br/>
<br/>        其实，她内心已经愤恨到了极点，甚至很想狠狠地给季思妍一巴掌。
<br/>
<br/>        但是，她并未那么做，甚至不知道自己为什么要把自己丈夫的联系方式给季思妍。
<br/>
<br/>        季思妍拿到傅昼景的联系方式后，不禁喜上眉梢，就在她眉开眼笑的时候，甚至都没想到过要去谢谢她童筱颖。
<br/>
<br/>        童筱颖在季思妍兴高采烈地带着那个孩子离开家门后，独自一个人背靠着家门的门板，在门口呆呆地坐着。

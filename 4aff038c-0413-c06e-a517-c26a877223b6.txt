　　“这些老古董才是圣龙宗真正的仇敌，杀了他们，从此之后圣龙秘境才算是真正归我们掌控。”

<br/>　　七名半步天人境强者，数量的确惊人，但是这一次楚风眠也是做好了准备。
<br/>　　这天星河，便是对付他们的第一利器。
<br/>　　而且楚风眠在踏入九劫古帝之后，还从未全力出手过，这一次正好是检测楚风眠实力的时候。
<br/>　　“走，我们进去。”
<br/>　　另一边，那些圣龙秘境老古董，看到天星河没有异样之后，便是一下子踏入到了天星河之中。

<br/>　　以天星河力量化为了海浪，向着那所有老古董轰击过去。

<br/>　　看到这周围突然的变化，法理古帝率先反应过来，从他的身上打出了一道巨大无比的阴阳阵图。
<br/>　　这所有的圣龙秘境老古董，一下子跃入到了这阵图之中去，化为了其中一枚枚的阵眼，凝聚灵力打入到阵图之中去，抵挡着天星河的轰击。
<br/>　　天星河化为的海浪，却是根本无法打破这阵图的防御。
<br/>　　“这些老家伙的手中，倒是有些好东西。”
<br/>　　楚风眠一眼就看出这阵图，也是一件品质极高的灵天级器，而且无比古老，应该是极为古老时代炼制出的古灵器。
<br/>　　不过品质在高的灵器，等落入到了楚风眠的手中之后，都会成为神树的养料。
<br/>　　“没想到天星河真的被人掌控了，楚风眠，你还不现身？”
<br/>　　抵挡住了天星河的一道道轰击，那圣院宗的宗主，凌山大吼一声。
<br/>　　只见一道身影，便是显现在了众人的面前。
<br/>　　这一道身影，正是楚风眠。
<br/>　　楚风眠从一开始，也没有打算藏头露尾，毕竟单单凭借天星河，对付七名半步天人境，数十位九劫古帝的联手，还不可能。
<br/>　　必须要楚风眠亲自出手才行。
<br/>　　青梦也是现身，站在了楚风眠的身边，冷眼看着在场的所有人。
<br/>　　“九劫古帝？”
<br/>　　看到楚风眠现身，众人打量了一眼楚风眠的境界，眼神中国都是露出了几分吃惊。
<br/>　　他们本以为楚风眠掌控了天星河，至少也是达到了半步天人境的境界才行。
<br/>　　却没想到，楚风眠只是九劫古帝。
<br/>　　一下子让他们心中的紧张，都是变的轻松了不少，看向楚风眠的一道道目光之中，都是无比轻视，不屑。
<br/>　　楚风眠才不过是九劫古帝，那就是得到些奇遇，才掌控了天星河，这种实力，在他们七位半步天人境强者的面前，根本不值一提。
<br/>　　“没想到我们闭关这么久，圣龙宗倒是出了一个妖孽，敢毁灭执法堂，踏平悟剑堂，万龙堂，现在更是攻打四宗，这种事，圣龙秘境数万年都未曾有人做过。”
<br/>　　“小辈，过刚易折，小小年纪，如此狂妄，只怕不是好事。”
<br/>　　那圣院宗的大长老，王长老也是仔细的打量了楚风眠一眼。
<br/>　　想要看看这大闹了圣龙秘境的人，到底是什么样子。
<br/>　　王长老开口对楚风眠的语气，仿佛便是在教训一个晚辈罢了，好像就是在看着一个小家伙的打闹一样。

<br/>　　楚风眠也听出了这王长老话中的轻视，冷笑一声开口道。

<br/>　　听到这话，突然传来一声怒吼，那六绝宗宗主，韩圣怒声吼道。
<br/>　　“怎么？不信吗？”
<br/>　　楚风眠看向韩圣，大手一挥，突然一团光芒浮现而出，正是灵泉，六绝宗地底之下的灵泉。
<br/>　　看到这灵泉的一刻，韩圣的眼神都是变的无比冰冷，露出无边杀意。
<br/>　　楚风眠就连六绝宗的灵泉都给收取了，那足以证明现在六绝宗，的确已经灭亡了。

<br/>　　这六绝神拳打出的一刻，天星河都在震荡，似乎是要被撕裂开来，强横的力量，刹那间笼罩在楚风眠的身上，向着楚风眠轰击而来。

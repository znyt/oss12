<br/>				一晃眼，回到云州已经三天时间。<br/>　　<br/>　　而在这三天的时间里，叶尘除了偶尔会和苏岚聚一聚，回了学校一趟，其余时间基本上闭门不出，一直潜心修炼，修为也逐渐提高，已经达到炼气八重中期。<br/>　　<br/>　　这天傍晚时分，唐业忽然来了，而且还带来一名四十岁上下的中年男子，这人的容貌，和唐清雅的父亲，倒是有几分相似，只不过一身军装，气质也更加锐利，肩头两杠三星，是一名上校，只不过军服有些奇怪，似乎不属于任何一种军种。<br/>　　<br/>　　“叶先生，这是我的小儿子唐明飞！”<br/>　　<br/>　　唐业笑着对叶尘介绍道。<br/>　　<br/>　　叶尘顿时恍然，<br/>　　<br/>　　唐业有三个儿子，大儿子唐明远从政，二儿子唐明志经商，三儿子据说在军中发展，而且好像还是在一个神秘部门，只是很少能回家，所以叶尘倒是第一次见。<br/>　　<br/>　　“清雅是我同学，那我就喊你唐叔叔吧！”<br/>　　<br/>　　唐明飞连忙道：<br/>　　<br/>　　“别！叶先生之名，如今响彻华夏，明飞可不敢造次！”<br/>　　<br/>　　叶尘也不强求，<br/>　　<br/>　　“不知道两位前来，有何贵干？”<br/>　　<br/>　　唐业和唐明飞对视了一眼，笑道：<br/>　　<br/>　　“此事可能涉及军机，老头子就先回避了！”<br/>　　<br/>　　说完冲着叶尘拱了拱手，便直接转身朝外面走去。<br/>　　<br/>　　唐业一走，唐明飞也霍然站起身来，眼眸中透着兴奋道：<br/>　　<br/>　　“听闻叶先生一掌灭杀了白千行，不知道此事是真是假？”<br/>　　<br/>　　叶尘点了点头，<br/>　　<br/>　　“白千行确实是被我一掌轰杀，你有何事，尽管明言！”<br/>　　<br/>　　唐明飞道：<br/>　　<br/>　　“叶先生快人快语，我也就直说了！”<br/>　　<br/>　　说到这里，唐明飞微微一顿，接着又道：<br/>　　<br/>　　“叶先生可曾听说华夏神龙卫？”<br/>　　<br/>　　叶尘摇了摇头，表示不知道。<br/>　　<br/>　　唐明飞连忙解释道：<br/>　　<br/>　　“其实以叶先生之能，应该很明白，这世上高人辈出，有修武者，有修道者，甚至西方国家，还有依靠科技变强的超能者，乃至天竺国那样精神修炼者等等！”<br/>　　<br/>　　“这些人的力量，远远超过普通人，只不过为了维护社会安定，全世界所有国家，都十分默契的，将这些人的信息尽可能的弱化，媒体也不会报道，所以普通人知之甚少……”<br/>　　<br/>　　“譬如世界第一强国的米国，就有专门的机构，负责管控这方面舆论，有的时候甚至要用药物消除知情者的记忆！”<br/>　　<br/>　　听完唐明飞的解释，叶尘频频点头，虽然前世他在国外待过，也曾经接触过唐明飞口中所说的超能者，譬如之前血影天罚的杀手，服用的狂化药水，其实就属于那一类人的手段。<br/>　　<br/>　　只不过，他所知道的，都是冰山一角，远远没有唐明飞说的这么系统。<br/>　　<br/>　　很显然，他应该经过相关的机密培训，所以才能对全世界的情况了解的如此透彻。<br/>　　<br/>　　只听到，唐明飞继续道：<br/>　　<br/>　　“自古以来，侠以武犯禁！为了维护社会的安宁，世界上所有国家机构，在面对这种问题上，几乎都会做出一个相同的选择，那就是把这些，远远超越普通人的力量聚集起来，成为国家的守卫者，而不是麻烦制造者！”<br/>　　<br/>　　唐明飞说到这里，见叶尘的表情平静，没有任何不悦之色，这才继续道：<br/>　　<br/>　　“譬如米国有超能局，西方联盟有教廷审判所，岛国有精国社……而我们华夏则是神龙卫！”<br/>　　<br/>　　叶尘听完唐明飞的解释之后，隐约已经猜到了唐明飞的来意，不过还是眉头微微一挑，<br/>　　<br/>　　“你为什么要和我说这些？”<br/>　　<br/>　　唐明飞微微一笑，<br/>　　<br/>　　“叶先生是聪明人，应该明白我的来意，我其实就是代表神龙卫的，来邀请先生加入的！”<br/>　　<br/>　　叶尘似乎不觉得惊讶，淡淡一笑，道：<br/>　　<br/>　　“让我加入你们，那对我而言，又有什么好处？”<br/>　　<br/>　　唐明飞见叶尘没有一口拒绝，顿时精神一震，连忙道：<br/>　　<br/>　　“我虽然只是神龙卫的外围编制，这次也只是负责打个头阵，询问一下叶先生的意向，不过我可以很负责的说，以叶先生足以斩杀白千行的实力，只要你愿意加入神龙卫，原则上任何条件都可以提！”<br/>　　<br/>　　叶尘听到这话，不由得眉头一挑，<br/>　　<br/>　　“那如

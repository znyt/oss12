
<br/>　　<br/>　　    王惊蛰走了，片刻之后就留下了一道萧瑟的背影，步子走出了义无反顾的风采，颇有风萧萧易水寒壮士一去不复返的架势，李红拂目送他出了墓崖村，直到人不见了都没有在动。<br/>　　<br/>　　    余生婆在她后面，后然说道：“不光是春暖花开，动物会到了交配的季节，有时夏意盎然也会勾引起生物最原始的冲动”<br/>　　<br/>　　    李红拂回头，皱眉的问道：“你在说什么？”<br/>　　<br/>　　    余生婆笑的挺阴的说道：“红拂，你发春了……”<br/>　　<br/>　　    李红拂当即一愣，随后怒声说道：“为老不尊！”<br/>　　<br/>　　    王惊蛰从墓崖村出来的时候，已经是下午了，天色还有三个多小时就黑了，他说的时间正好是因为，要去八拐里坡你得赶在子时的时候过去才行。<br/>　　<br/>　　    从墓崖村里出来，他一直往西，翻越了一座山头后就看见山下一条大河蜿蜒而过，那就是澜沧江了。<br/>　　<br/>　　    澜沧江旁都有摆渡的人，王惊蛰要去的却不是江对岸，而是要顺流而下顺着河水在往西走，直到进入八拐里坡的地界。<br/>　　<br/>　　    王惊蛰来到澜沧江旁的时候，河边只有两条竹筏在，两个穿着蓑衣带着斗笠的中年正闲坐在岸边上聊着天，他俩看见王惊蛰过来后，连忙起身问候了一句：“小哥，要过江么？”<br/>　　<br/>　　    “江不过，往下游走”王惊蛰大概估算了下距离，然后说道：“有二十多里远吧，天黑之前能到了就行”<br/>　　<br/>　　    两个船家一听他的话，顿时就惊诧了，其中一人说道：“你去那里干嘛，那个地方方圆几十里都没有村子人家，这个时候过去了你恐怕想回来就得要明天了”<br/>　　<br/>　　    王惊蛰指了指自己的一身长袍，解释道：“采药去的，今晚就要在那边歇息一夜了，明一早天还没亮，趁着露水没散我得要采几株草药，我要的药材太阳升起来就不好采了”<br/>　　<br/>　　    船家摇头说道：“我劝你最好还是换个地方吧，那里不好去的”<br/>　　<br/>　　    王惊蛰从长袍下面拿出一叠钱，数都没数的就递了过去：“我要的草药只有那一片才有，别的地方不行，师傅你就送我一趟吧”<br/>　　<br/>　　    船家一看他手里掐着的一把钱，就皱眉说道：“小哥，老实跟你说那个地方不太平啊，我们这些撑筏子的天黑后是从来不会往那边去的，我是为你好才不让你去的”<br/>　　<br/>　　    “啪，啪”王惊蛰拍了拍手里的钞票，说道：“有钱还不赚么船家？老实跟你讲，那个地方我也不是第一次去了，也没什么可怕的，这么跟你说吧，要不是今天进山累到了，我就顺着河岸自己往下走了，所以啊你不带我过去，我也能去，无非就是费点腿脚罢了，怎么样？这钱你赚还是不赚啊？”<br/>　　<br/>　　    这人还要再劝，另外一个船家上前一把就抓住了他手里的钱说道：“去，去，怎么就不去呢，上我的船吧”<br/>　　<br/>　　    那船家怒声说道：“这是上了你的贼船，王老五那个地方你又不是不知道，送他过去，岂不是要送去一条人命了？”<br/>　　<br/>　　    王老五笑嘻嘻的说道：“老哥你可别闹了，这小哥都说他去过一次了，他不比咱们懂啊，别劝了，人家心里有数着呢”<br/>　　<br/>　　    船家叹了口气，摇了摇头后就走向一边也不劝了，王惊蛰上了王老五的竹筏，他一撑竹竿喊道：“起船，走了……”<br/>　　<br/>　　    岸上的船家目送竹筏远去，忽然朝着地上就跪了下去，口中念念有词的说道：“山神，土地在上，保佑保佑吧！”<br/>　　<br/>　　    竹筏顺流而下，这澜沧江的水流比较湍急，一路上颠簸个不停，水花将他的长袍都给打湿了，他坐在竹椅上翘着腿，挺随意的问道：“船家，我要去的地方这是有什么说法啊，怎么那个人说什么都不去呢？”<br/>　　<br/>　　    王老五诧异的问道：“你不是说你去过了么？”<br/>　　<br/>　　    “去过也是几年前了，有好些时间没去采过药了”<br/>　　<br/>　　    王老五寻思了一下，顿了顿后才叹了口气，说道：“我今天要不是没接上一份活，我也不带送你过去的，这么和你说吧，那个地方我们附近的船家是没人愿意去的，特别是到了晚上，你啊要是来的在晚一点给我多少钱我也不会过去的”<br/>　　<br/>　　    王惊蛰笑呵呵的问道：“咦，那里闹鬼啊？”<br/>　　<br/>　　    船家忽然打了个哆嗦，缩着脖子说道：“可不就是闹鬼么，那里一到夜深人静的时候总有鬼哭狼嚎的动静冒出来，一夜一夜的都不散啊，还总有些鬼影冒出来，离着老远就能看到了，我们祖辈在江上撑竹筏的人都说，天一大黑了之后那片地方不管谁去了都是有去无回的，听说是那里是鬼门关，有好些个妖魔鬼怪盘踞着呢，几年前夜间的时候我曾经撑船路过一次，当时天也黑了，我刚过那天地方就听见岸上林子里有动静传来，你猜怎么着？嚯，我回头一看，好几个青面獠牙的小鬼正在岸上朝我跑来呢，我赶紧支着杆子就往下游走，吓的我一身冷汗都出来了”<br/>　　<br/>　　    王惊蛰“哦”了一声，点头说道：“那是真挺吓人的了”<br/>　　<br/>　　    船家小心翼翼的问道：“你该不是真要过去吧？”<br/>　　<br/>　　    “去看看，我这人天生胆大的很，什么都怕就是不怕鬼神……”<br/>　　<br/>　　    听王惊蛰这么一说，王老五也不再劝了，只是跟他说道到了地方后就尽量在岸边呆着别往林子里走，等到明一早天快亮的时候在过去吧。<br/>　　<br/>　　    二十多分钟后，竹筏顺流往下游走，就到了两山相夹的一处河段，王惊蛰顿时站了起来，背着手看着右边的一处山坡，此时天色刚要转黑，周边山林都是灰蒙蒙的，但唯独那处山坡显得更外漆黑了一点。<br/>　　<br/>　　    王老五撑着竹筏靠近河岸，然后跟他说道：“小哥到地方了，你这就下船吧”<br/>　　<br/>　　    竹筏到了岸边，王惊蛰没等靠岸就从竹筏上跃了过去，落到岸上后头也不回的说道：“明一早你过来接我，赚的还是我刚才给你的那些钱”<br/>　　<br/>　　    王惊蛰说完也不等王老五搭话，就从岸边往山坡上走了过去。<br/>　　<br/>　　    王老五瞠目结舌的开着他走远的身影，说道：“明早一来，我还能在看见你不成？”

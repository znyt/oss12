　　看着楚风眠一下子将那些龙党弟子，全部斩杀，在场的种子弟子，甚至是许许多多隐藏的长老，都极为震惊。

<br/>　　他的大手一挥，也是同样化为了一道龙爪。

<br/>　　在北邙学院之中，一直以来龙族武技的第一人，便是龙未央，据说这龙未央，可并非是得到了龙族奇遇的存在。

<br/>　　看到楚风眠同样凝聚龙爪轰击过来，龙未央冷笑起来。
<br/>　　“我可是有着龙王血脉！一切龙族，都要受到我的威压！龙王真身！给我出！”
<br/>　　龙未央的背后，一尊龙王虚影突然浮现出来，逐渐他的身躯，便是也化为了这龙王的模样。
<br/>　　如今一下子便是变的跟一尊真正的龙王一般，万龙之王！
<br/>　　一股股的龙力，凝聚在他的龙爪之上，一爪落下，简直是要将周围的一切空间，都要压碎。
<br/>　　“龙王真身！这龙未央的传闻并不假！他得到的并非是龙族奇遇，而是他的父亲，是一尊真正的龙王，他与生俱来便是生有这超越真龙的尊贵血脉！”
<br/>　　一些北邙学院的种子弟子，看到龙未央的变化，也是惊呼一声说道。
<br/>　　龙未央的来历，一直以来都是在传闻之中的，如今他这还是第一次，真正展现出自己的实力来。
<br/>　　这龙王的身躯，便是毫不掩饰的，将龙未央的底牌，尽数爆发出来。
<br/>　　“这楚风眠的实力，的确强大无比，不过他的奇遇，却是龙族奇遇，面对龙未央，天生被克制，他的实力本来就不如龙未央，实力多半依靠都是奇遇，失去了这龙族奇遇的优势，这楚风眠必败无疑。”

<br/>　　那剩下两位核心弟子，看向楚风眠，听到古傲寒的话，也是点了点头。
<br/>　　楚风眠最为依赖的手段，都被龙未央克制住。
<br/>　　似乎楚风眠，已经是并没有更多的手段，足以是跟龙未央抗衡了。

<br/>　　龙未央冷笑连连，看向楚风眠的眼神中便是自信满满，仿佛已经是将楚风眠死死的吃定了。

<br/>　　这龙未央拥有龙王血脉，对寻常龙族，的确是极为克制。

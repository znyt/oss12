        笑起来的样子，像阳光一样灿烂，又有如沐春风般的温暖。
<br/>
<br/>        “我叫……”男人刚想向童筱颖介绍自己的名字，但是童筱颖却没有想要跟他认识的意思，而是直接转身离去。
<br/>
<br/>        童筱颖并未把那个男人放心上，反倒是走进了那边的人群里，和同在一家民宿的旅客们一起看了一会儿海面上的海市蜃楼奇观，才和他们谈笑风生地离开。
<br/>
<br/>        回到民宿里后，童筱颖不由自主地拿起了手机，翻到傅夜沉的手机号码时，不知不觉地发起呆来。
<br/>
<br/>        这个自称是她老公的男人，至今都没有主动给她打过一个电话。
<br/>
<br/>        包括她那次去威尼斯旅游的时候也一样。
<br/>
<br/>        其实，她表面上走的很潇洒，但心里却总是在无意间地想到他。
<br/>
<br/>        童筱颖不明白，她和他之间到底出了什么问题，明明一开始他对她是那么温柔体贴，但现在他又突然间与她生疏了。
<br/>
<br/>        待到夜幕降临之际，民宿一楼的清吧也跟着热闹起来。
<br/>
<br/>        来清吧里的人，除了在这住宿的旅客以外，还有一些附近的当地人，大家聚在一起大多只是喝酒聊天。
<br/>
<br/>        这里没有聒噪的音乐，所以童筱颖住楼上只要把房门关了便不会觉得楼下很吵。
<br/>
<br/>        此刻，她正坐在露台上吹海风，门被人叩响后，她才起身去开门。
<br/>
<br/>        “筱颖，楼下有个男人拿着你的照片在找你，他说他是你的前男友，找你有很重要的事。”来叩门的人是民宿的老板娘，她说着语速比较缓慢的英文，好方便跟童筱颖沟通。
<br/>
<br/>        童筱颖一脸懵然地回答道：“我没有前男友。”
<br/>
<br/>        “你要不下去看看，他看起来不像是个坏人。”老板娘接着说道。
<br/>
<br/>        童筱颖点了点头，从屋内出来，并随手带关了房门。
<br/>
<br/>        在老板娘的带领下，童筱颖去了一楼角落里靠窗的那个位置。
<br/>
<br/>        木质高脚圆桌前，已经有个穿着白色休闲装的男人坐在那里了。
<br/>
<br/>        童筱颖走过去后，看到这个男人的面孔时，顿时感到诧异：“怎么是你啊？”
<br/>
<br/>        “原来你真的住在这里。”男人温文儒雅地微笑着，并将放在圆桌上的一束满天星递向了童筱颖，“这是送给你的见面礼。”
<br/>
<br/>        童筱颖并未伸手去接这男人的花，而是很纳闷地反问：“你哪来的我的照片？”
<br/>
<br/>        这个男人就是起初在沙滩上抢走她的紫贝壳的那个男人。
<br/>
<br/>        说来也奇怪，她对他的第一印象并不好，哪怕他没有跟她抢紫贝壳。
<br/>
<br/>        “我拿烟儿的照片，寻来的。”男人一边说着，一边掏出皮夹，展开来，举到了童筱颖的眼前。
<br/>
<br/>        皮夹里放着一张跟她长得很像的女孩的照片，女孩长发披肩，素颜，眉清目秀，嘴角微微上扬。
<br/>
<br/>        “烟儿是你的谁？”童筱颖下意识地问。
<br/>
<br/>        男人顿时微微垂下眼帘，深沉道：“她是我的前女友。”
<br/>
<br/>        “那她人呢？”童筱颖接着问道。
<br/>
<br/>        男人忽然心情沉重地回答道：“去了天国。”
<br/>
<br/>        “……”童筱颖怔了一下，连忙说了声，“对不起。”
<br/>
<br/>        “没关系。”男人微微抬眸，看向童筱颖，会心而笑，“我真的没想到，我还能遇到一个跟她长得很像的女孩。”
<br/>
<br/>        “那个……我有老公的，你不要对我抱有幻想哈！”童筱颖心直口快地说，总觉得这男人接近她的目的，只怕真的是为了怀旧。
<br/>
<br/>        男人愣了一下，完全没料到面前的这个女孩已经结婚了，目光下意识地落在她搭在圆桌的双手上。
<br/>
<br/>        童筱颖见他盯着自己的手指，不明所以然的时候，男人笑着说道：“你是第二个让我很想亲近，却是第一个如此直白地拒绝我的女孩。只不过，你这理由，倒是编得很有趣。没关系，我们可以先从朋友开始。”
<br/>
<br/>        “你要是没别的事了，我就先回房间了。”童筱颖不太想跟这个男人有过多的交谈。
<br/>
<br/>        男人看出了童筱颖对自己的抗拒，反而落落大方地自我介绍道：“我的中文名叫秦烈，今年三十一岁，未婚单身，现在是一家上市公司的老板。”
<br/>
<br/>        “你有没有听过傅氏集团？”童筱颖双手抱臂，撑在圆桌上，朝秦烈凑近了些，咧嘴问道。
<br/>
<br/>        秦烈会心一笑：“当然听过，三年前在美国上市，现在已经是个大型的跨国企业了。”
<br/>
<br/>        “那你知道傅氏集团的幕后大老板是谁吗？”童筱颖眨巴着眼睛，嘴角微扬。
<br/>
<br/>        秦烈笑了笑，回答道：“听说傅氏集团实行的是长子继承制，上一任继承者在下一任继承时可以享受百分之三十的股份，所以，傅氏集团幕后的大老板一定是姓冷。”

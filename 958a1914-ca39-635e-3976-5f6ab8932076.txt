　　吃土？若说别人吃土李承乾或许相信，不过李道宗说自己吃土，就算打死李承乾他都不会相信。
<br/>　　那个所谓的新华书局别的不说，每年光印刷大唐旬报之类的官方报纸就不知道赚了多少，更不要说还有一些其他书籍的印刷，贩卖。
<br/>　　虽然眼下印书的价格与从前相比缩水近十倍，但是由于大量机械的投入，现在的印刷坊除了排版与校对，基本上已经很少有人工参与，所以相对来说还是比较赚钱的。
<br/>　　而且不但如此，李道宗也还有一些其他的产业，每年他的收入基本上都保持在近五万贯左右，也就是说每月再怎么低也有四万多贯的进帐。
<br/>　　这已经是一笔不小的费用了，所以李承乾才不相信李道宗这老家伙会吃土。
<br/>　　可是不相信归不相信，这些话他也只有憋在心里，无奈的看了这个老家伙一眼并没有理他，而是转向杜如晦：“杜老，您怎么看？”
<br/>　　“陛下，俸禄的确是该涨了，事实上不只是基层官员，有些时候就算是顶级勋贵，家里如果没有其他产业支撑，这些俸禄也只够大家勉强维持温饱。”如今的杜如晦虽然还能坚持着上朝，但身体也已经大不如前，仅仅一句话，便停顿了不下三次。
<br/>　　李承乾看着老杜的样子，心中也是老大的不舒服，眼中露出一丝关切说道：“杜老，您的身体不要紧吧？要不要我把杜构或者杜荷调回来？”
<br/>　　“不必了，陛下的关心老臣心领了，可是年轻人总有年轻人的事业，不能让他们为了我一个老头子耽误了前途不是。”杜如晦笑了笑说道。
<br/>　　不管是李承乾对他的称呼，还是自称，老杜都看在眼里，听在耳中，能得一代帝王如此尊敬，老杜头儿已经心满意足。
<br/>　　可是杜家的未来还是要靠两个儿子来支撑，现在调他们回来固然是全了他们的孝心，但却很可能会耽误了杜家的未来，所以不到万不得以，老杜是真的不想让自己的两个儿子回来。
<br/>　　李承乾也明白杜如晦的意思，见他坚持也没有什么好办法，只能叹了口气说道：“那就这样好了，一会儿让医学院那边派一个医疗小组，由一位太医负责，到您府上常驻。”
<br/>　　“这……”杜如晦没想到李承乾竟然能为他做到这一步，一时间竟不知道说什么才好。
<br/>　　李承乾见场面似乎有些尴尬，也不等老杜说完，便接口说道：“您放心，这份开支由朝庭出，不用您的俸禄！”
<br/>　　此话一出，房间中立刻笑声一片，刚刚有些沉重的气氛缓和了不少，老杜再也无法拒绝，只能讪笑着拱拱手，算是谢过李承乾对他的关心。
<br/>　　不过你还别说，由老杜这件事情，李承乾还真的有了一个想法。
<br/>　　眼下不少老头子那一代的老家伙都已经老了，身体大不如前，正是需要有人照顾的时候，自己既然能派人去照顾老杜，那为什么不同样派人照顾其他人呢？
<br/>　　这样一来既可以让这些老家伙们的子女们放心，又可以拉拢人心，何乐而不为呢。
<br/>　　想到这里，李承乾心中已经有了决定，只等这次关于“俸禄调整”的议题一过，便把这件事也提到日程上来，制定一个比较完善的计划。
<br/>　　“陛下，其实如果单说俸禄，咱们大唐的俸禄还是不低的，可是当初在制定标准的时候，有些事情并没有考虑完全。”房玄龄等了一会儿，见气氛缓和了，这才委婉的说道。
<br/>　　“哦？房老能仔细说说么？”李承乾把思路收回来，认真的问道。
<br/>　　“其实是这样，我们这些人其实家中并不只有妻儿老小，还有一些管家、家将、仆役、侍女之流，而这些人也是要领工钱的，家族越大，这一类人就越多，所以就把俸禄给分薄了。”房玄龄说的很慢，就好像是怕李承乾听不明白一样。
<br/>　　不过好在李承乾的智力并不弱，很快就理解了他的意思，皱着眉想了半晌才说道：“那如果这样说来，其实这个俸禄标准还是很难制定的，毕竟有的人家中家将之类的人数少一些，有的人家中家将之类的人多一些，对吧？”
<br/>　　“正是如此，所以臣等才会为难。”房玄龄点头说道，而书房中的其他人也跟着点头，表示认同他的说法。
<br/>　　兵部尚书作为军方一系的代表，此时也开口说道：“是啊陛下，其实这是一件让人很为难的事情，而且那些家将……那些家将其实大多数都是以前跟着臣等从尸山血海中杀出来，臣等也没有办法不管他们，无奈只收入帐下，这也是客观存在的事实。”
<br/>　　“是啊，当初他们也是为了大唐血过流，流过汗的，现在老了总不能就不管他们。”于志宁也在边上跟着叹了口气，显然他家里也有不少这样的人。
<br/>　　李承乾面对这样的问题也有些尴尬，他以前可从来没有考虑过这样的事情，现在突然提出来了，也让他很是为难。
<br/>　　你说不涨俸禄吧，眼见现在物价越来越高，一些高端产品也越来越多，不涨总不是个事儿；可你说涨俸禄吧，又每个人的家庭条件都不一样，不可能作到一视同仁。
<br/>　　讨论就这样再次陷入僵局，所有人都在低头沉思，但却没有人开口说一句话。
<br/>　　“咳咳，陛下，老臣倒是有个想法，只是多少有些冒犯，不知当讲不当讲。”终于，还是杜如晦首先开口了，出于对李承乾知遇之恩吧，这老头儿决定再帮他一次，就像魏征用最后的一口气帮助他登上帝位一般。
<br/>　　李承乾见杜如晦开口，立刻笑着说道：“杜老请讲，现在讨论的是公事，何来冒犯之说。”
<br/>　　“嗯，老臣觉得，朝庭如果肯接收那些已经不能再上战场的家将，或许这件事情还是很好处理的。”
<br/>　　“愿闻其详！”李承乾一脸认真的说道。
<br/>　　“老臣的意思是这样，比如制定一个标准，国公家中最多有多少仆从，国候家中最多有多少仆从，这样一来大家便都站到了同一平台。而多出来的那些人，则要辞退，如果是于国家有功的，朝庭可以给予补偿，而那些身强力壮，有自力能力的，则可以让他们自己务工！”
<br/>(本章完)

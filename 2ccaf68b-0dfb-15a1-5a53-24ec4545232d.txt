　　然而，就在所有人都认为一切都结束了的时候，突然一声惨叫，从一个士兵的口中发出，大伙急忙转头看去，一个士兵惨叫着，腿上挂着一个人，头埋在在小腿肚子上，牙齿狠狠地咬着一块肉，咬的鲜血淋漓。
<br/>　　仔细一看，咬他的那个人根本就不是活人，而是之前被小三枪杀的二蛋，大伙只顾着注意前面那三个死者，却忽略了还有这个死人的存在。
<br/>　　小三冲过去，步枪对着二蛋的脑袋就是一枪，将脑袋轰碎，然后也不仔细看，高声喊叫起来：“大家快撤！这地方有古怪，快撤！”
<br/>　　说完也顾不上外面还在下雨，率先冲了出去。
<br/>　　吴连长等人紧随其后，逃命似的往外狂奔，那那个一直押着翠云的士兵也放下了她，这节骨眼上，谁还顾得上抢花姑娘。
<br/>　　他们都是当过兵杀过人的，又立志要去当土匪的，基本上算是天不怕地不怕，但也真的怕鬼，方才遇到的事情已经大大挑战了他们的底线，结果自然是落荒而逃亡。
<br/>　　“连长，我……”那个小腿被咬的士兵追赶出去，结果只换回吴连长的一枪：之前那几具尸体连续变成僵尸，给吴连长等人带去了心里阴影，生怕这个被僵尸咬的士兵也会变成僵尸，一个追着他们，干脆就一枪解决了。
<br/>　　士兵们奔跑的声音逐渐远去，外面只剩下了雨声。
<br/>　　山神庙里恢复了安静。
<br/>　　老万和陈毛子早就被吓呆了，浑浑噩噩地瘫坐在地上。翠云反而是第一个回过神来的，扑到叶少阳的身上，抱起他的脑袋，刚要开口叫他的名字，叶少阳睁开了眼睛，重重地吸了口气，冲她笑了笑。
<br/>　　翠云心中一块石头落了地，抓着他的双手，说道：“是你捣的鬼是不是？”
<br/>　　叶少阳点了点头，然后虚弱地闭上眼睛。
<br/>　　方才那段时间，他并不是睡着，而是灵魂出窍，依次附在几具尸体上，操控尸体，做出了那一系列让人胆寒的恐怖的行动，最终把那一干土匪吓走。
<br/>　　魂魄离体，对从前的他来说实在不算什么，但现在他只有以往十分之一的法力，这一连串作法，也是累的够呛，当下躺在翠云的腿上，大口喘气，累的爬不起来，不过幸好结局很好，唯一美中不足的是，那个阴险的名叫“小三”的家伙没死，活着逃了。
<br/>　　在他们最初讨论真的要去做土匪的时候，叶少阳就知道，自己这几个人性命不保了：这群立志要去当土匪的逃兵，绝对不会让见证了他们从逃兵转变成土匪这个过程的自己几人留下活口，当时想来想去，也是没什么好的办法，他虽然会茅山体术，比拳脚的话，对付这十几个士兵应该没问题，不过，人家有枪，一人手上一支枪。叶少阳不敢轻举妄动。
<br/>　　枪声一响，就算自己能躲得过，翠云他们几个也会跟着冒险。当叶少阳目光落在那几具被打死的士兵身上时，顿时就有了主意，于是跟翠云耳语，告诉她自己会昏睡过去，让她不要紧张，而且不要惧怕这些士兵，自己有办法对付他们。
<br/>　　在魂魄离体之后，叶少阳灵机一动，想附身在那个小三身上，借用他的枪打死同伴，结果他身上有一块灵力十分强大的灵物，护佑着他的身体。叶少阳试了一下，短时间内没法附身，为了大局考虑，也只好放弃了，按照计划附在了那些死尸的身上……
<br/>　　一切，都是他的表演。
<br/>　　叶少阳拉着翠云的手，笑道：“没事了，土匪都跑了。”
<br/>　　翠云用力抱住他，“少阳，你可吓死姐了！”
<br/>　　叶少阳笑道：“是不是以为要被抓去做压寨夫人了？”
<br/>　　翠云点了点头，在那种情况下，他根本想不到有什么办法能逃脱那些土匪的魔掌，当时心中一片惨然，没想到……叶少阳竟然做到了。
<br/>　　“我说过没事，就一定不会有事。”叶少阳拍着她的手背，坚定地说道。
<br/>　　翠云笑了笑，“好啦，姐相信你，将来再有什么说，你说什么我都听着。”
<br/>　　叶少阳躺在她怀里，叹了口气说道：“可惜让他们给跑了，我本来想着至少杀他们一半人，结果一个都没死。”
<br/>　　翠云道：“他们跑了就好了啊，至少我们都没事，万老爷子他们也没事。”
<br/>　　两人想到什么，一起转头朝二蛋的尸体望去，翠云想起二蛋之前对自己所做的那些事，作为同村的老乡，居然出卖自己，之后惨死，也算是咎由自取，不过怎么说也是老乡加同行的伙伴，翠云看到二蛋惨死，还是有些伤感，叹息不已。
<br/>　　叶少阳刚要劝她几句，翠云突然怔住，随后趴在他身上，哇的一声哭了起来。这一哭倒是吓坏了叶少阳，以为她是同情心泛滥，毕竟二蛋跟她是同村老乡，就怎么看着他惨死，可能一时间接受不了，于是拍着她的肩膀，说了很多诸如“人死不能复生”、“节哀顺变”之类的话。最终，翠云趴在他肩膀上，道出了让她痛哭流涕的真相：“金条被抢走了……”
<br/>　　叶少阳当场怔住，嘴角抽搐起来，敢情自己安慰了这么久，人家不是因为二蛋的死才难过啊？
<br/>　　金条没了……叶少阳讷讷地问道：“被抢走几根？”
<br/>　　“三根都抢走了，还有一包大洋，呜呜，我们没钱了……”
<br/>　　这真是一件悲伤的事情。叶少阳一只手捂着额头，甚至自己都想大哭一场了。
<br/>　　之前秋镇长一共给了四根金条，自己给了翠云三根，自己那根在镇上换成了现大洋，翠云得知之后，就让他先花大洋，至于给她的那三根金条，她也不要，帮叶少阳攒着，留着将来娶媳妇用。
<br/>　　因为翠云也打算走了就不回去了，因此三根金条和大部分的大洋，都被她缝在了裹住腰身的那条布带上，本来以为那才是最贴身最隐秘的地方，哪里想到……之前被几个逃兵抢了个干净，除了那三根金条，那个叫小三的当时看着布条的厚度和重量不对，知道里面有东西，当时也让士兵们抢走了。

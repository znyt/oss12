　　当叶尘再次恢复意识的时候，感觉自己的脸庞，贴在一个无比柔软的地方，而且散发着一股诱人的清香。
<br/>　　叶尘睁开双眼一看，首先映入眼帘的，竟然是一处雪白的沟壑~~
<br/>　　足足愣了好一会，叶尘这才反应过来，自己原来正躺在苏蔓的怀中。
<br/>　　“嗯~”
<br/>　　叶尘闷哼了一声，想要从苏蔓怀中起开，却发现浑身酸软无力，刚刚抬起头，便又再次重重跌落回苏蔓的怀中。
<br/>　　苏蔓此时也察觉到了动静，低头看到叶尘醒了，顿时大喜过望，
<br/>　　“小尘，你终于醒啦！你身上的伤势很重，别乱动！”
<br/>　　叶尘正打算坐起来，不料却又被苏蔓十分强势的按了回去。
<br/>　　叶尘顿时哭笑不得，
<br/>　　没想到他居然也有如此孱弱的时候，连苏蔓这个刚刚踏入修真的菜鸟都反抗不了！
<br/>　　不过话说回来，这其实也很正常，
<br/>　　他之前太过拼命，中了痛苦之源，心智大失，一口气运转到逆生百转的第五重！
<br/>　　倘若不是他，及时服下了两枚扶摇仙果，恐怕现在能不能醒过来，都是一个问题。
<br/>　　既然反抗不了，叶尘也不再动弹，索性心安理得的枕着苏蔓身前的那对饱满，开始观察四周的情况，
<br/>　　“蔓姐，我们这是在哪？”
<br/>　　叶尘发现，他们此刻正处身在一处，颇有古典韵味的房间之中。
<br/>　　苏蔓道：
<br/>　　“听小白说，这里叫太初城，你都已经昏迷一天一夜了！”
<br/>　　“太初城？”
<br/>　　叶尘顿时眉头一皱，
<br/>　　“这么说，我们现在还在蓬莱仙界？”
<br/>　　苏蔓点了点头，道：
<br/>　　“因为你受伤太重，所以我们赶路的速度有些慢，听小白的哥哥说，四大宗门已经发出悬赏令，要通缉我们几个，而且通往外界的传送阵，也增加了很多高手，我们想要离开，只怕很难！”
<br/>　　叶尘原本还想好好歇息一会，听完苏蔓这话，顿时便坐不住了，
<br/>　　“蔓姐，你快扶我坐起来，我必须要尽快运功恢复实力！”
<br/>　　苏蔓虽然极为不愿，可是见叶尘神色坚定，表情严肃，也只好如实照办。
<br/>　　片刻之后，
<br/>　　在苏蔓的帮助之下，叶尘盘腿而坐，开始进入修炼状态。
<br/>　　他如今虽然已经突破金丹境，可是还尚未来得及凝聚金丹，体内经脉几乎没有一寸完好，丹田之内更是空空如也，竟然一丝真元也无。
<br/>　　“这次受伤还真是够严重的！”
<br/>　　叶尘再次微微叹息了一声，当即从乾坤镯之中，取出了一堆药瓶，但凡是对治疗经脉，恢复真元又用的丹药，都吞了下去。
<br/>　　倘若是寻常修真者，肯定不敢这么干，可是叶尘身怀神级功法，自然无碍。
<br/>　　“吞噬！”
<br/>　　叶尘一口气吞下数十枚丹药之后，当即运转吞天神功，飞快的吞噬着丹药中的能量。
<br/>　　与此同时，叶尘体内的吞天鼎，也开始暗暗运转，将周围的天地灵气汇聚了过来。
<br/>　　“滋滋滋~”
<br/>　　叶尘全身的经脉，顿时便开始飞速的愈合，丹田内也很快汇聚了大量的真元。
<br/>　　“嗯？”
<br/>　　叶尘猛然发现，自己的修为突破金丹境之后，吞天神功的境界，也水涨船高！
<br/>　　“是了！我都差点忘了，修为每提升一个大境界，吞天神功也会随之精进，我如今不但吞噬能量的速度暴涨，甚至还可以直接吞噬其他修士的真元，来化为己用！”
<br/>　　这一发现，顿时让叶尘又惊又喜，当即将吞天神功全力施展开来。
<br/>　　果不其然，只用了十几分钟，叶尘居然就已经将数十枚丹药的能量全部炼化！
<br/>　　伤势也大为好转。
<br/>　　叶尘大喜之下，当即将手里的丹药，全部一股脑的吞了下去，然后又开始动用吞天神功炼化起来。
<br/>　　半个小时之后，叶尘体内的伤势，已经缓和了许多。
<br/>　　就在叶尘准备一鼓作气，凝聚出金丹的时候，门外忽然响起了敲门声，却是祝明飞和祝小白两兄弟回来了。
<br/>　　叶尘站起身来，随手一挥，房门便自动打开，
<br/>　　两兄弟走进屋内，看到叶尘已经醒来，先是纷纷一喜，继而又露出满脸震惊之色。
<br/>　　叶尘的伤势有多重，他们之前可是看在眼里的，甚至在祝明飞为叶尘诊断之后，断定他至少也要一年半载，才有可能恢复过来。
<br/>　　然而，这才短短半个小时，叶尘不但站了起来，而且气息浑厚，伤势已经明显大为好转。
<br/>　　这种情况，他简直闻所未闻，见所未见！
<br/>　　“叶大哥，你的伤……好了？”
<br/>　　祝小白冲了上来，满脸惊喜之色道。
<br/>　　叶尘微微一笑，
<br/>　　“哪有那么容易！也就恢复了三四成吧！”
<br/>　　说到这里，叶尘转而望向一旁的祝明飞，
<br/>　　“祝兄，如今外面什么情况？”
<br/>　　祝明飞听叶尘问起这个，这才从之前的震惊中回过神来，叹了一口气道：
<br/>　　“情况很不妙！据说太乙门宗主罗霄和逍遥派宗主青阳子，逃回宗门之后，已经联合了四大宗门，以及他们麾下的数十大门派，在整个蓬莱仙界发布了通缉令，如今我们四人已经成为蓬莱的头号公敌！”
<br/>　　“至于传送阵那边，四大宗门派出是十余名元婴境强者，同样防卫森严，我们想要离开蓬莱，只怕不太容易……”
<br/>　　听完祝明飞的描述之后，叶尘也不由得眉头紧皱，
<br/>　　倘若是在全盛之时，区区十几名元婴境高手，他自然不会放在眼里，直接冲杀出去即可。
<br/>　　可是如今，他伤势未愈，实力大减，就有些力不从心了。
<br/>　　祝明飞又道：
<br/>　　“叶兄弟，四大宗门已经开始派人搜城，早晚会搜到我们这里，我想来想去，现在也只有一个办法了……”
<br/>　　叶尘眉头一挑，
<br/>　　“你且说说看！”
<br/>　　祝明飞深吸了一口气道：
<br/>　　“我负责把看守传送阵的那些高手引开，叶兄弟你带着苏姑娘和舍弟，趁机逃离蓬莱！”
<br/>　　很显然，祝明飞打算用声东击西的办法，为他们争取时间。
<br/>　　祝小白也明白了哥哥的意图，小脸上顿时露出一抹担忧之色。
<br/>　　叶尘沉吟了片刻后，方才缓缓道：
<br/>　　“这确实是一个办法，只不过，负责引开那些高手的任务，必须由我来做，因为他们的主要目标是我，而不是你！”

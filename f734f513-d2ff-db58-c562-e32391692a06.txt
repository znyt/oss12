
<br/>　　<br/>　　    茅小草的话让凉绣觉得一阵胸里发闷，这肯定不是因为那啥太大给压的，而是因为面前这个她一直都认为呆呆萌萌的女人，完全有可能是在扮猪吃老虎，从一开始她就已经知道自己是在刻意接近她了。<br/>　　<br/>　　    四周的墨家死士已经听见了两人的谈话，至少五六个人全都围拢了过来，虎视眈眈的盯着小草，大有一言不合就动手的架势了，凉绣强自镇定的压下了心头的慌乱，呼吸有点急促的说道：“你知道了又能如何，你现在不是还在我的手里呢么？”<br/>　　<br/>　　    小草用手里的筷子戳了戳那盘鲤鱼，说道：“你看你是得有多蠢啊，我刚才不是说了在算计你么？既然都已经知道你刻意接近我，还心甘情愿的被你给掠走了，那能没有点故事么？故事啊就在这鲤鱼上了……店家做得还算可以吧，嗯，虽然比我的手艺还差了那么一点，但还勉强可以下咽呢，哪里会让人一吃就吐了出来？”<br/>　　<br/>　　    茅小草在说这句话的时候，就意味深长的望向了凉绣，她“唰”的一下脸色巨变，哆嗦着嘴唇说道：“你，下毒？”<br/>　　<br/>　　    “下毒？咯咯，多肤浅啊，你觉得有什么毒能下到让人神不知鬼不觉，潜伏了将近一个月都不发？你觉得……”小草忽然凑近凉绣，用手指刮了下她的脸，轻声说道：“你觉得又能有什么毒，可以让一个女人以十几二十倍的速度衰老下去，提前进入更年期，凉绣啊你洗澡的时候就没有发现，自己身上的皮肤松弛老化了很多么？唉，你都没发现自从遇到我以后，你的月事一直都没来？这还只是开始可以，再有一个月你那满头乌黑的绣花会变的粗糙，干枯，一点光泽都没有，下一个月你的脸上就开始会出现皱纹了，然后胸下也会慢慢的下垂干瘪，又过了一月，你的脸上就会彻底黯淡无光了，不出一年，你的芳华将会变成个五十来岁的老妇女，两年后不到三十岁的你看着就跟七老八十一样了，除非你肯自行了断，否则你将会眼睁睁的看着自己迅速老下去，啧啧，到时候你想想照镜子的时候，会不会给自己吓一大跳呢？”<br/>　　<br/>　　    凉绣茫然，慌乱，眼中顿时就毫无光彩起来，小草说的轻松，仿佛好像是在和她唠家常一样，但她说的每个字都好像插在了她的心头，一席话说完，就跟万箭穿心了差不多。<br/>　　<br/>　　    “不，这，这不可能，你，你在吓我，世上没有什么毒是会这个样子的……”凉绣无助的摇了摇头，苍白的吐出了一句，自己都不太相信的话。<br/>　　<br/>　　    “毒，自然没有，我也不会，可我下的是蛊呀？”小草用一种很鄙夷的眼神，望着她说道：“第一次吃的那条潍河鲤鱼是毫无问题的，但当我怀疑了你以后，再在河上吃鱼的时候，我就在锅里下了蛊虫，本来呢我也只是怀疑而已，如果你无心害我，这蛊自然无所谓了，可正好呢，你还真是有心接近我的，那这蛊虫就能一点一点的吞噬掉你的生命精华了，让你的精气神快快的耗费下去，直到你逐渐老死而已，再过几年啊除了你老掉以后，你恐怕连抬抬胳膊都难了，到时躺在床上的你，连大小便都失禁了，就像那些生活不能自理的老人一样！”<br/>　　<br/>　　    “你撒谎，这根本就不可能，你骗我，你根本就不会下蛊，你们天道峰也不可能会，这只有在黔南，苗疆一带……”<br/>　　<br/>　　    凉绣说到这里突然之间就顿住了，因为她记得当初在调查茅小草和王惊蛰的时候，小草其中有一段时间是失踪了的，两人最后出现的地方，就在黔西南的深山中。<br/>　　<br/>　　    可那时凉绣哪里会想到，就她失踪的那段时间，小草居然是去学蛊了？<br/>　　<br/>　　    她注意到了这个消息，但却偏偏没往这地方想，一步走错，满盘皆输，这是她唯一忽略掉的因素，却也成为了压死凉绣的一根稻草。<br/>　　<br/>　　    “你至少现在还在我手里，留住你，我不信你肯跟我同归于尽……”<br/>　　<br/>　　    凉绣颤抖着伸出手，朝着身后的死士刚要下令，茅小草鄙夷的眯着眼睛，说道：“你总是说这个呆傻，那个呆萌的，其实殊不知啊，真不怪王惊蛰说你，脑子让门夹的都进屎了，他为什么要把你约到滇西这边？首先无非是因为出了唐梁山，离开了你们墨家的机关大阵，不是主场作战你还是个啥？再一个呢就是，来到这里，我们会有更充足的时间来布置，比如你们家的那些死士，自然有人来对付他们”<br/>　　<br/>　　    “啪！”小草说完，忽然打了个响指。<br/>　　<br/>　　    鱼馆后厨方向，之前曾经来点菜的老板，背着手迈着四方步走了过来，温文尔雅，礼敬有加的冲着凉绣弯腰行了一礼，笑道：“自我介绍一下凉绣姑娘，孔家德容”<br/>　　<br/>　　    凉绣脸上的表情瞬间凝固，四周死士听闻孔德荣的话，顿时全都蓄势待发，谨慎的盯着他。<br/>　　<br/>　　    “孔家，孔府？是你们……”凉绣压着心头的怒火，咬牙说道：“真是没想到你们居然也掺和了进来，我也没想到王惊蛰会去向你们求援了”<br/>　　<br/>　　    “不是求援，是合作！”孔德荣扫了一眼那帮虎视眈眈的死士，说道：“我们不会下蛊，也不会下毒，孔府众中人还不屑于去干这种下三滥的勾当，所以我们下了蒙汗药……让你们墨家的这帮死士都老实点吧，再有几分钟这药劲就该出来了，到时别说是喊打喊杀的了，可能连打个哈欠都难了”<br/>　　<br/>　　    凉绣“腾”的一下就站了起来，吼道：“多少年了，孔墨两家都已经对当年的仇怨视而不见了，你这是还想要把两家的旧恨重新拉出来不成？”<br/>　　<br/>　　    孔德荣淡淡的说道：“要是你有这样的机会来针对我们孔府，恐怕你也不会错过吧……”

　　楚风眠占据圣子峰的消息，几乎一瞬，便是传遍了整个北邙圣域。
<br/>　　圣子峰，是圣党的老巢，如今楚风眠占据了圣子峰，便是也就足以代表着圣党的毁灭。
<br/>　　“这山峰的灵气，真是浓郁，我可以布置阵法，将其中的灵气击中到一起，在以一吞天神蟒，放入其中，每天都足以凝聚出大量的灵石来，作为财富！”
<br/>　　楚风眠感受这悟剑峰上的灵气，大手一挥。
<br/>　　一座巨大无比的阵法，突然笼罩在其中，所有的灵气，都是聚集在了其中一点。
<br/>　　随后楚风眠身上的剑意凝聚，随着他手指一点，其中一缕，直接分化出去，这一缕剑意，便是化为了一条吞天神蟒，就在这灵气之中，不断的吞噬这。
<br/>　　远远不断的吞噬这其中的灵气，等到凝聚到临界点，便是就足以吐出来一枚灵石。
<br/>　　足以给楚风眠带来源源不断的财富。
<br/>　　这圣子峰，曾经是足以支撑整个圣党之中，无数圣党弟子修炼，其中的灵气到底有多浓郁，不言而喻，现在被楚风眠一个人占据，足以得到大量的好处。
<br/>　　现在楚风眠拥有血海，虽然说财富多的吓人，但是这毕竟还是楚风眠并未度过生死大劫。
<br/>　　一旦是度过生死大劫，楚风眠也就需要源源不断的灵力，来支撑他的境界，这血海的财富，都不一定够，楚风眠必须要继续积攒他的财富才行。
<br/>　　“轰隆！”
<br/>　　就在楚风眠布置完这一切之后，一声巨大的轰鸣声，猛然暴起。
<br/>　　只见这悟剑峰上面的禁制，都被直接打破，整个悟剑峰，都在颤动这。
<br/>　　两道身影，从这虚空之中突然出现，直接便是撼动这悟剑峰。
<br/>　　其中出手的那人，是一尊白发老者，正是曾经在武斗大会上，对楚风眠出手，差一点轰杀了楚风眠的长生之境的大人物，夜牧老祖。
<br/>　　“小畜生，你犯下了如此罪孽，还敢回来！”
<br/>　　夜牧老者看着楚风眠，心中的怒火，便是已经无法忍受。
<br/>　　“你残杀同门，丧心病狂，已经是一尊魔头了，你回到学院，正好，我就杀了你，来为学院，铲除一害！”
<br/>　　夜牧老祖，毫不犹豫，手中直接出现了一把青色长刀，猛然一刀直接向着楚风眠斩杀过去。
<br/>　　曾经在武斗大会的擂台上，这夜牧老祖，便是以刀锋，轰击的楚风眠。
<br/>　　但是那个时候，那一道刀锋可是穿越无数虚空，更是打破了擂台上的禁制，剩下的力量，不足半数。
<br/>　　而这一次，就在这北邙圣域之中，夜牧老祖直接出手，凝聚爆发而出的力量，恐怖到了极致。
<br/>　　这一刀的力量，若是直接落下，足以是将楚风眠的脚下的悟剑峰，都给直接毁灭。
<br/>　　“住手！”
<br/>　　就在这个时候，天空的空间，再次被撕裂。
<br/>　　能够在这北邙圣域之中，撕裂空间的人物，都至少是超越了生死境的老祖，本来一个夜牧老祖的出现，已经是够让无数人吃惊的了。
<br/>　　众人没想到，居然这个时候，又是一尊老祖出手了。
<br/>　　而这一尊老者，撕裂空间，直接便是一股灵力，向着楚风眠飞了过去。
<br/>　　不过这一股灵力，却并非是要来攻击楚风眠，而是在楚风眠的面前，化为了一道屏障，直接便是将那夜牧老祖的攻击抵挡住。
<br/>　　“师尊？”
<br/>　　楚风眠看着空中，轻声一句。
<br/>　　在这个时候，有可能出手救下楚风眠的人，只有玄北圣者。
<br/>　　楚风眠回到北邙学院，他的一举一动，都受到了无数人的注视，如今看着夜牧老祖出手来对付楚风眠，玄北圣者也是出手了。
<br/>　　“夜牧，你对我的弟子出手，是个什么意思？”
<br/>　　玄北圣者从这虚空中现身，看着那夜牧老祖，脸上尽是冰冷。
<br/>　　“玄北，你的弟子，杀了我们夜家的人，莫非就不给个交代不成？”
<br/>　　夜牧老祖看着玄北圣者，声音尖锐这吼道。
<br/>　　“今天这小子必须得死！才可以偿还他的罪孽，平息我们夜家的怒火。”
<br/>　　“我们夜家的怒火，不是你可以担待得起的，速速将这个小畜生交出来！”
<br/>　　夜牧老祖气焰极为嚣张，大声吼道。
<br/>　　他的实力，不如玄北圣者，但是他的背后可是夜家，北邙学院最为深处的存在，也是北邙学院背后最大的一方势力。
<br/>　　他敢出手直接轰杀楚风眠，也自然清楚，楚风眠的师尊，便是这玄北圣者。
<br/>　　“夜天君的事，我已经知道了，在武斗大会上，死在擂台上，生死各安天命，不过你们夜家终究是失去了一尊天才，这样如何？我给你们夜家一些补偿，这件事就这样过去了吧。”
<br/>　　玄北圣者看着夜牧老祖，平静的开口道。
<br/>　　“补偿？今天老夫就要这小畜生死！除此之外，没有任何的可能！”
<br/>　　夜牧老祖大声叫嚣着。
<br/>　　“是吗？既然如此，那我倒是要看看，你今天到底能不能杀了我的弟子！”
<br/>　　玄北圣者毫不示弱。
<br/>　　他已经退了一步，既然这夜牧老祖不答应，那他也不会再退。
<br/>　　他可是一尊超越生死的老古董，在北邙学院，夜家虽大，但是玄北圣者，也不一定会真的怕了夜家。
<br/>　　“你！”
<br/>　　听到玄北圣者的话，夜牧老祖眼神中，杀机闪现，甚至是已经在考虑，如何在玄北圣者的保护之下，强行轰击了楚风眠。
<br/>　　整个北邙圣域，如今都是寂静一片。
<br/>　　这样两尊超越生死的老祖，如此争锋相对，也是在不少弟子，都是不敢出声。
<br/>　　北邙圣域的弟子，至少都是种子弟子，天之骄子，但是这些人，在这些真正的老古董，大人物面前，都只是蝼蚁，一个个大气都不敢喘，生怕触碰到夜牧老祖的怒火。
<br/>　　就在这一触即发之际。
<br/>　　天空之上。
<br/>　　又一道裂痕，突然出现。
<br/>　　无数目光，都纷纷看了过去。
<br/>　　在这个时候，居然又有第三尊老祖，出现了。
<br/>　　这样超越生死的老祖，本来都是北邙学院的老古董，平日里都是传说中的人物，几乎从未出现过。
<br/>　　可是如今，已经是接二连三的出现了。
<br/>　　“夜牧，玄北，你们两人都住手！”
<br/>　　一道威严的声音，从这裂缝之中，突然响起。

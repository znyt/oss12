　　“什么？”
<br/>　　看到这名古月峡谷弟子，也是落入到了楚风眠的手中，被直接镇压。
<br/>　　剩下的另外三名古月峡谷弟子的脸色，都是极为凝重。
<br/>　　要是说苏师弟的败在楚风眠手中，也许还跟他轻敌有关，但是现在，这古月峡谷弟子都是拜倒在了楚风眠的手里。
<br/>　　这种实力已经不是他们所想象中的那般简单，能够做到这一切，就算是在七大宗门弟子中，都寥寥无几。
<br/>　　如今楚风眠一个人，便是将他们逼到了如此地步。
<br/>　　“不好，我们必须要救出两位师弟！”
<br/>　　“这小子不知道是从哪里，得到了这一身奇遇，现在不杀了他，必成大患！”
<br/>　　那另外两名古月峡谷弟子，看着楚风眠，眼神中都是露出了几分杀机。
<br/>　　一个北域弟子，居然是有着如此实力，除非是楚风眠的身上，曾经得到过惊天奇遇，不然的话，怎么可能会拥有如此实力。
<br/>　　“北域弟子？莫非这小子就是得到了颜古龙帝洞府之人？”
<br/>　　一名古月峡谷弟子，看到楚风眠身上的龙威，突然想到了这一点。
<br/>　　在北域之中，如今最大的机缘，便是曾经在虚空海中开辟的古洞府，颜古龙帝的洞府。
<br/>　　“颜古龙帝的洞府，祖龙至尊剑！”
<br/>　　想到这里，那两名古月峡谷弟子的眼神中，都是露出了几分贪婪。
<br/>　　祖龙至尊剑，可是龙族至宝，天地之间甚至足以称得上是最为强大的天级灵器之一。
<br/>　　就算是他们七大宗门之中，都不一定有着如此品级的灵器，谁要是可以得到，简直是万古机缘。
<br/>　　“小子，将祖龙至尊剑，跟你身上的无上剑气交出来，这一次我可以饶过你，放你全身而退！”
<br/>　　那公孙道，看着楚风眠，突然开口。
<br/>　　他的话语之中，还是一如既往一般，如同命令的语气。
<br/>　　“怎么？七大宗门的人，现在只能是逞口舌之勇了吗？”
<br/>　　楚风眠看向那公孙道，笑着开口道。
<br/>　　“现在，应该是该我开口的时候了，现在你将你身上所有的无上剑气，交出来，然后在跪下，给我磕几个响头，我可以放过你们这一次，如何？”
<br/>　　楚风眠的话，虽然是在调笑，但是说出来，那些古月峡谷弟子，听到这番话，都是一个个脸色铁青。
<br/>　　他们古月峡谷，乃是七大宗门之一，九域武者哪一个对他们，不是毕恭毕敬的。
<br/>　　如今一个小小的北域武者，居然就敢跟他们，说出这种话来。
<br/>　　“找死？”
<br/>　　公孙道的双眼，顿时变的冰冷无比，从他的身上，突然浮现出无数红色。
<br/>　　这红色，乃是他身上的血管，都是全部爆开，一团团血液，顺着他身上的伤痕，留了出来。
<br/>　　但是这些血液，却是没有流到地上，反而是在空中凝聚，一只是化为了一只巨大无比的血狮子。
<br/>　　“一个得到些奇遇的小蝼蚁，就敢跟我们古月峡谷抗衡，很好，看来我们沉寂的太久了，什么阿猫阿狗，都敢来挑衅。”
<br/>　　公孙道双眼看着楚风眠，一双眼睛中，蕴含有着无尽杀意。
<br/>　　而就在他的面前，这一只雪狮子，也是突然之间，咆哮一声，直接活了过来，张开大嘴，想要楚风眠直接冲了过来。
<br/>　　一张大嘴，咆哮着，似乎都是要将楚风眠给活活咬死。
<br/>　　“狂暴血体？”
<br/>　　楚风眠打量了一眼这公孙道，这才发现，眼前这公孙道，居然是有着一道先天神体之一的狂暴血体。
<br/>　　这狂暴血体，平日里看来，便是跟寻常武者，没有任何的区别，但是唯有他被激怒的时刻，才会真正显现出来这狂暴血体的真正模样。
<br/>　　一旦狂暴起来，他身上的血液，都是受到他的控制，化为最为强大的力量。
<br/>　　这一头血狮子，便是以他身上血液，来凝聚而出的，真正实力，只怕是不亚于一头真正的生死境巅峰的妖兽、
<br/>　　顺着一声咆哮，直接扑了过来。
<br/>　　公孙道，在古月峡谷之中，都是地位极高的天才，在中域百绝榜上，都是排名第二位。
<br/>　　他虽然狂妄，却也有着狂妄的资本，在他看来，他要让楚风眠乖乖交出无上剑气，楚风眠便是就连还手都不可能，只能是跪下，交出无上剑气来，祈求他的原谅。

<br/>　　就算是超越生死的老祖，都不曾对他说过这种话，在他看来的蝼蚁，楚风眠却是说了。

<br/>　　这一头血狮子，都是感受掉了主人的愤怒，疯狂的咆哮着，向着楚风眠扑了过来，要将楚风眠撕成碎片。
<br/>　　“古月峡谷？先天神体？也没什么了不起的，这头血狮子的力量倒是不错，正好可以当成我的晚餐！”
<br/>　　楚风眠冷笑一声，他的手指突然出现一道裂痕，从中，一滴精粹无比的金色血液，突然浮现出来。

<br/>　　就在空中，这一头血狮子，便是在远古战龙的面前，被一爪爪的撕成无数碎片，其中的灵力，全部都是被尽数吞噬。

<br/>　　公孙道也没料到，楚风眠的实力，居然是要远远超乎他的想象。
<br/>　　甚至说在对付那苏师弟的时候，都没有真正的动用全力。
<br/>　　他凝聚的这一头血狮子，丝毫没有伤及到楚风眠，反而是所有灵力，都被楚风眠吞噬，简直是送给了楚风眠一份大礼。
<br/>　　公孙道一声怒喝，突然天际之上，都是显现出来一片乌云，这乌云呈现出一种诡异的血色。

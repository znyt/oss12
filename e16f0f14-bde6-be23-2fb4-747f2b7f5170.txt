　　元始一路披荆戴月，不断地穿梭于星空之中，朝着太阳星飞去。
<br/>　　太阳星为盘古神尊左眼所化，为两大至尊星辰之一，与太阴星齐名。
<br/>　　这颗巨大无比的星辰，无时无刻不在散发着强烈的至阳之力，烤炙着洪荒大地，万物生灵！
<br/>　　太阳星表面，有着无数烈焰演化出来的太古毒龙奔腾肆虐，怒吼咆哮，好似想要挣脱这太阳星的束缚，也到这美丽的洪荒大地之中游玩一番。
<br/>　　与枯燥乏味的太阴星相比，这太阳星无疑是要热闹了许多。无数的烈焰化为无数火焰生物，在这太阳星上奔跑肆虐，各自都在绽放着属于自己的炙热。
<br/>　　“踏踏踏！”
<br/>　　这一日，一高贵威严的中年男子踏上了太阳星。
<br/>　　无数的神火演化的太古毒龙、火焰兵甲，毒焰兽等，看到他来，仿佛遇到了天敌一般，纷纷退避，不敢靠近。
<br/>　　中年男子好似火中帝皇一般，如履平地的踏入太阳深处，此人正是离开太阴星的元始。
<br/>　　元始自修成五帝神通之后，遇上五行之物，便不再惧怕，他对于五行来说，就好像帝皇一般。
<br/>　　如这太阳上的神火，元始只要祭出赤帝火皇法相，一个全身赤红，身披火皇帝袍，头戴赤色帝冠的男子出现在元始头顶，他手托大印，火焰遇到他，纷纷避让，如同遇见君王，纷纷跪拜。
<br/>　　元始大口朝着无数太阳真火一吸，瞬间，太阳真火如同潮流一般涌向元始口中，化为最精纯的火焰之力，注入到元始心之神藏道宫之中。
<br/>　　就这样，一路急行，元始的元神沿着感应着火焰之中的气息。
<br/>　　“吾以赤帝火皇之名，敕令天下万火，听吾号令！”
<br/>　　顿时，太阳星上，无数火焰纷纷摇动，朝着元始所在方向低头拜服。
<br/>　　“汝等可知，太阳星上的扶桑神树所在之地？”元始向它们传出一道信息。
<br/>　　顿时，无数火焰，一道传一道，瞬间就将扶桑树所在之地传到元始的脑海之中，而且还有一些太阳星上的基本信息。
<br/>　　这太阳星上，仅有有四个生物，一就是元始所要寻找的扶桑神树。另外三个就是三只金乌。
<br/>　　三只金乌，不错，就是三只金乌！
<br/>　　两只公的，一只母的。
<br/>　　两只公的金乌的名字，元始如雷贯耳，便是帝俊与太一。至于那只母的，元始同样知道，她名为羲和，是后来妖帝帝俊的妻子之一，为他生下十只小金乌的那个。
<br/>　　不过，根据无数火焰传来的消息，元始知道，太一与帝俊都在不久之前离开了太阳星，至于到哪里了，这些太阳真火就不知道了。
<br/>　　元始微微一笑，迈开脚步，朝着神火指出的扶桑所在之地而去。
<br/>　　一路前行，不久，元始便找到了一处太阳真火浓郁的地方，那里就是扶桑神树所在之地。
<br/>　　进入真火之中，元始远远地就感受到了此地的非同一般。此地火焰浓度极高，至少元始没有见过比这浓度更高的火焰。而且，此处还有一些太阳金晶与至阳神铁等神材，在如此浓度的太阳神火之中，它们都没有被融化，可见其珍贵之处。
<br/>　　元始毫不客气地将之收了起来。遇到了就收起来，作为今后炼器为用。不然他都不会放过自己的。
<br/>　　踏入神火深处，温度越来越高，元始便看到了一株火红的通天巨树，屹立于无数神火之中。巨树通体赤红，树干、枝条、树叶，无不散发着炙热的神火，全身就仿佛是火之一道的化身。
<br/>　　此刻，一股浩大的威势从巨树上传来，压向元始。
<br/>　　“哼！”
<br/>　　元始冷哼一声，头顶，赤帝火皇法相高高屹立，滔天帝威压向扶桑神树。
<br/>　　顿时，周边无数神火拜服，只有扶桑神树与赤帝火皇在对立，两者势均力敌，谁也不放过谁，要争出一个高低，谁为火中至尊？
<br/>　　元始没有管赤帝火皇的事，赤帝火皇在源源不断地吸收着周边的神火，威压也在一点又一点的变强，帝威变得越来越浑厚，底蕴也变得越来越强。而扶桑神树却一点也不变，这样迟早会输给赤帝火皇的。
<br/>　　元始的赤帝火皇法相吸收了无数神火，什么太阳真火、太阴真火、九玄天火、三昧真火、紫薇天火、风雷怒火、焚心烈火等，他吸收了不知多少。
<br/>　　而扶桑神树只在这太阳星上，吸收到的大多是太阳真火，其他真火很少。虽然它吸收的太阳真火比赤帝火皇要多，但它的底蕴却要比赤帝火皇要差许多。这样僵持下去，迟早会输给赤帝。
<br/>　　元始不再多管，谁知道他们要争到何时？他可不会花费无数时间陪着两道身影争锋。布置了无数防御、警示的阵法后，元始开始来到扶桑树下闭关修炼。
<br/>　　开始，他利用无数太阳真火来让元神适应这种至阳之力。元始并没有急着向太阳星深处延伸而去，而是一点一点地适应这种阳性元神。
<br/>　　此时，元始就遇到了一个令人十分头疼的麻烦，他之前已经尽数将元神转化为至阴属性，此刻再来接引这至阳真火，两者如同水火一般，互不相存，一旦碰撞在一起，便会引发剧烈爆炸，损坏元神。
<br/>　　而这时，元始想到了太极之道。心中顿时有了解决的办法。
<br/>　　因为至阴元神太大的缘故，元始将所有至阴元神凝结成一条黑鱼，之后，从黑鱼口中缓缓吐出一丝丝元神，这一丝丝元神虽然也是至寒元神，但对比之前一整个元神的至寒之力，还是微不足道的。
<br/>　　之后，元始便让这一丝丝元神接引太阳真火来烤炙，慢慢地，他也不心急，他知道此刻心急是解决不了问题的。
<br/>　　刚开始之时，那一丝丝含有至阴之力的元神还是经受不住太阳之火的烤炙，瞬间便会毁灭。
<br/>　　但随着时间的延长，毁灭了无数神魂之后，元始还是找到了一点门道，那一丝丝元神渐渐变得炙热起来。开始从至阴属性转化为至阳属性。
<br/>　　有了一丝，那就有第二丝，第三丝……
<br/>(本章完)

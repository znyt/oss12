　　现场的观众在比赛结束后就安静了下来，即使有认识的人在交流大家的声音也都是非常的小，仿佛大家都在为比赛的时候欢呼而留着力气。
<br/>　　“摩擦摩擦，在光滑的地上摩擦！！！”
<br/>　　不知道哪里突然传出了一阵手机铃声，听着叶帝那中二的声音，现场的观众不知为何一下子全都笑喷了，本来双方粉丝因为之前‘干架’的行为有些嫌隙，这个时候也都是化干戈为玉帛了。
<br/>　　现场十万名粉丝，听着这个手机铃声跟着一起呼喊了起来。
<br/>　　“有了——-滑板鞋，天黑——-都不怕！！！”
<br/>　　“一步两步，一步两步，一步一步似爪牙，似魔鬼的步伐！！！”
<br/>　　.......
<br/>　　感受着现场突然变的热烈起来的情绪，直播间里的观众却是听的一脸懵逼，他们可是没有听到手机铃声这个的。
<br/>　　“我去，要不要这么嗨啊，去现场唱滑板鞋，谁都不服，就服你们！”
<br/>　　“666666666666，现在越听大魔王的这首滑板鞋，就越感觉中毒不轻。”
<br/>　　“不好意思兄弟们，我要毒发身亡了，临死之前想在听一遍我的滑板鞋。”
<br/>　　......
<br/>　　听着现场这些观众的呼喊着，那名来电话的女生，这个时候干脆也不接电话了，现场这么吵闹就算是她接电话，她肯定听不清对面要说什么，干脆跟着大声唱了起来。
<br/>　　一段小插曲过后，众人的情绪都变的轻松了起来，十分钟过后，场上双方的决战终于是正式开始。
<br/>　　“这一局怎么打？”白队比赛室，夜雪狂歌看着叶帝开口问道。
<br/>　　“随便打，另外一个大招英雄大家不是一直想看么，这一次就给你们看看。”叶帝犹豫了一下后笑着说道。
<br/>　　“这么早就拿出来？”听着叶帝的话，虽然香蕉几人也都好奇叶帝那最后一个大招，但是这么早就拿出来，几人还是有些不舍得。
<br/>　　既然能够被叶帝说成是比螳螂还要厉害的，那这个英雄肯定是能够改变战局的，如果在世界赛决赛中拿出来的话，他们的胜率肯定会更大的。
<br/>　　“在决赛中拿出这个英雄岂不是非常的没意思，咱们要赢就凭借着自己的硬实力赢，我相信就算是让他们知道这个大招，最后的冠军也一定会是我们的。”叶帝自信的说道。
<br/>　　这也是他犹豫了很久之后做出的决定，留着这个大招确实可以让他在最后的决赛中获胜的几率更大一些，不过如果就那样取胜的话，在叶帝的心里多多少少还是有些遗憾的。
<br/>　　他想要胜利，但是不是这种颇有些投机取巧的方式，只有那种双方都了解彼此的较量，才是最让人热血沸腾，也是最让他期待的。
<br/>　　“没错，我们是最强的，根本不屑用套路取胜他们。”听着叶帝这么说，香蕉顿时气势十足的开口说道。
<br/>　　就是连萝卜和茄子两个人一瞬间也都是变的信心百倍。
<br/>　　这一局叶帝他们再次回归蓝色方，双方飞快的ban人，各自ban的还是对方几个值得针对的点，秦霄的锤石、叶帝的妖姬一如既往的都没有被放出来。
<br/>　　“不知道这一局大魔王又会拿出什么样的英雄来？”米粒有些好奇的说道。
<br/>　　双方的几个位置很快的便确定好了英雄，轮到白队这边最后一手选人的时候，所有人的目光都集中在了这里。
<br/>　　因为这最后一手选人，不出意外的白队这边又将这个位置留给了叶帝，虽然他们这边在蓝色方，最后一手选人并不是counter位，不过这也并不妨碍他们如此。
<br/>　　“这一局我估计大魔王应该会拿出螳螂这种英雄来打中单吧。”林雅楠开口说着。

<br/>　　看到英雄确定的一刹那，米粒顿时大喊出声，声音中充满了诧异。
<br/>　　“这个英雄好像出场率不高吧，就算是在路人局中也没什么人使用他啊。”林雅楠同样充满了好奇。
<br/>　　“不会是大魔王所说的最后一个大招吧，如果真的是那样的话，那么这个英雄倒是真的要值得商榷一下了，不要忘了大魔王可是非常擅长开发那些冷门英雄的。”米粒跟着开口说道。
<br/>　　而另一边在蓝队的比赛室，在看到茄子这边已经确定好了最后一个英雄后，秦霄几人的眉头也都是瞬间皱了起来。
<br/>　　他们这边将最后一个counter位留给了蒋念，但是面对着这样一个从来没有见过的中单，一时之间他们也是变的手足无措了起来。
<br/>　　“景禹怎么选人？”秦霄开口问道。
<br/>　　“不出意外的话，这应该就是小叶子说的最后一个大招了，选一个稳点的英雄打吧，这个英雄说真的，就算是我也不怎么了解，不知道这一局小叶子要怎么玩了。”李景禹有些无奈的说着。
<br/>　　“哈哈，我就才小叶子肯定不会将这个大招留到最后的，果然如此，他那性格就是想要和别人堂堂正正的一战，不过还是太年轻了，这一局就算是输了，下一局我肯定也能够将这个英雄练好。”蒋念倒是看的开，有些兴奋的说道。
<br/>　　听着蒋念的话，秦霄几人也都是笑着摇了摇头。

<br/>　　一瞬间所有人都跟着激动了起来，很多人更是直接用浏览器在查找流浪英雄，搜索起了这个英雄的属性。
<br/>　　不过看到流浪法师的介绍的时候，众人还是一脸的懵逼。
<br/>　　蓝量越高会提升法强，这尼玛什么鬼，还有这样的机制？
<br/>　　不过看到这特殊的一点，在想到叶帝所谓的大招，众人心里还真的有了一些期待。
<br/>　　片刻之后，在现场无数观众的欢呼声中，这一场比赛终于是正式开始。

　　第86章
<br/>　　春种秋收，夏生冬藏。
<br/>　　幽州的冬天总是比别的地方来的更早些，过了十月，西北风呼呼刮着，天气便一日冷过一日。
<br/>　　到了冬天，大部分人便停下手中的伙计，钻进屋子里老婆孩子热炕头，准备猫过这个冬天。只有一些缺衣少粮的人，才不得不顶着西北风在街上奔波，拼一家老小的吃食。
<br/>　　总之，这个时代的生活节奏是很慢的，相比之下，程大雷竟还算勤奋的，在他大半年的努力下，蛤蟆寨已经走上正轨。
<br/>　　换句话说，冬天狗熊都冬眠了，有必要让自己忙得像条狗嘛。
<br/>　　所以程大雷打算歇一歇，做些自己感兴趣的事情。
<br/>　　他决定搞搞事情，获得一些恐惧值。
<br/>　　每一万恐惧值可在点将台召唤一次，连抽十次还送一次，且必然获得优秀品质。程大雷打算趁着冬天没事，积攒十万恐惧值，来一个十一连抽。
<br/>　　上次抽到刘关张三兄弟，虽然目前三人的等级还很低，可潜力却不低。这提醒程大雷，想要获得高品质的猛人，还得靠系统。这次来个十一连抽，说不定就能抽到个缩水版诸葛，实在不行，抽到个缩水版吕布也是可以的。
<br/>　　可如何快速获得恐惧值，却难住了程大雷。
<br/>　　现在他这个蛤蟆寨的蛤蟆大王已然臭名远博，事情过去好久了，偶尔还有某人给他贡献一些恐惧值，这段时间，程大雷又攒下三千。可相比十万的目标，实在有些杯水车薪。
<br/>　　如何才能在最短时间内迅速获得十万恐惧值？
<br/>　　作为山贼，似乎只有打家劫舍一条路走。落叶城快成程大雷的后花园，为了落叶城的繁荣，自己暂时不能去落叶城捣乱。
<br/>　　不如……自己去黑石城碰碰运气？
<br/>　　说来，自己已有好长时间没和薛半川打交道了，自己还是蛮想念他的。
<br/>　　可黑石城也有一个问题，距离蛤蟆岭太远了。两地相距一百五十公里，快马也要走一天，没办法打闪电战，去了别再被人抓住。
<br/>　　想不到解决办法，只好暂时把这件事情搁下。
<br/>　　这一天，程大雷去见了见杜茂。刚走到地牢，就迎来杜茂一阵破口大骂。
<br/>　　“姓程的，你干脆一刀把我杀了，若不然，我做鬼也不会放了你！”
<br/>　　看到杜茂时，程大雷吓了一跳。只见他披头散发，两眼红肿，不知道有多少天没睡觉了。
<br/>　　“杜老大，你怎么了，是没休息好么，我让人再送床被子过来。”
<br/>　　“姓程的，你要折磨我到什么时候！”
<br/>　　“折磨，这话从何所起，优待战俘一直是我们的原则，每天好酒好肉，还专门有人陪你聊天，我都没有这待遇啊。”
<br/>　　听到『陪你聊天』四个字，杜茂眼底闪过一丝难以隐藏的恐惧。
<br/>　　刘悲除了给大家看病，每天都会抽时间过来和杜茂聊聊。他简直就是个负能量爆棚的病原体，就连程大雷在和他的日常接触中，偶尔也会冒出要不我去跳个崖玩玩的心思。
<br/>　　而杜茂是每天都要和刘悲面对面，做关于人生和哲学方面的交谈。当然，很多时间都是刘悲自说自话，可刘悲身上的负能量，却无孔不入的侵袭着杜茂这个糙汉子的内心。
<br/>　　亏得他性子坚强，现在还没有寻死的念头。但这样的煎熬，于他而言分外痛苦。
<br/>　　程大雷离开的时候，正好撞见要进来的刘悲。他依旧一副生无可恋的表情，两只手抱在一起。
<br/>　　“努力，思想工作要继续做下去。”程大雷本来想拍拍他的肩膀，但举起的手则又放下了，还是注意隔离的好。
<br/>　　离开地牢，程大雷又找到关鱼，打算好好和他谈谈。
<br/>　　刘关张三兄弟中，只有关鱼是个战斗人才，关鱼的技能是练兵，稍稍培养一下，就能顶替秦蛮现在的工作，成为山寨的支柱。
<br/>　　可经过短暂接触，程大雷就发现，关鱼这个人……不太好接触。
<br/>　　他是眼睛长到天灵盖上的人，除了刘悲，眼睛里根本不搁任何人。在他面前，程大雷总有矮一头的感觉。
<br/>　　你说你要是真的关二爷，这种脾气也就罢了。我给你牵马我都心服口服……估计以程大雷现在的本事，去给二爷牵马都不够格。
<br/>　　但你毕竟只是个缩水版，凭现在的程大雷一巴掌就能打倒，你再是这种脾气，可就真有些欠揍了。
<br/>　　但程大雷不仅不能揍，还得尽量惯着，关二爷如果没有这只脾气，或许也没办法成为关二爷。为了日后你能成长为，那过五关斩六将，温酒揍华雄的云长关，一切，我都忍着吧。
<br/>　　程大雷这次主要是想和关鱼聊聊，自己前生所知的练兵经验。比如纪律，统治，军心……程大雷虽然所知不祥，但数千年人类通过无数战争积累的经验，程大雷只知道一些皮毛，却也足够受用了。
<br/>　　把这些传授给关鱼，说不定他就能突破，达到优秀等级。那么，他日后将会发挥难以相信的作用。
<br/>　　一个上限是传说品质的人，升级应该不是很困难吧。
<br/>　　不得不说，关鱼的接受能力很强，对于程大雷的话，往往能举一反三，有些东西程大雷都没有想到过。
<br/>　　“这个……为什么你们三兄弟都没有习过武呢？”
<br/>　　“我大哥说武夫上阵，不过是和一人对敌，我们要么不学，要学就学万人敌。”
<br/>　　呵，吹什么牛，到现在你也不过只是普通等级。
<br/>　　“那么……多少学点么，防身也好嘛？”程大雷只好捧着聊。
<br/>　　“呃……也是没人教。”关鱼略带一些遗憾大，虽然很骄傲，可随便来个人就能把自己打趴下，骄傲得一点底气都没有。
<br/>　　嘟，抢劫压寨夫人第三段任务完成，奖励条件抽奖次数1
<br/>　　嘟，条件不足，无法触发第四阶段任务。
<br/>　　正这个时候，程大雷忽然收到系统提示。苏樱的酒楼已经迈入正规，日日客满为患，苏樱正在筹划开分店的事情了。
<br/>　　如此看来，苏樱的酒楼已经成为落叶城第一酒楼，自己的任务是完成了。
<br/>　　嘟，是否现在决定抽奖？
<br/>　　“抽奖……”
<br/>　　程大雷心底默念了一声，脑海中扭蛋机再次出现，骨碌碌旋转，紧跟着一物掉了下来。
<br/>　　嘟，恭喜获得刀法秘籍一本……
<br/>　　
<br/>(本章完)

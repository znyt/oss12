        片刻后，严文斌点的那三个菜都被服务员端上了桌，随后没几分钟，两个服务员又端上来了一盘烤全乳猪。
<br/>
<br/>        严文斌见状，顿时吓傻了眼，连忙说道：“我们没点这个！”
<br/>
<br/>        “是我们免费送的体验菜，你们放心品尝。不收钱的。”服务员微笑着说道，放下烤全乳猪后，又端上了一盘烧鹅和清蒸鲥鱼。
<br/>
<br/>        连星儿看着这三个菜，顿时满心欢喜，迫不及待地想要开吃。
<br/>
<br/>        严文斌却在心里盘算着，这三样菜加起来起码最少要九百块钱。
<br/>
<br/>        烤全乳猪大概五百块左右，烧鹅便宜点，一百块左右，至于那条鲥鱼，应该要两三百吧！
<br/>
<br/>        餐馆是开着门做生意的，又岂会做这种赔钱的买卖？
<br/>
<br/>        严文斌担心这是个“坑”，但是为时已晚。
<br/>
<br/>        因为，连星儿已经毫不客气地拽下一个猪蹄，自顾自地吃了起来。
<br/>
<br/>        “这个好好吃，严文斌，你也快尝尝。外皮焦脆，里面肉嫩多汁。”连星儿一边吃着一边催促道。
<br/>
<br/>        严文斌咽了咽口水，拿起一旁的切刀，战战兢兢地切下烤乳猪腹部上的一块肉，然后拿筷子夹进蘸料碗中吃了起来。
<br/>
<br/>        “怎么样，是不是很好吃？”连星儿看着严文斌，笑眯眯地问。
<br/>
<br/>        严文斌却有些尴尬地点了点头。
<br/>
<br/>        屏风后，明司寒听着连星儿那甚是欢喜的声音，嘴角情不自禁地微微上扬。
<br/>
<br/>        用自己赚的钱，给这丫头买最好吃的东西，他居然会觉得特别开心，还真是一种奇怪的嗜好。
<br/>
<br/>        吃到一半的时候，严文斌才发现连星儿对于自己所点的那三道菜是一点都未沾。
<br/>
<br/>        连星儿也及时发现了这点，连忙拿起汤勺，给自己盛了碗汤。
<br/>
<br/>        严文斌苦笑了一番后，忽然语重心长地说道：“星儿，我有话想对你说。”
<br/>
<br/>        “嗯，你说。”连星儿端起碗，应了声后，便开始拿起勺子啜汤。
<br/>
<br/>        而严文斌这个时候却没有下文，只是含情脉脉地看着连星儿。
<br/>
<br/>        连星儿是高三的时候转到他们班上来的，从她被班主任领进教室站上讲台做自我介绍的那一刻，他坐在讲台下，隔着一段距离，却对她一见钟情。
<br/>
<br/>        他不否认，她是他所见过的女孩子中长得最漂亮的一个，同时在他的心底，也只认为，只有像她连星儿这般，既漂亮学习成绩又好的女孩子才能配得上他。
<br/>
<br/>        其实，他一直想跟她表白，但是一直找不到机会。
<br/>
<br/>        如今，此时此刻，机会难得，他不想再错过。
<br/>
<br/>        毕竟，错过了，他与她各奔东西，就真的再也没有机会了。
<br/>
<br/>        “怎么不说话呢？”连星儿怔愣地看着严文斌，见他看着自己发呆，心里顿时有种不好的预感。
<br/>
<br/>        严文斌随之对上连星儿的目光，在与她四目相对时，心脏也跟着“扑通扑通”地加速跳动。
<br/>
<br/>        他的脸颊微红，呼吸变得凝重了几分。
<br/>
<br/>        严文斌呼出一口气后，又深深地吸了一口。
<br/>
<br/>        当他张开嘴巴，准备说什么的时候，连星儿却在此时连忙叫“停”。
<br/>
<br/>        “寝不语，食不言。吃饭的时候，我们还是不要聊天了！”连星儿接着用极快的语速说完后，立马低下头去，淡定地继续喝汤。
<br/>
<br/>        严文斌想表白的话到了嘴边，不得不硬生生地又咽回肚子里去。
<br/>
<br/>        饱餐一顿后，严文斌让服务员拿餐盒过来将剩下的烤乳猪肉给打了包，然后递给了连星儿。
<br/>
<br/>        “我觉得你似乎很喜欢吃这个，虽然是剩下的，但是我没用我自己的筷子动过，所以很干净，你不要浪费了。可以带回去吃。”严文斌微笑道。
<br/>
<br/>        连星儿愣了一下，然后才接过严文斌手中的袋子。
<br/>
<br/>        对于这餐饭，严文斌全程是吃的战战兢兢的，直到去埋单的时候，发现确实只需要他支付他所点的那三道素菜，心里这才像卸了重担一样轻松了不少。
<br/>
<br/>        毕竟，天下没有免费的午餐，他的担心并不是多余的。
<br/>
<br/>        严文斌买完单后，和连星儿一起走出了餐馆。
<br/>
<br/>        就站在餐馆门外的马路边上，严文斌打算把刚刚在饭桌上没说完的话继续说完。
<br/>
<br/>        “星儿，我……”
<br/>
<br/>        “你可不可以借我一百块钱？我出门的时候忘记带钱包和手机了！你放心，回去后，我一定会还给你！手机给你转账！”连星儿直接打断了严文斌的话。
<br/>
<br/>        严文斌有些无言以对，很无奈地从皮夹里掏出了一百块钱递给了连星儿。
<br/>
<br/>        “那我就先走了，谢谢你的热情款待。”连星儿拿到钱后，向严文斌挥了挥手，完全不给他说话的机会，便转身跑开了。

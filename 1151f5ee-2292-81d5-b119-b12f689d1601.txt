砰一声，霍璇撞到墙壁之后又摔在了地毯上，喉头腥甜，她当即吐出了一口血。
<br/>
<br/>        刚才顾夜瑾那一挥手差点要了她半条命。
<br/>
<br/>        “夜瑾…哥哥，是我…”霍璇抬头看向顾夜瑾。
<br/>
<br/>        顾夜瑾颀长如玉的伫立在窗台前，他居高临下的看着此刻狼狈而痛苦的趴在地上的霍璇，他比任何人都清楚叶翎的手感，叶翎软若无骨，身上还有一股诱人的馨香，这些都不是霍璇能比拟的。
<br/>
<br/>        他抱上去的瞬间就该察觉的，但是他被欲迷了心，房卡又是给她的，没想到会是别人。
<br/>
<br/>        只要想到刚才他抱了别人几秒，他就忍不住的想去把自己狠狠冲洗上几遍，把自己洗干净。
<br/>
<br/>        顾夜瑾那双清寒的黑眸像淬了毒般阴冷的盯着霍璇，良久他从喉头里滚出森然的音节，“怎么是你？快点说，如果你还不想死的话！”
<br/>
<br/>        霍璇万万没有想到他会这么大的反应，刚才她还很有自信的，但是谁知道他见自己不是叶翎就想将自己给弄死了。
<br/>
<br/>        霍璇很害怕，刚才的旖旎心思也在骤痛里消失不见了，她觉得顾夜瑾不是开玩笑的，她迅速道，“夜瑾哥哥，我…是叶翎让我进来的，房卡也是…是叶翎给我的…”
<br/>
<br/>        房卡是叶翎给的。
<br/>
<br/>        顾夜瑾早想到了，现在被霍璇证实了他勾了一下薄唇，精健的胸膛上下起伏，他抬脚踹翻了一张椅子。
<br/>
<br/>        她真敢！
<br/>
<br/>        好，她真的很好！
<br/>
<br/>        霍璇吓得瑟瑟发抖，“夜瑾哥哥，我说的…都是真的，叶翎还说…说你喜欢放荡的女人，让我努力伺候你，她还说…说你是她不要了的…垃圾…”
<br/>
<br/>        这些年顾夜瑾觉得自己最引以为傲的就是自制力，毕竟在那样的原生态家庭里长大他没点自制力估计早就疯了，但是叶翎总是他人生里的例外，他最引以为傲的自制力在她面前溃不成军。
<br/>
<br/>        顾夜瑾抬手，将柜台上的红酒全部拂落在地，他砸碎了房间里所有的东西。
<br/>
<br/>        这么大的动静自然吸引了外面的注意，霍西泽踹门而入，“夜瑾哥，发生什么了，你在干什么…艹，霍璇，你怎么在这里？”
<br/>
<br/>        霍璇是第一次看到顾夜瑾如此疯狂的模样，现在看到霍西泽她像是看到了救命稻草，“哥，哥快救我！”
<br/>
<br/>        霍西泽不用想都知道肯定是自己的妹妹来献身惹怒了顾夜瑾，毕竟是自己的妹妹，不能见死不救，“夜瑾哥，这一次肯定是我妹妹的错，但求你饶了她这一次好不好，以后我保证她绝对不敢再来烦你了。”
<br/>
<br/>        房间里一片狼藉，顾夜瑾俊美的五官在昏暗的光线里阴沉的能滴出水来了，他敛了一下眸子，吐出一个字眼，“滚！”
<br/>
<br/>        “霍璇，快走！”霍西泽迅速将霍璇给带走了。
<br/>
<br/>        顾夜瑾进了沐浴间，开始冲冷水澡，这时一串悠扬的手机铃声响起了，来电话了。
<br/>
<br/>        他沾着水滴的大手伸出来接通了电话，那端传来了私人秘书恭敬的嗓音，“少主，叶翎已经离开1949了。”
<br/>
<br/>        顾夜瑾默了几秒，然后漫不经心的开腔，“让人暂停叶翎的一切活动，将她雪藏…告诉她，什么时候想通了来找我。”

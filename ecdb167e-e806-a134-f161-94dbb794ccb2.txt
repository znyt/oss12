        童筱颖看了一番资料后，将明司寒的个人信息单独抽了出来：“大哥，我觉得这个男孩子不错。只比星儿大两岁。”
<br/>
<br/>        “我跟你说正事。”傅夜沉忽然发现，男人的世界观似乎真的与女人不同。
<br/>
<br/>        童筱颖放下手中的资料，跨过去一屁股坐在了傅夜沉的大腿上，双手搂着傅夜沉的脖子，会心一笑：“问问星儿的意见吧！如果星儿愿意，你就允诺这门婚事。如果星儿不愿意，那么今后傅氏集团的命运，只能走一步算一步了。”
<br/>
<br/>        “那你打电话把星儿叫回来。”傅夜沉深沉道。
<br/>
<br/>        童筱颖微笑着点了点头，指尖从傅夜沉的脖颈后滑到了他的胸膛上。
<br/>
<br/>        傅夜沉的身上只穿着一件单薄的睡衣，布料贴在肌理纹路上，透着成熟的荷尔蒙气息。
<br/>
<br/>        下一秒，他捏住她的下巴，倾身吻了上去。
<br/>
<br/>        不知不觉中，童筱颖沦陷在傅夜沉的热吻之中，胸前也突然传来一袭凉意。
<br/>
<br/>        翌日。
<br/>
<br/>        连星儿是被童筱颖一个电话给叫回了滨江城。
<br/>
<br/>        另一边，明司寒则被他的两个亲哥哥以及另外三个堂哥给“合谋算计”了。
<br/>
<br/>        爷爷提出明家跟傅家联姻，对方竟然还是个十八岁的黄毛丫头，哥哥们可是一脸不乐意，但是一个个又无法拒绝爷爷的“霸权”。
<br/>
<br/>        “我听说，对方是傅夜沉的女儿，才十八岁！如果成了，就先订婚，等她到了法定年龄就结婚。那岂不是还得等两年！这种事情，我才不干啊！小寒，你为了我们五个哥哥的幸福，就牺牲一下色相，去勾引那位大小姐，让她选你吧！”电话里，明希哲大吐苦水。
<br/>
<br/>        明司寒也不仅纳闷起来，起初爷爷是强硬反对他跟连星儿在一起的，怎么这会儿突然来了一个三百六十度的大转弯，想起要明冷两家联姻？
<br/>
<br/>        “喂喂喂，小寒，你到底有没有在听我说话？撇开你其他几个哥哥不谈，就我来说，我可不能订婚啊！我若是这么年轻就订婚了，在娱乐圈里还怎么混啊？”明希哲抱怨连连。
<br/>
<br/>        明司寒突然上文不接下文地回答道：“你之前不是要我帮你查洛羽的后台吗？”
<br/>
<br/>        “关洛羽什么事？”明希哲茫然道。
<br/>
<br/>        明司寒接着回答道：“洛羽是傅夜沉的女儿的男朋友。”
<br/>
<br/>        “我靠！”明希哲震惊不已，“有男朋友了，爷爷还拿我们几个兄弟送上门去给人家选？我们明家至于这么落魄吗？”
<br/>
<br/>        “爷爷这么做，只是给傅家面子罢了。更何况，爷爷反倒是把难题扔给了傅家，让傅家进退两难。”明司寒倒是看得十分通透。
<br/>
<br/>        明希哲却不关心这些：“既然她有男朋友了，那我就放心了。不过，话说回来啊！如果，这是两家长辈都很强势，你就得做牺牲了。”
<br/>
<br/>        “……”顿时，明司寒很无语地挂了电话。
<br/>
<br/>        跟洛羽比起来，他明司寒似乎确实更有优势，毕竟他跟连星儿算得上是门当户对。
<br/>
<br/>        连星儿回到家后，和童筱颖一起坐在了客厅的沙发上，翻看着明家这两个未婚未恋的孙子的资料。
<br/>
<br/>        “妈咪，这明家的孙子各个都长得好！这个明启辉，我去北城看哥哥的时候见过。他是哥哥的上司。明希哲是当红偶像明星，是我家洛羽的死对头。至于这个……”连星儿翻到明司寒的资料时，欲言又止了。
<br/>
<br/>        童筱颖立即替连星儿补充道：“是不是觉得就这个男孩子适合你？”
<br/>
<br/>        “……”连星儿没吭声，但心里却打起了小算盘。
<br/>
<br/>        至于自己迟早是要被“豪门联姻”的这件事情，她从小就耳濡目染，虽然不反感，但也不想顺从。
<br/>
<br/>        只不过，妈咪跟她说了利弊之后，她也只是打算先看看候选对象，实在没有中意的话，那就跟妈咪摊牌。
<br/>
<br/>        好在，明司寒也在候选人之中，这件事情就好办了！
<br/>
<br/>        “妈咪，我就选他好了！”连星儿举起明司寒的资料，手指点了点资料上明司寒的照片，嬉皮笑脸地对童筱颖说道。
<br/>
<br/>        童筱颖顿时难以置信地看着连星儿，总觉得这小丫头葫芦里在卖着其它药。
<br/>
<br/>        连星儿不禁得意地扬了扬眉梢。
<br/>
<br/>        明司寒不是看她碍眼吗？
<br/>
<br/>        他要是知道她选的是他，他一定会去跟他爷爷提出抗议的。
<br/>
<br/>        总而言之，她就一口咬定他，其他的五个明家哥哥们她都看不上。到时候，终止联姻的这件事情，就不算是他们傅家故意为之了。
<br/>
<br/>        到时候，两家联姻的事情一定得黄。
<br/>
<br/>        “你决定好了？”童筱颖试探性地问。

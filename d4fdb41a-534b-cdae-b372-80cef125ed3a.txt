　　因为时间是不停流逝的，穿越到了别的世界，自己世界的时间也依然在流逝，等自己回到时间虚空时，就算找得到自己的时间轴，也容易误入别的时间节点，计算得再准确也不行，哪怕差一秒钟，那也不是自己原本的世界。
<br/>　　在山海印上留下一丝血液，只要回到时间虚空，就可以通过血液找到回来的路，而不用精心去计算时间。
<br/>　　“可真是一场前所未有的旅行。”小九拉着叶少阳的人，两人仰面躺在床上，透过窗户望着外面的一汪夜色。
<br/>　　时空之旅，连小九也觉得异常新奇。
<br/>　　叶少阳半天不做声。
<br/>　　“你那个干姐姐，人好像很不错，就像……就像家人一样。”
<br/>　　“她对你，那是爱屋及乌啦。”
<br/>　　小九想到翠云把她认作是叶少阳的媳妇，但他一直没有纠正，脸上微微发烫，偷偷看了他一眼，叶少阳神色凝重，也不知道在想什么。
<br/>　　小九就静静地抱着他的胳膊，两人都不再说话，后来不知什么时候叶少阳睡着了，小九就这样抱着他，一直睡到天亮。
<br/>　　接下来几天，四宝和吴嘉伟也回来了，大家一起办年货，忙到年三十，瓜瓜来了，还将捉鬼联盟的兄弟们都带回来了，包括七宝那三个小鬼，吵着找叶少阳给压岁钱。
<br/>　　叶少阳当大厨，四宝等人打下手，从清晨忙到中午，居然做出了满满一桌菜，林三生他们也在做菜，用的是从鬼域和从空界找来的食材，熬了一些汤，用加了盖子的竹筒装起来，留着中午当酒喝。
<br/>　　“霍，都来了啊，恭喜发财恭喜发财！”老郭推门进来，瓜瓜立刻带了几个小鬼过去要新年礼物，老郭还真有准备，一人送了一只鬼书二号plus。
<br/>　　所谓的鬼书二号plus，是鬼书二号的升级版，改良了一些功能，但因为内核大体没变，不能算是第三代，老郭就给它取了这么个时髦的名字。
<br/>　　“你不是要在家过年吗？”叶少阳从厨房探出脑袋，问他。
<br/>　　“你们都来了，我一个好意思不来啊。”老郭嘿嘿笑着，表示已经跟家人商量，晚上再过节，又说小鱼本也吵着要来的，但他不想留郭大嫂一个人在家，于是就留小鱼在家陪他妈了。
<br/>　　“不过小鱼可说了，你这个当叔叔的，红包和礼物可不能少。”
<br/>　　“早就准备好了！”
<br/>　　叶少阳洗了手，让四宝他们负责端饭，从兜里摸出一枚亮晶晶的铜钱，交给老郭。
<br/>　　“雕母大钱！”老郭震惊，立刻变了脸色，表示这礼物太重，绝对不要。
<br/>　　雕母大钱虽然不是专属法器，但因为数量稀少而通灵性极强，是各门派不可得的上品法器，茅山这么多年下来，也不过传下来几枚，几乎相当于一件九段光的法器，老郭本来找叶少阳要礼物，也就是开玩笑性质，图个彩头，哪里想到叶少阳会送这么珍贵的东西。
<br/>　　“我也没啥东西能拿的出手，这枚铜钱被我刻了符文，昨天开了光，你拿去给小鱼当个附身符吧。好了好了大家都去端菜。”叶少阳把钱给他，又钻进了厨房。
<br/>　　老郭碰着雕母大钱，呆呆地看着叶少阳，他当然知道这枚铜钱的价值，更不要说还是他亲自开光，别说当护身符了，拿去一个小门派做镇山之宝都绰绰有余了。
<br/>　　老郭握紧手中的雕母大钱，什么也没说，放进了贴身口袋里。
<br/>　　菜摆上的时候，橙子也到了，带着萧逸云一起来的，叶少阳热情招呼，萧逸云喝了林三生泡的阴人茶，来到叶少阳面前，跟他说起阴司最近的情况。
<br/>　　“几大元帅已经回兵，与太阴山的鬼兵厮杀起来，四条线一起作战，不说势如破竹，但也占据了上风，照这个情况下去，阴司还是能赢的，只是大家都怀疑，无极鬼王既然对阴司展开全面进攻，肯定还有后手吧，但到底是什么就不知道了。”
<br/>　　叶少阳切着最后的凉菜猪耳朵，一边听萧逸云说话。
<br/>　　“还有……徐福的残魂去了海眼，说他已经将山海印交给你了，要求投胎，但因为他剩下半魂还在阴司昭狱被现两地被太阴山鬼兵阻隔，互相过不去，他还在海眼等着。”
<br/>　　叶少阳切菜的动作停了一下，问道：“阴司怎么说？”
<br/>　　“还能怎么说，找你要山海印你又不给，眼下鬼兵围城，谁还有工夫管你。”
<br/>　　果然跟徐福推测的一样。
<br/>　　萧逸云又说了几件鬼域最近发生的事，叶少阳切完了猪耳朵，就让他停下别说了。“今天过节，聊点轻松的，先不要说这些了。”
<br/>　　一桌菜端上去，大家倒酒开吃。林三生、美华和瓜瓜等几个鬼魂都端着林三生酿的酒，跟叶少阳他们干杯。
<br/>　　“好香啊！”美华喝了一口，大声赞道。
<br/>　　瓜瓜等人也交口称赞，喝了又喝。
<br/>　　林三生笑道：“这可是我亲自酿的酒，你们以为呢。想当年我在建文帝麾下，一心报国却不受重用，那些年苦闷之中，除了写字作画，便是酿酒了。后来做了鬼，这门手艺也没放下。你们可慢点喝，这酒稀少，我就带来一桶，一人三杯，喝完就没有了。”
<br/>　　“这酒叫什么名字？”美华陶醉地品尝着，问了一声。
<br/>　　“醉三生。”
<br/>　　大家都觉得这名字取得极好。
<br/>　　林三生他们虽然不吃菜，但用鼻子闻一闻，也跟吃了菜一样满足，端着酒杯，跟人间的几个推杯换盏，十分热闹，一场别开生面的阴阳两界的聚会，在愉快的氛围中进行，一直吃到下午两点多结束，接着又一起喝茶聊天，尽兴之后，林三生表示还要回去跟自己手下们喝一场，也算是与民同乐，收买人心。
<br/>　　叶少阳干脆让美华他们也都跟着去，他们走后，橙子和萧逸云留下，说了一会话特告辞了。叶少阳四人各自去睡了一觉，傍晚起床，酒也醒了，叶少阳强令他们帮自己一起刷锅洗碗，把一切收拾好，然后去老郭家参加家庭聚会。

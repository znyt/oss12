　　叶少阳二人假装是去庙里烧香的，跟着这些村民一起上了山，进了山神庙之后，看到庙堂的一面墙被熏得黢黑，显然是昨晚上那把火的结果。
<br/>　　陈老汉招呼那几个人去修缮被火烧过的墙壁，叶少阳和毛小方在被发现之前，绕到了山神庙的侧面，叶少阳让毛小方等着，自己爬上房顶，看着陈老汉带着几个人修墙去了，自己悄悄跳进院子，轻轻推了推那个叫云儿的姑娘的房门。
<br/>　　门从里面拴上了，叶少阳只好轻轻敲了敲门，怕被陈老汉他们听见，也不敢说话。
<br/>　　屋里传来脚步声，之后房门打开，云儿也没看，开了门转身就往里面走，叶少阳也不敢出声，赶紧进去，把门关上，然后转身去看云儿，当场就呆住了：
<br/>　　云儿一边走一边把身上衣服脱了，光溜溜地爬上床去躺下，也不朝他这边看一眼。
<br/>　　那曼妙的身躯……叶少阳浑身一颤，深吸一口气，尽量不去看他。
<br/>　　叶少阳一想就明白了：这姑娘……肯定是把自己当成是五通神了！
<br/>　　他猜的没错，那五通神欲望极强，不光晚上回来，偶尔白天也会来找她，她逆来顺受，早就放弃挣扎了，只是实在不想看他那张蛤蟆一样让人恶心的脸。而且这房间镇上人都知道的，除了五通神，不可能有人敢进来，那个陈老汉虽然就住在隔壁，看她的眼神有些色心，不过也从来没这个胆子，因此她根本没有多想，开门后就上床了，结果等了一会不见动静，忍不住转头看了一眼。
<br/>　　不是那个蛤蟆脸……
<br/>　　而是一个还算眉清目秀的小伙，傻傻地瞪着她。
<br/>　　云儿一下子就懵了。
<br/>　　“不是不是，云儿姑娘，”叶少阳急的摆手，压低嗓门，“不是你想的那样，我……”
<br/>　　眼看着云儿张嘴要叫出来，叶少阳怕他叫声引来外面那些人，一下子扑到床上去，捂住了她的嘴。他倒不是怕外面这些人，而是担心事情闹起来，把五通神引来，麻烦就大了。
<br/>　　“我不是坏人，我……”叶少阳突然发现自己趴在云儿的身上，一只手捂着人家的嘴，这个样子……说自己不是坏人，真的很难让人相信。
<br/>　　“我松开手，你别叫了啊！”叶少阳松开手，爬了起来，坐在床上。
<br/>　　云儿傻傻地看着他，冷笑道：“五通神的女人你也敢碰，你不要命了？”
<br/>　　“我……”叶少阳一时间有点不知道怎么开口，“我先下去，你先把衣服穿好。”
<br/>　　叶少阳刚要起身，云儿突然一把拉住他，叶少阳没防备，被她重新拉到身上趴着，云儿突然把嘴巴凑上来，贴上他的嘴，同时两只手紧紧箍住叶少阳的脖子。
<br/>　　这是什么情况……
<br/>　　叶少阳完全触不及防，用力将她的脑袋掰开，喘了口气，颤声道：“我说大妹子，你这是干啥，我是……”
<br/>　　“你快一点，完事就走，不要让五通神发现，快点……”
<br/>　　叶少阳刚要开口解释，云儿又把嘴巴贴了上来，两只手开始脱他的衣服。
<br/>　　那柔软的身体，尤其是胸前跟自己贴在一起的那两团，叶少阳怎么说也是男人，而且是个万年老处男，顿时感到一股邪火直冲头顶，急忙再一次推开她，按住她的脖子，说道：“你这是干什么啊！到底怎么回事！”
<br/>　　之前偷窥，让他相信这个姑娘不是真的那种水性杨花的，而且人很善良。
<br/>　　“你既然来了，我就把身子给你，这样五通神就不会再要我了，我不想一辈子成为他的奴隶，我也是刚想到的，要了我你就走，快点……”
<br/>　　快点……
<br/>　　云儿在他怀里各种扭动，十分主动，叶少阳用力推开她，奈何她两条腿从下面夹着自己的腿，挣脱不出来，叶少阳也不敢真使劲，不然怕伤了她，只好抓住她两只胳膊，按住之后，从后面抱着她，将她紧紧搂在怀里，让她不能活动，不过这样一来，两个人的接触更加亲密了……
<br/>　　再这样下去，非得被强行**了不可。叶少阳用力吸了一口气，一口气说道：“我给你说大妹子，我知道你叫小云，我不是来找你……嗯，我是道士，来救你下山的，你赶紧跟我走，快点！”
<br/>　　云儿本来正在亲他，听他说到后面，一下子就愣住了，怔怔地望着他，“你说什么？”
<br/>　　“我是道士，我昨天来看到了你的遭遇，我是特意来救你的，你跟我走。”
<br/>　　“你疯了！五通神不会放过你的！”
<br/>　　“我要是怕，也不会来找你了，相信我，跟我走，绝对让他找不到你。”叶少阳一脸诚恳地望着她，云儿也盯着他看，目光闪动，叹了口气道：“走不了的。不过，还是试试吧。”
<br/>　　叶少阳赶紧起来，让她穿衣服，目光实在忍不住往她胸前瞄了一眼，心神一颤，急忙移开。
<br/>　　等云儿穿好衣服，叶少阳让她先等着，自己过去轻轻打开门，见外面没人，只能听到庙堂侧面有干活的声音，于是对云儿招了招手，云儿刚要跑出来，叶少阳一把抱起她，把她扛在肩上，眨眼间冲出了庙门，一路狂奔下山，这才想起来忘了去找毛小方，转头看去，毛小方应该是居高临下发现了他，冲了下来。
<br/>　　叶少阳一刻不停，朝着小镇的方向冲过去。毛小方跟在后面跑。
<br/>　　“少阳子！”
<br/>　　叶少阳跑了有十几分钟左右，听见毛小方的喊声，转头看去，却是一团乌云从山神庙的方向飘过来，速度很快。
<br/>　　果然还是被发现了……
<br/>　　云儿也看到这一幕，啜泣起来，用力抓住叶少阳的肩膀，抓得他肩膀都疼了，哭着说道：“道长，你有没有办法帮我把魂魄毁了？”
<br/>　　“什么？”叶少阳一惊，问道。
<br/>　　“或者毁了我的身体也行……我以前自杀过，就算我死了，他依然能将我救活……我不想再回去了，求道长杀了我……将我分尸，或者将我魂魄毁了，我就算是魂飞魄散，也不想再回去受罪了！”

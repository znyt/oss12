　　叶少阳手托下巴，望着凤兮，微笑着说道：“坦白说，你比她漂亮的多。”
<br/>　　凤兮开心地捂嘴笑起来，招呼大家重新坐下。叶少阳迫不及待地问道：“哦对了，那我为什么成通缉犯了？这也是你们弄得？”
<br/>　　“对啊，我们想要找你，但是在这时代，基本上就是大海捞针，因此崔郎就给我出了个主意，利用我父亲的权势，在这一带的县城里贴告示，但是这个年代没有寻人启事这种事，会很突兀，为了影响更大，我们就让县长去找警察所，弄了个通缉犯的告示，这样对警察所来说，也会名正言顺……我们这么做，本来也是尽最大努力，没有想到竟然真的把老大找来了，也真的是运气好。”
<br/>　　叶少阳无语，“也亏得你们点子多，居然想到这么狠的办法……不过我想问，为啥是拐带少女……”
<br/>　　凤兮和崔颖噗嗤一笑。“这个啊，是崔郎想到的主意，说是这样的罪名，跟容易让别人浮想联翩，会加深印象……”
<br/>　　“好吧，你们……我服。”叶少阳实在找不到什么语言来表示自己的心情。
<br/>　　之后，在凤兮的询问下，叶少阳也道出了自己的打算。凤兮听完之后，表示愿意跟随叶少阳一起去桃花山。
<br/>　　“那你父母这边……”
<br/>　　“我父亲去前线了，他们都住在那边，这边是他们以前的府邸，算是我们自己的家。”
<br/>　　凤兮算是自己半个门人，而今也是用人之际，叶少阳当然不会跟她客气。
<br/>　　聊了一会之后，凤兮叫来下人，给叶少阳三人安排了几间客房，去休息之前，叶少阳又问了凤兮一些关于白起和徐福的事情，可惜凤兮知道的也不多，徐福把他们送到这里来，重塑了肉身之后，立刻就离开了，对她提出的那些问题没做任何解释。
<br/>　　来到客房之后，叶少阳枕着双臂，躺在床上，开始思索之前凤兮跟自己说的事情。最让他感到震惊的，还是道风跟徐福的合作，去昭狱救人……救的还是白起这种重犯，阴司一定会震怒，道风虽然一向不服管，但这件事势必会给他带去巨大的压力。
<br/>　　道风，为什么要这么做？为什么要帮助徐福？
<br/>　　叶少阳实在想不通，但是通过这件事，他想到了一个细节：既然道风跟徐福合作，一起救出白起，那么道风不可能不向徐福打听自己的情况，按照正常逻辑，他肯定会要求徐福带他来这边救自己。既然他没来，那肯定是被徐福给说服了……
<br/>　　叶少阳很想知道徐福是怎么说服道风不来救自己、并且帮他一起额去干劫狱买卖。
<br/>　　叶少阳越想越气，别人都还算了，道风明明都见到徐福了，这贱人居然不想办法来接自己回去！叶少阳要疯了。
<br/>　　但是气归气，静下心来想，叶少阳也很清楚，道风这么做，肯定有他的理由，而且这理由一定是为了自己好。
<br/>　　这一点，他丝毫不怀疑。全天下都可以背叛自己，但是捉鬼联盟不会，道风不会。
<br/>　　从道风，叶少阳又想到了徐福，通过凤兮这件事来看，徐福对自己应该是善意的，只是，不知道要完成什么条件才能重新回到自己的时代。
<br/>　　叶少阳躺在床上，静静地想了很久，最后也只能是按照既定计划去做：先去桃花山。如果徐福通过凤兮告诉自己的没错，那在这一路上，肯定有什么事情在等着自己。
<br/>　　叶少阳深吸了一口气，眉宇之间的愁色减轻了许多，神色也变得更加坚毅起来。
<br/>　　第二天，叶少阳一行人就出发了，只不过队伍多了凤兮和崔颖。
<br/>　　凤兮找仆人弄来两匹马，一人一骑，叶少阳三人的马，凤兮也派人去到客栈里给牵了过来。五个人，加上美华和包子，一行人马不停蹄地赶往杭州。
<br/>　　在经过一些军阀和城池设置的关卡时，凤兮的身份给了极大的方便，一路上根本没遇到任何阻拦，一路飞驰，不到几天的工夫，便来到了杭州。
<br/>　　杭州城外，桃花山下，山下一带，有好几个镇子。那个时代也没有手机地图可以用，没法提前订酒店，叶少阳一行人只能骑着马在几个镇上晃悠，寻找可以打尖的客栈。
<br/>　　每个镇子上，都是有一两家客栈的，但是去问都住满了，道渊真人告诉他们，这些客栈基本上都被各大门派过来的人给包了。这些人中，大部分是受命于法术公会，前来参加龙华会的，也有一些是茅山、龙虎山这样的大门派，提前来打探消息，并且互相串联的，也住在这里，还有就是法术公会的人。
<br/>　　不管是投靠法术公会的门派，还是反对的门派，都只是一种倾向性，互相之间虽然看着不顺眼，但明面上也没撕破脸皮，就算是坚决不妥协的几大门派，跟法术公会在明面上也会有所往来，不可能真刀真枪地去斗。所以整个法术界看上去还是很平静的，实际上却是暗流涌动，只差一个恰当的事件将这些矛盾激化，到那时候，整个法术界将四分五裂，陷入巨大的纷争之中，没有任何门派可以独善其身。
<br/>　　来到一个镇上，还没进去，立刻有一些村民打扮的上前询问他们要不要住宿，叶少阳起初还纳闷这些小镇客栈的怎么跟百年后似的，还主动揽客，一交谈才知道，这些人并不是客栈的，而是当地村民，因为最近总有人来投宿，但是客栈都住满了，因此一些比较活络的村民，就想到了这个赚钱的门路，一旦有外地人进村，立刻就上去询问。
<br/>　　至于这些打扮奇怪的人是来干什么的，当地人虽然好奇，但也不是特别感兴趣。
<br/>　　叶少阳一行人跟一个村民谈好了价格，然后跟着他走，住进了自家的民宅里。
<br/>　　住下之后，叶少阳跟房东打听，得知镇上除了客栈住满了，连很多村民家里都住进了外地人，看来都是各个门派的。

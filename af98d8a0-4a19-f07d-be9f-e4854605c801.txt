　　声音落下的同时，一道人影如同鬼魅般，出现在众人的眼前。
<br/>　　“小尘！”
<br/>　　苏蔓先是一愣，继而确认不是自己的幻觉，顿时激动的高喊了一声。
<br/>　　“你是什么人？竟然敢擅闯我们药王宗！”
<br/>　　那四名药王宗弟子，看到突然出现的叶尘，顿时纷纷大喝了一声。
<br/>　　“哼！”
<br/>　　叶尘冷哼一声，
<br/>　　“杀你们的人！”
<br/>　　声音落下，叶尘身上猛然升起一股强大的杀气，犹如泰山压顶一般，朝几人狂涌而去！
<br/>　　轰隆！
<br/>　　那四名药王宗弟子，只有筑基境的水准，如何是叶尘的对手？
<br/>　　单单是一道杀意，都不是他们可以抵挡的！
<br/>　　噗噗！
<br/>　　噗通！
<br/>　　四人在叶尘的强大威压之下，再也承受不住，直接全部跪倒在了地上。
<br/>　　就连后面的公孙茂，也不由得一脸倒退了数步，脸色大变！
<br/>　　“你、你就是叶狂仙？”
<br/>　　虽然，公孙茂之前听说过有关于叶尘的事迹，总觉得一个不到二十岁的少年，纵然再天才，又能又多强，多半是以讹传讹罢了。
<br/>　　如今正是感受到了叶尘的强悍实力，这才意识到，自己错的有多么离谱！
<br/>　　叶尘嘴角微微一扬，
<br/>　　“死吧！”
<br/>　　他也懒得再和这些人废话，直接一巴掌便拍了过去！
<br/>　　嘭嘭嘭彭！
<br/>　　那四名药王宗弟子，瞬间被叶尘强大的掌力，拍成了一团团血雾，死的不能再死了。
<br/>　　而那公孙茂，实力比四名药王宗弟子，要强出不少，而且又有防备，全力阻挡之下，虽然也狂喷了一口鲜血，倒也勉强保住了一条老命。
<br/>　　“小子！你敢在我们药王宗杀人，今日你必死无疑！”
<br/>　　公孙茂在说出这话的同时，身体也在迅速暴退，眼看就要逃离大牢。
<br/>　　“死！”
<br/>　　叶尘抬手便是一记虚空神拳。
<br/>　　轰！
<br/>　　可怜那公孙茂，还以为直接已经逃过一劫，猛然感觉到身后，传来了一道强悍无匹的掌力，瞬间将他淹没其中，
<br/>　　“不！！”
<br/>　　公孙茂不甘的大吼了一声，整个人瞬间被叶尘一拳砸成了一滩肉泥。
<br/>　　“嘶~~”
<br/>　　在一旁目睹了这一切的祝小白，看到这一幕，顿时倒吸了一口凉气，同时满脸的惊骇之色。
<br/>　　他虽然一直觉得叶尘的实力深不可测，可是却怎么也没有想到，竟然强大到了如此地步！
<br/>　　那公孙茂乃是药王宗长老，而且在众多长老之中，实力也在中上游，便是他的那位天才哥哥，只怕想要赢此人，也未必那么容易。
<br/>　　可是如今，竟然被叶尘轻松秒杀！
<br/>　　就算是天榜前几位的大佬，恐怕也未必能做到如此随意吧？
<br/>　　这人的实力到底有多强？
<br/>　　祝小白已经不敢想象。
<br/>　　……
<br/>　　轻描淡写的解决了公孙茂等人，叶尘随手一抓，便已经将苏蔓从刑架上解救下来。
<br/>　　“对不起，蔓姐，我来晚了，你没事吧？”
<br/>　　苏蔓重获自由后，一头便扎进了叶尘的怀里，先是哇哇大哭了一阵，继而一对粉拳，在叶尘的胸膛上，如雨点般砸落，
<br/>　　“臭小子！你怎么现在才来救我？你知不知道，你老姐我差点被他们……呜呜呜~”
<br/>　　叶尘暗暗好笑，
<br/>　　“蔓姐，你这可冤枉我了！我知道你被人掳走之后，一刻也没敢耽搁，走吧，我先带你离开这里再说！”
<br/>　　说着，叶尘一手搂着苏蔓的纤腰，同时用真元包裹着祝小白，身体一晃，便已经出了药王宗的大牢。
<br/>　　叶尘原本打算，先将苏蔓两人送出药王谷，然后在回来找药王宗的人算账。
<br/>　　然而，
<br/>　　他们三人才刚刚踏出药王宗的大牢，便有人从四面八方包围了过来。
<br/>　　“是谁胆敢擅闯我药王宗？”
<br/>　　说话的是一名中年男子，身后还跟着十几名药王宗弟子。
<br/>　　叶尘也不说话，直接随手一掌拍落，
<br/>　　那中年男子比公孙茂还多有不如，哪里抵挡住叶尘一掌？
<br/>　　彭！
<br/>　　瞬间便被叶尘一掌拍成了肉渣。
<br/>　　甚至连同他身后的那十几名药王宗弟子，也顿时死伤一片。
<br/>　　“啊！！”
<br/>　　侥幸逃脱一劫的药王宗弟子，见来人如此强悍，顿时纷纷惊呼了一声，拔腿便欲逃跑。
<br/>　　叶尘屈指连弹，犹如手弹琵琶，
<br/>　　嘭嘭嘭！
<br/>　　一道道剑气射出，剩下的几个漏网之鱼，也被叶尘轻松斩杀。
<br/>　　“是什么人这么大胆？敢在我药王宗撒野！”
<br/>　　一道充满威严的声音，陡然在天空之上响起，
<br/>　　继而，
<br/>　　叶尘猛然感应到好几道，远在那公孙茂之上的气息。
<br/>　　很显然，药王宗真正的高手，也已经赶了过来！
<br/>　　叶尘顿时眉头一皱，
<br/>　　他如果独自一人，自然毫不畏惧，可是如今带着苏蔓和祝小白，情况就有些棘手了，难免束手束脚，难以发挥出实力。
<br/>　　“先将他们两人带离此地，这笔账以后再算！”
<br/>　　叶尘迅速做出决定，当即将速度催动到极致，打算先带着两人逃离药王谷。
<br/>　　然而就在这时，
<br/>　　怀里的苏蔓，却忽然急促的喘息起来。
<br/>　　“蔓姐，你怎么了？”
<br/>　　叶尘并不知道，苏蔓被人逼服了毒药，心中暗暗好奇，一边朝药王谷外面飞驰，一边开口询问道。
<br/>　　但是令叶尘感到奇怪的是，苏蔓一言不发，反而整个人紧紧贴在叶尘的身上，开始不停的扭动起来，同时还夹杂着诱人的呻吟声。
<br/>　　叶尘慌忙低头一看，这才发现苏蔓整个人的状态，极为不正常。
<br/>　　只见她双腮泛红，眼角含春，柔软的娇躯，更是犹如一滩烂泥般，死死贴在叶尘的胸膛之上，口中喃喃低语，
<br/>　　“小尘，我、我好热~好难受~”
<br/>　　叶尘这才反应过来，顿时又急又气，
<br/>　　“药王宗的这帮杂碎，竟然给蔓姐吃了那种淫秽之物！你们给我等着，我一定要将你们连根拔起！”
<br/>　　现在的当务之急，是要先替苏蔓解毒。
<br/>　　或许是因为心绪受到了干扰，叶尘一晃神，竟然走错了方位，一头扎进了一处绝谷之中。
<br/>　　“哈哈！这小子进入万毒窟了！真是自寻死路！追！”
<br/>　　那道威严的声音再次响起，
<br/>　　叶尘顿时脸色微微一变，
<br/>　　因为他发现，眼前这个绝谷，周围竟然还有阵法禁制，虽然他可以破解，可是却需要一些时间。
<br/>　　事到如今，他已经别无选择，只好一条道走到黑，继续朝前方深入。
<br/>　　然而很快，三人已经来到这个绝谷的尽头，前方无路可走，唯有一处深不见底的洞穴，透着一股诡异的气息。

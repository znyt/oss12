　　人们常说天涯海角，却不知道真正的天涯海角在何处？
<br/>　　三十三天尽头，无进混沌虚空，便是真正的天之涯！天之涯处，便是混沌虚空。
<br/>　　不过，那靠近天界的混沌虚空与真正的天外混沌又有一些区别。真正的混沌，那可是非强大的大罗金仙以上者不可入，其他的，不管是仙、魔、神、妖，进去不过片刻，便会被同化，化为混沌之气。如之前混沌魔神们的所在之地。
<br/>　　而这接近天界的混沌虚空，那是盘古开天之时已经破碎过的混沌，混沌的威力已经被极度减弱了，更有世界外层的保护，与天外混沌对峙，这才出现如今这种情况。地水火风不停涌出，能量混乱。
<br/>　　万劫魔尊望着那天之涯，混沌不断崩碎，化为地水火风，不断补充进天地之间。他已经不是第一次来天外混沌了，他自是知道，正是因为这些地水火风能量的补充，进入天地，化为灵气，才使得这天地间的灵气不会枯竭。
<br/>　　面对此情此景，万劫魔尊不得不感慨一声，盘古大神，不愧是能够开天辟地的存在。他的实力，恐怕早就半只脚踏入大道境了。
<br/>　　一路搜寻而来，不到片刻，万劫魔尊便找到了天庭宝库所在。没有费多大的劲，在万劫魔尊灭了守门的几人后，便破门而入。
<br/>　　看着眼前五光十色的宝库，里面的宝物简直应有尽有，有洪荒最为顶级的神材、神铁、有各种顶级的仙果道果、还有各种兵器、妖族功法……等等，看得万劫魔尊眼中直冒精光。
<br/>　　这可是妖族亿万年收集到的顶级宝物，若是不将之取走实在是对不起他们啊。
<br/>　　万劫魔尊毫不客气，如同是在自己家中，他大袖一挥，对着那宝库之中的各种宝物刷去，尽管上面有着妖族大神通者布下的禁制，但这都挡不住他的手段。
<br/>　　不到一时三刻，万劫魔尊便将妖族宝库洗劫一空，连根毛都没有剩下。
<br/>　　“不错，这些宝物应该足够劫魔殿发展一段时间了吧。”万劫魔尊低声喃喃道。
<br/>　　之后，他一个闪身，便消失在空中，不知所踪。只留下一地被破坏的惨败不堪的妖族宝库。
<br/>　　而此时，妖族与巫族还在开战，大战正酣，谁也没有发现妖族宝库失窃。
<br/>　　“杀杀杀！”
<br/>　　天庭之上，到处都是战场，血腥、屠杀、杀机充斥满天。巫族之人肉体强大，悍不惧死，力量无穷，而妖族却占据数量优势，以各种法宝、天赋神通攻击。
<br/>　　“啊！”“吼！”“杀啊！”
<br/>　　各具优势，势均力敌，战况惨烈无比，喊杀声和惨叫之音不绝于耳！
<br/>　　轰隆隆！
<br/>　　两族的准圣与祖巫都快要大战到星空之上了。太一手持混元剑，到了后期，甚至一人大战四大祖巫，强悍无比，四大祖巫也御使各种法则之力，相互冲击，发出轰鸣声。
<br/>　　“锵！”
<br/>　　帝俊手持金色帝剑，与帝江，强良、奢比尸大战起来。这一战打了三个月，都还没有停下来！
<br/>　　天庭早已血流成河，战场早已不限定在天庭里了，不周山，洪荒大地，到处都在大战。无数巫妖强者陨落，许多巫族更是以一种极端的方式——自爆，来杀敌。
<br/>　　“嘶！”
<br/>　　好多关注此战的洪荒大神通者看得眼皮直跳，只绝得这个世界都疯了。他们纷纷被巫、妖两族展现出来的凶威给震慑了，这样的战争，简直是不死不休，比远古时期的龙、凤、麒麟三族大劫之战也丝毫不差。
<br/>　　星空之中，除了太阳、太阴以及三百六十五颗太古星辰之外，力量不可撼动，其他的星辰，被打碎了不知多少。许多大神通者，一个神通，便将无数星辰湮灭，简直是不可思议。
<br/>　　大战之中，谁也没有发现，万劫魔尊正在不周山地下一个空间之中，肆无忌惮的吞噬着天地之间的劫气、煞气、杀气等各种负面之气，他的修为在不断攀升。在他的旁边，有一个巨大的血池，正在牵引着巫、妖两族强者受伤、陨落之时流下的精血。
<br/>　　这个血池足足有三丈之大，但好似无底洞一般，不管来多少鲜血，它都能吞噬。同时在血池之上，散发着摄人的魔威，血池在翻滚、在沸腾，不断有精血被牵引过来，被炼化。
<br/>　　这是劫魔转生池！
<br/>　　只需要有足够的精血，以及劫魔之气，再加上邪恶之魂，便可以源源不断地产生劫魔。
<br/>　　劫魔就是万劫魔尊创出来不断破坏的一种魔，天地间只有杀戮，他才能更快的修炼。他创造出劫魔，为的就是不断地杀戮。
<br/>　　邪恶之魂越强，转生出来的劫魔就越强大，而且他们只能听万劫魔尊一人的话，绝不会背叛。至于邪恶之魂，在这吃人的洪荒中，还会少吗？
<br/>　　同时，在昆仑山的方向，元始一样关注着这场大战，对他来说，巫、妖两族谁死谁生都对他没有影响，他主要看的是巫、妖两族的最终底牌。
<br/>　　“帝江！尔等巫族，该死！”
<br/>　　帝俊面色冷峻，冷冷地喝道。他御使妖帝剑，以河图洛书护身，战帝江、奢比、强良等祖巫，一时部分胜负。其他妖族成员也被拖住了，但妖族群众甚多，祖巫数量又少，是故，形式对于巫族来说，还是颇为不利的。
<br/>　　“哼！帝俊，果然厉害，不过我祖巫也不是那么好欺负的。”
<br/>　　帝江怒吼，面色阴沉，大声喝道：
<br/>　　“诸位兄弟，速速结都天神煞大阵，让他们知道我盘古正宗的厉害！”
<br/>　　帝江见形势对于巫族越来越不利了，虽然他们修炼了《巫神决》，但时间终究是太短了，此刻也不是讲个人主义的时候。
<br/>　　“吼！”
<br/>　　十二祖巫纷纷响应，都跳出战场之外，合力汇聚在一起，准备布出十二都天神煞大阵！
<br/>　　轰隆隆！
<br/>　　“起！”
<br/>　　十二祖巫大喝一声，只见那里陡然传来一阵轰鸣之声，帝俊、太一等人看去，便见祖巫们各自运转法则之力。顿时，虚空显化，金木水火土五行相生，时间、空间、雷、电、雨等十二中世界本源法则。
<br/>　　“盘古意志，不死不灭！
<br/>　　十二都天，神煞之威！”
<br/>　　“盘古都天，神煞大阵，现！”
<br/>　　随着帝江等人一声大喝，众人只见在帝江等人的背后，漫天煞气凝聚，十二祖巫竟然凝聚成一人，隐隐看去，吟哦浩瀚、广博、苍茫、亘古的虚影显现！
<br/>　　……

        孙莉莉仔仔细细，一张张查看诺筱颖提交上来的设计图。
<br/>
<br/>        诺筱颖就站在她的面前，紧张地等待着上司的答复。
<br/>
<br/>        当孙莉莉看完后，诺筱颖本以为她会很满意，却在无意间地抬眸时，撞上了孙莉莉那双略带怒火的眼睛。
<br/>
<br/>        孙莉莉突然将手里的设计图往办公桌上一摔，拍着办公桌，疾言厉色地呵斥道：“诺筱颖，你这抄得也太明目张胆了吧！”
<br/>
<br/>        “抄？！”诺筱颖一脸懵。
<br/>
<br/>        这间总裁办的设计图，是她独立完成的。
<br/>
<br/>        从地板到背景墙再天花板，从办公桌到书柜再到椅子，从地毯到沙发再茶几……
<br/>
<br/>        设计图里每一样家具，每一样摆件，每一样装饰，都是她消耗了时间，花费了心思，翻阅了资料，还请教了傅昼景，才原创设计完成的。
<br/>
<br/>        如今，孙莉莉却说她是抄袭的？凭什么？！
<br/>
<br/>        诺筱颖回过神来后，深吸了口气，不满地质问：“这是我的原创，孙副部长为何说我抄袭？”
<br/>
<br/>        孙莉莉不禁朝诺筱颖翻了个白眼，拉开一旁的抽屉，从里面拿出另一套设计图来，狠狠地甩在了诺筱颖的设计图上，怒吼：“自己看看！还有脸说自己没抄袭！”
<br/>
<br/>        诺筱颖连忙拿起设计图翻看了起来。
<br/>
<br/>        这套设计图也是一套办公室的装修设计图，虽然面积大小不同，但是与她为傅氏集团的新任总裁的办公室设计的装修图如出一辙。
<br/>
<br/>        除了沙发和茶几以及地毯不一样外，其余全部一模一样，就连其中几张局部效果图都是一样的。
<br/>
<br/>        “不、不可能……”诺筱颖看傻了眼，一边否认，一边摇头，难以置信，自己亲自设计的装修图，竟然会有人盗去反过来反咬她一口。
<br/>
<br/>        “这是崔素妮三天前就交给我的设计图，你说你不是抄袭了她的，那是抄袭了谁的？”孙莉莉冷斥，不禁再次对诺筱颖翻白眼，“亏我之前还对你刮目相看，没想到，你的人品就犹如你的学历一样！”
<br/>
<br/>        “我没有抄她的！一定是她盗了我的设计图！”诺筱颖急红了眼睛，理直气壮地替自己辩解。
<br/>
<br/>        孙莉莉见状，不得不一通内线电话，把崔素妮也叫进了她的独立办公室。
<br/>
<br/>        崔素妮走进来的时候，一脸惊怔地看了看孙莉莉，又疑惑不解地看了看诺筱颖，并站到了诺筱颖的身边，向孙莉莉欠了欠身，弱弱地问道：“孙副部长，您找我有什么事吗？”
<br/>
<br/>        “小梁说，是你盗了她的设计图。”孙莉莉往后靠在了老板椅上，双手抱臂，抬起头来审视着面前的这个两个员工。
<br/>
<br/>        崔素妮惊诧地看了诺筱颖一眼，满脸迷茫地反问：“孙副部长，这到底是怎么了？我盗小梁的设计图？您别冤枉我啊！”
<br/>
<br/>        “你两的设计图，就在这桌上了。你两自己说说看吧！”孙莉莉恼火地说道。
<br/>
<br/>        搞他们这行，借鉴别人的设计图是常有的事情，但是，这种明目张胆地抄袭，还抄得百分之九十以上的雷同，并且还是在同一家公司里，这种行为是不是太过分了点？！
<br/>
<br/>        崔素妮连忙拿起桌上的两套设计图对比着看了看，看完后，她便指着诺筱颖的鼻子，焦急又气愤地控诉：“诺筱颖，你怎么能抄袭我的设计图呢？”
<br/>
<br/>        “我没抄！是你盗走了我的图纸！”诺筱颖不禁恼火了起来。
<br/>
<br/>        崔素妮微微摇头，难以置信地斜睨了诺筱颖一眼后，瘪着嘴，委屈地向孙莉莉说道：“孙副部长，你可要给我做主啊！我这设计图，三天前就交到您手上了。您还说，我设计得非常令客户满意。如今，诺筱颖盗了我的设计图，抄得有模有样！还在这里反咬我一口！我委屈！”
<br/>
<br/>        “我没有！”诺筱颖瞪着崔素妮，咬牙切齿。
<br/>
<br/>        孙莉莉不想在这儿听她两费口舌，于是从老板椅上起身，走了出去：“你两跟我来！”
<br/>
<br/>        随后，诺筱颖和崔素妮互看互不顺眼地跟着孙莉莉走了出去。
<br/>
<br/>        孙莉莉让两人把他们的设计图文件打开，孙莉莉分别查看了一下，她两操作软件的时间。
<br/>
<br/>        这不看还好，一看便让孙莉莉更加恼火了。
<br/>
<br/>        诺筱颖的电脑里，软件上居然没有设计图的历史过程。
<br/>
<br/>        而她所有的资料，显示都是最近这一周才创建开始的，而崔素妮的资料，两周前就已经开始设计了。
<br/>
<br/>        这一目了然的事情，孙莉莉站直了身子，偏过头去质问诺筱颖：“诺筱颖，现在，你还有什么可说的？”
<br/>
<br/>        “不可能！我的电脑一定被人动过！”诺筱颖也是到现在才发现，之前她的初稿文件夹里的设计图资料全部不见了。
<br/>
<br/>        “啊！我想起来了！孙副部长，你一定得为我做主！诺筱颖她，她有动过我的电脑，我‘3DMAX’软件出了问题，还是她重新帮我装好的，而且，我还把我的电脑密码告诉过她。当时，大家都在，大家可以帮我作证！”崔素妮振振有词地说。
<br/>
<br/>        孙莉莉随即看了一眼集中办公室里的其他员工，只见他们纷纷为崔素妮点头。
<br/>
<br/>        诺筱颖顿时是哑巴吃黄连，有苦说不出，更是跳进黄河也洗不清了。
<br/>
<br/>        “诺筱颖，你是自己写辞职信，还是让我给你写辞退信？”孙莉莉疾言厉色地质问。
<br/>
<br/>        诺筱颖鼻子一酸，泪水在眼眶里打转，却强忍着不让自己在这些人面前掉泪。
<br/>
<br/>        孙莉莉不想看这种女人在自己面前装无辜，于是甩手转身，回到了自己的独立办公室里。
<br/>
<br/>        诺筱颖坐回自己的办公桌前，呆呆地看着自己的电脑。
<br/>
<br/>        周遭的同事，全都用异样的目光偷偷打量着她，在背后对她议论纷纷。
<br/>
<br/>        良久后，崔素妮又被孙莉莉给叫了过去，并把傅氏集团未来新任总裁的新办公室装修设计图任务重新委派给崔素妮去做。
<br/>
<br/>        崔素妮得知自己要给傅氏集团未来总裁设计办公室，心底顿时乐开了花。
<br/>
<br/>        其实，一开始，她就想接这个单子，但是，孙副部长却将这个单子派给了诺筱颖，让她失落了许久。

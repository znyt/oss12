　　嗖！
<br/>　　空中再次划过一道紫色剑芒。
<br/>　　噗！
<br/>　　莫长林的右臂，瞬间便被斩落了下来。
<br/>　　“啊！！！”
<br/>　　莫长林顿时惨叫了一声，捂着手臂，低着头，脸上露出一抹恨色，不过很快便又消失不见，恭恭敬敬冲着而出行了一礼，
<br/>　　“多谢……叶前辈饶命之恩！！”
<br/>　　虽然他嘴上这么说，但是眼眸深处，却透着一抹恨色。
<br/>　　虽然在修真界中，有一些能够让断臂重生的法门，可是经此一事，他的实力必然大大削减，而且当着这么多人的面，向叶尘苦苦求饶，也令他颜面尽失，这个流光派掌门之位，怕是做不久了！
<br/>　　叶尘自然也察觉到莫长林的怨恨之意，不过他才懒得去理会，随手朝一旁的云灵儿一挥，后者便被一股无形的力量托起，不由自主的就站了起来。
<br/>　　“带我去传送阵！”
<br/>　　叶尘没有废话，他要立刻离开天狼星域，前往更高级的星域。
<br/>　　云灵儿连忙点头称是。
<br/>　　……
<br/>　　很快，
<br/>　　在云灵儿的带领下，叶尘来到流光派设立的传送阵前。
<br/>　　“这个传送阵，最远可以传送到哪儿？”
<br/>　　叶尘直接开门见山道。
<br/>　　云灵儿当即解释道：
<br/>　　“叶前辈，这是三级传送阵，最远可以传送到天玄星域！不过，天玄星域可是赤阳府的地盘，您刚刚杀了南日山，所以还是避开吧！”
<br/>　　叶尘听到这话，也不由得眉头微微一皱，
<br/>　　天玄星域，应该周围数十个星域中，级别最高的星域。
<br/>　　即便是如此，恐怕也至少好几日才能抵达，而叶尘想要前往更高级的星域，则必须需要更高级的传送阵。
<br/>　　而高级传送阵，必然都掌握在更强大的势力手中。
<br/>　　倘若叶尘绕开天玄星域，就只能在一些低级星域之前，不停的穿梭，必然要走很多弯路，如此一来，就太浪费时间了。
<br/>　　而他现如今，最缺的就是时间！
<br/>　　一念及此，叶尘摆了摆手，
<br/>　　“不必了！天玄星域这么大，赤阳府想要找我的麻烦，也没有那么容易，就去天玄星域！”
<br/>　　云灵儿见叶尘如此坚决，也不敢相劝，只好一脸黯然的点了点头，往手腕上的乾坤镯一拍，取出了几块上等灵石，放入传送阵的凹槽之中。
<br/>　　不一会，
<br/>　　注入灵石后的传送阵，被云灵儿施法激活，开始运转起来，
<br/>　　“叶前辈，传送通道已经打开！”
<br/>　　叶尘点了点头，却并没有着急进入其中，右手忽然一翻，掌心中已经多出了一面镜子，赫然正是那柄，从岛国羽鹤手中得来的八咫镜。
<br/>　　“灵儿姑娘，我这次出门匆忙，没带什么东西，这个送你，就当做是见面礼吧！”
<br/>　　说完，将八咫镜直接丢给了云灵儿，然后便转身踏入传送阵之中，很快消失不见。
<br/>　　云灵儿抱着那面镜子，足足愣了好一会，忽然美眸大亮，露出浓浓的惊喜之色，
<br/>　　“这……这居然是一件圣器法宝？这位叶前辈……也太大方了吧？”
<br/>　　要知道，在他们天狼星域，即便是宗主一级的大人物，所使用的法宝，也不过是灵器级别而已。
<br/>　　叶尘一出手就是一件防御圣器，而且在圣器中，应该也品阶不低。
<br/>　　这要是让其他人知道，估计非得抢破脑袋脑袋不可！
<br/>　　云灵儿又惊又喜，慌忙将八咫镜塞入乾坤镯之中，准备回去之后，私下再仔细研究其中奥妙。
<br/>　　另外一边，
<br/>　　莫长林和流光派等一众长老，在确定叶尘已经离开天狼星域之后，顿时全都松了一口长气。
<br/>　　“可怕！太可怕了！此人到底是什么来头？竟然一口气斩杀了南日山和数十名烈日宗高手！”
<br/>　　“还好他没有追究我们流光派，否则即便玄离师兄在门中，我们也无力抵抗此人，非得被他灭门不可！”
<br/>　　“真是侥幸逃过一劫啊！”
<br/>　　……
<br/>　　众人纷纷感慨着，忽然一道冷笑声响起，
<br/>　　“哼！你们以为此事，这么容易就结束了吗？”
<br/>　　说话之人，赫然正是之前，被叶尘废掉一条胳膊的流光派掌门莫长林。
<br/>　　莫长林又低头望了一眼自己右臂的伤口，眼眸中泛起一丝寒意，冷冷道：
<br/>　　“诸位可不要望了，那烈日宗背后的靠山是赤阳府，还有南日山的舅舅，乃是赤阳府三大太上长老之一，若是让他得知，南日山死在我们流光派，你觉得他会放过我们吗？”
<br/>　　“这……”
<br/>　　众人听到莫长林的这番话，顿时纷纷脸色大变，
<br/>　　诚然，叶尘虽然厉害，但也仅仅只有一人，而且未必是化神境的强者。
<br/>　　而那赤羽天君，可是实打实的化神境强者，而且背靠赤阳府，绝对是一方霸主级的大人物。
<br/>　　莫说一个小小的流光派，便是将整个天狼星域灭掉，那也是分分钟的事情！
<br/>　　相比之下，这才是真正的灭顶之灾啊！
<br/>　　“掌门言之有理！我们整个天狼星域，其实都是赤阳府的附庸，那位赤羽天君，我们说什么也得罪不起！”
<br/>　　“如今南日山死在我们流光派，虽然不是我们所为，却也难逃干系啊！”
<br/>　　“这可如何是好？”
<br/>　　……
<br/>　　众人面面相觑，一时间都没了主意，个个面如土色。
<br/>　　最终，
<br/>　　还是莫长林缓缓开口道：
<br/>　　“办法其实也不是没有！我们现在要做的，就是要尽快撇清和那姓叶的关系，和烈日宗站在同一战线，这样才能夹缝中，求的一线生机！”
<br/>　　莫长林这话说出，众人顿时再次相顾愕然，一时间谁也没有接口。
<br/>　　没办法，之前叶尘斩杀南日山和烈日宗众高手的情景，还都历历在目，如今都还心有余悸。
<br/>　　莫长林见众人犹豫不定，顿时急了，
<br/>　　“诸位可要想清楚了，那小子虽然厉害，可是和赤阳府相比，也不过就是蝼蚁一只罢了！”
<br/>　　“只要我们将此子的行踪，上报给赤阳府，不但可以将我们自己撇清，还能借赤阳府之手，除掉这小子，何乐而不为？”
<br/>　　众人沉吟了片刻之后，最终纷纷点头同意。
<br/>　　……
<br/>　　叶尘自然不知道，他前脚刚走，流光派的高层，竟然就开始阴谋算计于他！
<br/>　　此时的他，正脚踩紫琼剑，在空间通道中穿梭着。

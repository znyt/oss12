现在这本医典就像是一把钥匙，为夏夕绾打开了全新的篇章。
<br/>
<br/>        这时夏夕绾脑袋里电闪雷鸣了一下，这上面所有的针法融合起来就像是一张太极八卦图，更像是…那个宝箱的钥匙。
<br/>
<br/>        林水瑶留在枢密研究院里的那个宝箱，夏夕绾去试过，血气翻涌输在了第三关，这上面的针法好像就是接下来的钥匙。
<br/>
<br/>        夏夕绾闭了一下眼，脑袋很乱，这些玄妙深奥的针法她一时之间还没有办法全部领悟，很混乱。
<br/>
<br/>        这时“叩叩”的敲门声响起了，叶翎走了过来，“绾绾，你怎么了，头疼就不要看了，说不定在某个时刻你就全部领悟了，整个人豁然开朗。”
<br/>
<br/>        夏夕绾点头，她收起了医典，医学都是需要时间来学习领悟和临床实践的，没有办法速成，今天她有点累，打算先休息。
<br/>
<br/>        “绾绾，两天后你就走了，这两天你有什么安排，我陪你一起逛逛街？”叶翎笑道。
<br/>
<br/>        夏夕绾摇头，拉住了叶翎的小手，“最近发生了很多事，这座海城毕竟是我生长的地方，我不想逛街，这两天我只想一个人走一走看一看，因为也许我以后再也不会回来了。”
<br/>
<br/>        叶翎认同，她完全尊重夏夕绾的每一个决定，“绾绾，我很快就会帝都城找你的。”
<br/>
<br/>        “翎翎，你也去帝都城？”
<br/>
<br/>        “是啊，这座海城也没有我的亲人了，也没有值得我留恋的东西，我们姐妹可以一起去闯帝都城啊，而且我接了一个新代言，Fly珠宝全球代言人，我要在帝都城工作一段时间。”
<br/>
<br/>        “Fly珠宝？”夏夕绾瞳仁狠狠一缩。
<br/>
<br/>        “是啊，绾绾，你怎么这么大反应，你不会没有听说过Fly珠宝吧，Fly珠宝诞生在三十多年前，一经诞生就惊艳了整个珠宝界，后来Fly每一季的新款都像是刮过的一阵龙卷风被那些名媛贵妇抢购一空，那真是空前的盛况，这些年Fly晋升珠宝殿堂，成为了经典，听说Fly创始人是一个传奇人物。”
<br/>
<br/>        夏夕绾当然听说过Fly，那是陆寒霆的妈咪一手所创的珠宝品牌。
<br/>
<br/>        叶翎见夏夕绾对Fly感兴趣，所以就将几本时尚杂志递了过去，“这就是Fly创始人，也就是一代珠宝教母柳招娣。”
<br/>
<br/>        夏夕绾垂下纤长的羽捷，杂志封面上是一个女人，柳招娣。
<br/>
<br/>        柳招娣生的好美，纵容夏夕绾自己也是一个美人，但见到柳招娣的时候还是呼吸一滞，杂志上的柳招娣穿着一身红裙，那张脸堪称绝色倾国，每一个线条都像是天工雕琢，拼凑在一张脸上简直让人忘记呼吸。
<br/>
<br/>        “绾绾，柳招娣很美吧，这是她四十岁拍的杂志，但是她的脸上一点都没有岁月的痕迹，简直逆天生长，说她二十岁都不为过，当年她可是帝都城的第一美人。”
<br/>
<br/>        这个柳招娣就是陆寒霆的后母，也就是陆子羡的亲生母亲。
<br/>
<br/>        叶翎甚至整个帝都城的人都不知道当年柳家有二女，柳招娣和柳璎珞，她们进行了一场换位人生。

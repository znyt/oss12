        &lt;h3&gt;第三卷第二百六十九章  解疑与误会&lt;/h3&gt;
<br/>
<br/>        陈子昂说了自己的决定，让江俞轩心里通透了，他们原本就应该一起奋斗的，当初的豪言壮语还在耳边萦绕，他还是原来的他，而她也是原来的她。
<br/>
<br/>        易倾秋离开会议室去工作了，留下江俞轩和陈子昂。
<br/>
<br/>        “明天宣布决定吧。”陈子昂对江俞轩说。
<br/>
<br/>        “你真的答应我去甸城？”江俞轩不确定的问。
<br/>
<br/>        “你是江家大少，有名的江家未来的接班人，我不答应你你不是一样有能力去甸城吗？”陈子昂笑微微的看着江俞轩。
<br/>
<br/>        江俞轩一脸的忐忑和不确定。
<br/>
<br/>        陈子昂沉默了一下继续说：“从明天起，你和陈子昂没有任何关系了。也和陈子昂的公司没有任何关系了，明天的新闻会发布江家大少辞去某公司副总裁之职，专心江家产业。”
<br/>
<br/>        “你......”江俞轩觉得好似一盆凉水从头泼到脚，他浑身都冷透了，他紧紧的咬着唇，看着陈子昂的眼神充满了悲哀，难道我们连在一起工作的机会都没有了吗？
<br/>
<br/>        “最毒不过妇人心，陈子昂啊陈子昂，你果然冷血冷情。”
<br/>
<br/>        “俞轩。”陈子昂站起来，她看着江俞轩那痛苦的模样，心里有些疼，她静静的看着江俞轩，抬起了手，伸到江俞轩的面前，终于是在离江俞轩五公分远的地方垂下来了。
<br/>
<br/>        “我这样做是有目的的，我去甸城，张函和张倩楠都会过去，闫欢已经提前过去了，甸城是一个不安分的地方，我知道你会去，而且你也有这样的打算了。我们还会在一起奋斗，不过是从明面上转换了一个位置而已。”陈子昂终于是不忍心看着江俞轩难过，说出了自己的想法。
<br/>
<br/>        江俞轩看着陈子昂一言不发，这一段时间没有和陈子昂联系，她到底发生了什么？
<br/>
<br/>        “第一：你和张倩楠已经结婚了，我不想你们之间有任何误会产生。第二：去了甸城，你和我撇清关系有很多有利的条件，最起码你不会受到排挤，这样你才能更好的帮助我。”陈子昂看着江俞轩：“你明白吗？”
<br/>
<br/>        “你这是想和我撇清关系，我不明白！”江俞轩从陈子昂的话里知道了很多信息。
<br/>
<br/>        陈子昂微微一笑：“你明白的，你怎么会不明白我呢。”
<br/>
<br/>        “明天我来召开新闻发布会吧。”江俞轩听了陈子昂的解释，心里豁然开朗，张函去甸城不可能仅仅是为了陈子昂，陈子昂也不仅仅是为了找哥哥，他们肯定是有重要的事情要做。
<br/>
<br/>        张函和陆家，他和张家，他们形成了一层裙带关系，而陈子昂不愿意破坏这层关系，如果他还在陈子昂的公司继续工作下去，张家和陆家是不会将他拉到统一战线上的。
<br/>
<br/>        陆家如果知道陈子昂有了张函孩子的事情，势必会有所行动，江俞轩现在能做的也只能是暗暗的守护她了。
<br/>
<br/>        “好！对了，今天上午的新闻发布会很成功啊，你和文文真的是太帅了。”陈子昂笑得很灿烂，似乎刚刚他们在讨论一件与自己无关的事情。
<br/>
<br/>        “我一直都很帅啊。难不成你今天才发现我帅？”江俞轩也从悲伤中回过神，他刚才真是坐了一趟过山车，落地了才发现自己只是虚惊一场。
<br/>
<br/>        “反正，某人从始至终一直都很自恋。”陈子昂瞪了一眼江俞轩。
<br/>
<br/>        “子昂，我告诉你，我要买婚房，张倩楠竟然不同意。”江俞轩开心的对陈子昂说。
<br/>
<br/>        “你怎么这么高兴？”陈子昂感觉很奇怪，江俞轩又不是没钱的人，张倩楠不买婚房，他竟然如此开心？
<br/>
<br/>        “我觉得我和张倩楠目前不适合在一起。”江俞轩看着陈子昂的眼睛，她的眼睛还是一如既往地明亮，还是像布满星辰的碧空，纯净而幽深。
<br/>
<br/>        “怎么不合适在一起？你们都结婚了，现在才说不合适在一起，拿婚姻当儿戏啊？”陈子昂微微皱着眉头。
<br/>
<br/>        “我和她之间缺一些东西，况且我们只是名义上的夫妻。”江俞轩解释着。
<br/>
<br/>        “缺什么东西？”陈子昂原本是走着的，站下来看着江俞轩。
<br/>
<br/>        “一种叫爱情的东西。我对张倩楠没有动心的感觉，我只是一味的接受她温柔的付出，我以为那就是爱情。现在冷静下来了，我才发现，那根本就不是爱情，那只是习惯！习惯是可以改的，但是爱情没有办法改。”江俞轩这一段时间算是明白了自己的心，尤其是张倩楠骂他的话，如醐醍灌顶。
<br/>
<br/>        陈子昂默默的听着江俞轩的话，沉默不语，她垂下眼眸，长长的睫毛抖动着，当习惯深入骨髓的时候如何能改变？就像我习惯了你在我身边，你不在的时候我是有多么的想念，想念你的笑，想念你的好，想念你霸道而温柔的絮絮叨叨。
<br/>
<br/>        “而张倩楠也有可能发现了这个问题，所以她拒绝和我在一起，她在逃避。”江俞轩没有发现陈子昂微妙的变化继续说。
<br/>
<br/>        “那你现在准备怎么办？”陈子昂轻声的问江俞轩。
<br/>
<br/>        “凉拌！大家都冷静一段时间，好好的理理自己的心，想一想到底想要什么！”江俞轩似乎是对自己说也似乎是对陈子昂说，都给大家一点时间吧，余生还很长，我们都别着急。
<br/>
<br/>        “渣渣轩，你最好是找张倩楠好好谈一谈，张倩楠是一个不错的女孩，家世也不错，能配得上你，是合适的妻子的人选。”
<br/>
<br/>        “你叫我渣渣轩？为什么？”
<br/>
<br/>        “因为你的想法有点渣，如果不爱又何必纠缠？这么多年当游戏人间呢？我最讨厌那种三心二意的男人了。”陈子昂语气有点不太好。
<br/>
<br/>        “我哪里渣了？我就是发现了真相，我就是没有爱过她，非要我装着去爱她？”江俞轩急了。
<br/>
<br/>        “你不爱她，那你爱谁？”陈子昂看着江俞轩问。
<br/>
<br/>        “我爱.......”
<br/>
<br/>        “啪。”江俞轩脸上被陈子昂拍了一巴掌。
<br/>
<br/>        “有一只蚊子，哟，没有打到，飞了。”陈子昂转过身继续朝前走，江俞轩没有发现陈子昂的脸上飞起了两朵红云。
<br/>
<br/>        江俞轩的话被陈子昂的那一巴掌拍回去了，他站在原地，特么的，连话都不让他说清楚，看着一天天拽得和二百五一样，原来也不过是一个懦夫，你让我把话说完会死人啊？
<br/>
<br/>        啊啊啊啊，江俞轩觉得他应该刺激一下陈子昂。
<br/>
<br/>        “我喜欢陈天天！”江俞轩冲着陈子昂的背影喊。
<br/>
<br/>        听到江俞轩那兀突的喊声，陈子昂的背影一滞，明显的看到了脊背在剧烈的抖动，过了很久，陈子昂的声音传过来：“喜欢了就去追啊，站在这里说有什么意义呢？走吧，晚上还要参加聚会，你难道想让大哥他们等你？”
<br/>
<br/>        他们之间就差了几步，江俞轩被自己脱口而出的话吓了一跳，可是说出去的话就如同泼出去的水，再也收不回来了。
<br/>
<br/>        他只是很生气陈子昂的态度，明明她的心里也是有自己的，为什么还要假装深沉？为什么不让他把话说出来？
<br/>
<br/>        他就是想气气她！
<br/>
<br/>        陈子昂在听到江俞轩说他和张倩楠没有爱情的时候，心里竟然微微的有一丝喜悦，她很鄙视自己那点龌龊的小心思，当江俞轩说：“我爱.......”的时候，她心里又有些害怕，然后伸手拍回去了江俞轩的话。
<br/>
<br/>        当江俞轩喊出了“我喜欢陈天天！”的时候，陈子昂的心彻底被冰冻了。
<br/>
<br/>        从此之后，真的是路归路，桥归桥！江湖路远，余生安好！
<br/>
<br/>        陈子昂，你还是努力的工作吧，感情什么的就不要去奢望了！

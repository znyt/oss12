　　见两人都被黑袍笼罩，根本看不出任何信息，老者指着前面的座椅，道：“两位，请坐。”
<br/>　　慕清霄丝毫没有坐下的意思，而是取出一个玉瓶，轻轻的放在老者面前。
<br/>　　“丹药？”
<br/>　　望着面前的玉瓶，老者略微一愣，眼中闪过一丝诧异，随后小心翼翼的拿起玉瓶，倒出色泽饱满的丹药闻了闻。
<br/>　　片刻后，老者抬头，惊讶的望着他，道：“极品清灵丹，两道丹纹，价值不菲，你是炼药师？”
<br/>　　“我是否是炼药师傅，貌似与本拍卖行没有关系吧？”黑袍下，慕清霄并没有特地压低声音，语气淡然至极。
<br/>　　老者一怔，微笑道：“阁下说的是，这里十颗极品清灵丹，两纹清灵丹，每颗价格在15-20w左右，竞拍过后，拍出三十万也很正常。”
<br/>　　慕清霄微微点头，再次拿出一个玉瓶，推到老者身前，道：“将这个也放在拍卖会上吧。”
<br/>　　“嗯？”
<br/>　　老者连忙拿起玉瓶，望着其中龙眼大小，通体碧绿的丹药，神情错愕，道：“四品聚气散，品质极品，真是好东西啊，你是四品炼药师？”
<br/>　　密室中陷入一片寂静，见慕清霄不回答，老者干咳两声，道：“极品聚气散底价设定在八十万左右，拍出一百万的高价也是有可能的。”
<br/>　　清灵丹虽价值不菲，但终究只是三品丹药，而四品丹药，价格就相差数倍，所以说越高等的炼药师，财富往往就越多，仅是一颗丹药的价格，就能让大多数普通修炼者绝望。
<br/>　　慕清霄微微点头，拍卖行会在拍卖价中抽取5%，他倒也不在意。
<br/>　　“十颗极品清灵丹，六颗极品聚气散，阁下拍卖的物品的价值，已经达到本拍卖行一级贵宾等级，拥有专属的包厢，午时，拍卖会就将开始，到时候贵宾卡，本拍卖行的小姐会替两位领路。”
<br/>　　说着，老者就拿出一张绿色刻有拍卖行标志的卡片递了过来：“取钱的时候，也需要阁下出示贵宾卡。”
<br/>　　慕清霄接过贵宾卡，不再停留，转身就离开密室，小医仙紧随其后。
<br/>　　待两人的背影消失在门口，老者拿着两个玉瓶，浑浊的双眼中闪过一丝奇异。
<br/>　　“不以真面目示人，能够拿出如此多的丹药，极品聚气散，此人可能是一位五品炼药师，门主肯定会感兴趣。”
<br/>　　......
<br/>　　......
<br/>　　拍卖会下午开始，离开拍卖行后，两人就在附近找了家客栈。
<br/>　　或许是因为有拍卖行的震慑，周围的秩序要比其他地方好的多，只是价格昂贵，一般人根本住不起。
<br/>　　慕清霄倒是不在意，两人的关系就差最后一步，于是就只开了一间客房，羞得小医仙不敢正眼看他。
<br/>　　除了住宿，其中的食物也昂贵，但味道也稍微对得起他的价格，住上一天估计就要五百金币。
<br/>　　午时，清脆的钟声，缓缓在拍卖行中响彻。
<br/>　　随着钟声的响起，拍卖行中喧闹的声音渐渐平息，慕清霄与小医仙已经抵达拍卖行内部，由小姐带领到了专属包厢中。
<br/>　　专属包厢，顾名思义，一级贵宾专属的包厢，其中都是一些颇有财富的主，至于一级贵宾上面，还有顶级贵宾和特殊贵宾包厢，环境往往对应着身份。
<br/>　　一级包厢设在二楼，倒不像一楼那般拥挤。
<br/>　　其中，贵宾的穿着与一楼也不是一个档次，总共二十张座位摆设在最边缘，坐在上面便能将一楼与拍卖台尽收眼底。
<br/>　　慕清霄与小医仙并没有取下黑袍，缓缓坐在座位上，倒是引来了其他贵宾的目光。
<br/>　　“诸位，想必是等的有些不耐烦了，既然如此，我也不再说官面话来讨嫌。”
<br/>　　拍卖台上，身着华贵的白发老者，视线扫过场内黑压压的人头，作为拍卖行经验丰富的拍卖师，他自然能摸清楚客户的喜爱。
<br/>　　“作为一年一度的拍卖大会，我们八扇门依旧不会让诸位失望。”老者轻轻拍手，朗声道：“我宣布，黑印城拍卖会，现在开始。”
<br/>　　随着老者朗声落下，巨大的水晶台豁然散发出一片刺眼的光亮。
<br/>　　半响过后，光亮消散，白发老者面前的拍卖台上，已经出现一把通体蔚蓝的长剑，在灯光的照耀下，反射出一股森寒光泽，看剑身上所流转的能量痕迹，显然是一把精心锻造过的魔核武器。
<br/>　　“此剑名寒锋，百年寒铁所铸，堪称削铁如泥，底价十万，诸位请吧。”
<br/>　　瞥了眼长剑，慕清霄就彻底失去了兴趣，对别人来说是宝，对他来说不过是柄烂铁，至于小医仙就更不用说了，根本不会用剑。
<br/>　　当然，寒锋剑对于他和小医仙没用，对于斗者来说，却是一把不错的武器，因此，老者话语刚落，拍卖场中就接连响起竞价声。
<br/>　　第一次的竞价，持续了几分钟后结束，寒锋也落入一名瘦小的斗者以十六万金币拍获。
<br/>　　在第一次竞拍顺利进行后，拍卖台上就不断出现各种各样的东西，武器，丹药，宝甲，斗技，药材......
<br/>　　慕清霄坐在软椅上，颇为失望，因为根本没有能让他心动的东西，唯一能让他瞧上一眼的，也就一些珍贵的药材，可却不是他炼丹所需的灵药。
<br/>　　拍卖会前期，没有出现太过宝贵的东西，竞拍的人都是一些没权没势的人，那些高级包厢中的根本就没有出过手。
<br/>　　就在这时，只见一名小姐端着银盘递到老者手中，银盘上有两个玉瓶，玉瓶中有着通体碧绿的丹药。
<br/>　　拍卖会上出现丹药，便是吸引了许多目光，毕竟，在斗气大陆上，丹药可是与功法斗技相提并论的重量级需求物品。
<br/>　　甚至有一些能够直接用于提升实力的丹药，锻造资质的丹药，更是让无尽修炼者趋之若鹜。
<br/>　　“清灵丹，想必诸位都不陌生，这玉瓶中，有着十颗极品清灵丹，效果极佳，每颗竞拍价十八万起。”
<br/>　　老者神秘一笑，道：“缓缓将玉瓶打开，一股浓郁的清香就弥漫整个拍卖场，这是六颗极品聚气散，能够轻松打破壁障，运气好的，说不定能推升到二星，每颗竞拍价八十万起。”

　　　　在楚风眠轻易灭杀了那其他三名黑袍圣者的一刻。
<br/>　　寒月漓跟夺花尊者，已经在空中，连连交手。
<br/>　　他们二人的境界，虽然还都是在神力境的范畴，但是真正在力量上，二人却都是不折不扣的生死境实力。
<br/>　　一招一式之间，便是极为轰动。
<br/>　　周围的空间都被打碎，化为了一阵阵的空间乱流。
<br/>　　“小美人，束手就擒吧！你虽然是先天神体，但是远远不是本少的对手！我的力量比起强大的多，你再反抗也没有任何意义！”
<br/>　　夺花尊者看着寒月漓，调笑连连。
<br/>　　说话之间，他也是手心一动，一把折扇，便是出现。
<br/>　　这一把折扇，威力极强，可是一件地级极品的灵器。
<br/>　　折扇一出，打开，在其中便是有着一副群虎啸山林的图像。
<br/>　　这其中的群虎，却是刻画的仿佛如同真的一样，随着夺花尊者一股灵力注入，顿时一只只猛虎，便是从这折扇图像中跳了出来。
<br/>　　这每一只猛虎的实力，可都不在巅峰圣者之下，出来一声声咆哮，便是冲着寒月漓，猛然扑了过去。
<br/>　　“这夺花尊者，实力倒是不错。”
<br/>　　楚风眠在下方，安静的看着夺花尊者跟寒月漓只见的战斗。
<br/>　　他并没有想要出手的兴趣，因为对寒月漓来说，这场战斗，也是一件不错的磨练。
<br/>　　一尊武者，力量并非是实力的根本，战斗经验，也是同样重要。
<br/>　　寒月漓如今差的，可不是力量，她身躯中的玄寒之力，已经是被楚风眠彻底的引了出来。
<br/>　　单纯按照力量来说，她的力量还要胜过这夺花尊者一筹。
<br/>　　不过因为这玄寒之力是突然引出的，白白得到如此大量的力量，让寒月漓一时间有些无法控制好。
<br/>　　这跟强者的战斗，正是足以让她积累经验来，好好的控制住自己的力量。
<br/>　　所以楚风眠这才不慌不忙的看着他们二人交战，这对寒月漓，也是一场锻炼，他不是出手相助。
<br/>　　“冰魄！”
<br/>　　面对着无数猛虎突然出现，向着她扑了过去。
<br/>　　寒月漓表情平静，手中剑锋，连连闪动。
<br/>　　“咻！咻！咻！”
<br/>　　这剑锋每一次打出，都是带出去其中一些小小的冰魄。
<br/>　　这一道道小小的冰魄，看起来甚至都不如手指大小。
<br/>　　但是当接触到那猛虎的时刻，便是猛然灵力催化。
<br/>　　瞬息之间，那无数猛虎，尽数都被这冰魄，冻着，成为一只只冻老虎。
<br/>　　随后更是随着寒月漓的剑锋，全部被斩破。
<br/>　　寒月漓毕竟也是曾经寒雪阁的天才，虽然她踏入神力境后，便是少有战斗，但是也并非是一个弱女子。
<br/>　　如今熟悉之下，她的力量，也是丝毫不弱，面对着夺花尊者，不落下风。
<br/>　　“哦？倒是有着几分能耐，不过这样更好，本少先将你镇压！先尝尝的你滋味，在将你身上的先天之力夺取再说！”
<br/>　　夺花尊者冷笑一声，一身灵力猛然爆发。
<br/>　　顿时手中折扇一扇，便是一股灵力，猛然轰击了出去，将寒月漓给逼退。
<br/>　　同时他也是暗地里，沟通灵力，悄然之间，埋伏了无数灵力。
<br/>　　随着寒月漓再次出手，他的嘴角，突然露出了几分诡异微笑，冷哼一声。
<br/>　　就在寒月漓向前走出的一瞬间，顿时在她的脚下，便是无数灵力，突然出现。
<br/>　　这一道道灵力，顿时化为了一座灵力的牢笼，将她彻底的困入在了其中。
<br/>　　“小美人，你的实力不错，可是空有一身实力罢了，你放心，本少虽然会夺取了你的先天之力，但是不会杀了你的，我会将你留在身边，好好的享用你的！”
<br/>　　夺花尊者舔了舔嘴角，仿佛兴奋到了极点。
<br/>　　他本来便就是好色如命之途，寒月漓可不止是有着玄寒之体，样貌也是令他陶醉。
<br/>　　这一下子将寒月漓镇压，他的表情，兴奋到了极点。
<br/>　　“放心，做了我夺花尊者的女人，我保证让你欲生欲死！”
<br/>　　夺花尊者大手一挥到。
<br/>　　“镇压！”
<br/>　　这灵力牢笼，便是瞬间开始收缩起来，要将寒月漓，彻底的困入在其中，彻底镇压。
<br/>　　但是几乎就在这一刻。
<br/>　　一个剑芒，却是无声无息的出现了。
<br/>　　在灵力牢笼之中的“寒月漓”，却是“砰”的一声，破碎了，化为了一团玄冰。
<br/>　　这其中的寒月漓，却是一个假人，只是以寒月漓的一股灵力，化为的样子。
<br/>　　真正的她，早就注意到了这夺花尊者的计谋，早早的准备好了对策。
<br/>　　突然出手，这一剑，快到极致。
<br/>　　顿时夺花尊者的胸口，一下子被破开，其中伤口处，都是开始结冰，血流不止，根本无法止血。
<br/>　　“该死！”
<br/>　　夺花尊者大惊，他也料想不到，寒月漓居然是根本没有被他镇压。
<br/>　　如今这突然一道剑光，划破他的胸口，顿时他的力量，都是开始减弱，其中无数圣者精血，都流淌了出去。
<br/>　　“冰连千峰！”
<br/>　　一招击伤了这夺花尊者，寒月漓也没有任何停滞，顿时再次施展出剑术来。
<br/>　　漫天冰晶，每一道都是锋利如剑，向着夺花尊者刺去。
<br/>　　她的脸上，面带怒容，要将这夺花尊者，彻底斩杀。
<br/>　　“哼！”
<br/>　　夺花尊者冷哼一声，自知不敌，身形一动，顿时躲开。
<br/>　　他的实力，本就跟寒月漓是在伯仲之间，如今他受了伤，自然不可能在是寒月漓的对手。
<br/>　　现在就算是想逃，他都没有办法，毕竟双方实力差不多的情况下，一方重伤，逃走都不可能。
<br/>　　夺花尊者的，眼神一动，猛然看见了下方的楚风眠，脸色突然露出几分阴冷笑容，直接向着楚风眠，冲了过去。
<br/>　　“小美人！你要是敢杀了我！我就杀了这两个废物，看你们之间的关系不同寻常，今日你若是想要让这两个废物活下去，就乖乖放我离开！”
<br/>　　夺花尊者一边冲着楚风眠抓来，一边转过头对着寒月漓开口道。
<br/>　　他便是打算，抓住楚风眠，来威胁寒月漓。
<br/>　　不过当他的话音一出。
<br/>　　寒月漓跟拓跋鼎的表情，都是变的极为诡异的许多，并没有任何的担心，反而是露出了几分似笑非笑的样子。

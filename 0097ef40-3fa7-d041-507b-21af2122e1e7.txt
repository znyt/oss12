        就在连星儿与郎安擦肩而过后，郎安忽然开口说道：“我闻到了血的味道。”
<br/>
<br/>        “血？！”裴冉身心一怔，不禁皱起眉头，伸手轻轻地拍过郎安的头顶，斥责道，“你伤过她一次又一次，刚刚还想伤她是不是？我警告你，你最好给我收敛点！不然……”
<br/>
<br/>        “我知道了。”郎安面无表情地打断了裴冉的话，淡然地转身往前走去。
<br/>
<br/>        裴冉随后跟了上去，又开始说起教来：“你最好不要再打连星儿的主意，要是让连老头知道了，到时候，别指望我再救你了。”
<br/>
<br/>        另一边，连星儿回到了连中凯的身旁，连中凯已经把那保温饭盒里的佛跳墙吃完了，正在拿纸巾擦着嘴巴，问道：“你是拉肚子了吗？上个洗手间，去了这么久。”
<br/>
<br/>        “外公，你这实验室里，有狼对不对？”连星儿索性凑到连中凯的身边，直接开门见山地咧嘴问道。
<br/>
<br/>        “什么狼？你还没从昨天的幻觉中醒过来啊？”连中凯淡定地反问。
<br/>
<br/>        连星儿见连中凯嘴硬还不承认，于是转过身去，拉下肩头上的衣领，说道：“我昨天洗澡的时候发现的，我肩膀后这里有两个獠牙印。”
<br/>
<br/>        “哪有？你眼花了吧！”连中凯忍不住翻了个白眼。
<br/>
<br/>        连星儿偏过头去看了看，赫然发现自己的肩膀后并无异常。
<br/>
<br/>        可是，昨天她洗澡的时候，明明有看到……
<br/>
<br/>        连星儿抬起手来，纳闷地在自己肩膀后摸了摸。
<br/>
<br/>        难道是她记错边了吗？
<br/>
<br/>        心想着，她又拉开另一边的衣领，在肩膀后摸了摸。
<br/>
<br/>        还是什么也没有！
<br/>
<br/>        连星儿不死心，又去照了镜子。
<br/>
<br/>        结果，她后背光滑干净，什么印子也没有。
<br/>
<br/>        “我说你这丫头，真会拿外公开玩笑。”连中凯笑呵呵地说道。
<br/>
<br/>        连星儿皱起了眉头，若有所思起来。
<br/>
<br/>        “好啦！别想多了，外公这儿没有狼。”连中凯一边说着，一边从白大褂的衣兜里掏出了一张名片，递给了连星儿，“这就是我那个朋友的名片，她是个非常好的心理医生。你给她打电话，就说你是我连中凯的外孙女连星儿，她就知道了。”
<br/>
<br/>        “外公，你事先打好招呼呢？”连星儿双手接过名片，好奇地问道。
<br/>
<br/>        连中凯会心一笑：“你昨天来找我，回去后，我就已经给你打好招呼了！你那朋友有什么事，可以随时随地联系她。她会无条件替你朋友治疗的。”
<br/>
<br/>        “嘻嘻，妈咪有些地方还真像外公了。我昨天一回去就说想吃佛跳墙，妈咪昨晚就开始给我准备了。”连星儿笑眯眯地说道。
<br/>
<br/>        “那可不，谁叫你是咱们连家的宝贝小公主呢？”连中凯忍不住伸手揉了揉连星儿的头顶。
<br/>
<br/>        连星儿拿着名片，笑不拢嘴地谢道：“那谢谢外公啦！”
<br/>
<br/>        “那你早点回去吧！”连中凯将保温饭盒递给了连星儿。
<br/>
<br/>        连星儿接过保温饭盒，微笑着点了点头。
<br/>
<br/>        离开实验室的时候，连星儿总感觉背后有一双眼睛正虎视眈眈地盯着自己。
<br/>
<br/>        当她回头时，不经意间地与站在一张合金门口的郎安四目相视。
<br/>
<br/>        他仍旧面无表情，两眼深邃，双手抄着裤袋，打着赤脚。
<br/>
<br/>        连星儿看着他，微微一笑，友好地挥了挥手：“拜拜。”
<br/>
<br/>        然后，她转过了身去，只身往出口走去。
<br/>
<br/>        郎安一直站在原地，目不转睛地盯着连星儿的背影。
<br/>
<br/>        当他盯着她盯出神时，背后突然传来一阵刺痛感，“啪”地一声鞭响，他痛得单膝跪地。
<br/>
<br/>        “郎安，我警告你，你要是还敢伤害星儿，我会杀了你！”连中凯不知何时出现在了他的身后，手里拿着铁鞭，呵斥道。
<br/>
<br/>        郎安眉头也不皱一下，甚至看都不看连中凯一眼，转身进了屋。
<br/>
<br/>        他背上有一道铁鞭留下的血痕，因为侵染了身上的衣服而留下格外深的一条印子。
<br/>
<br/>        合金门随之自动关上。
<br/>
<br/>        连中凯气呼呼地收起铁鞭，转身离去。
<br/>
<br/>        连星儿离开实验室的大门后，走进了那片整整齐齐的人工树林。
<br/>
<br/>        这片树林是为了掩护实验室的存在而种植的，所以，人走在里面不会觉得阴森，只是觉得这一排排方方正正的参天大树，像军队的列阵。
<br/>
<br/>        只可惜今天天公不作美，连星儿刚走出树林，天空突然毫无征兆地下起了瓢泼大雨。
<br/>
<br/>        当她正举起手提包准备用来挡雨时，却意外地发现，大雨并未对她兜头而下。
<br/>
<br/>        连星儿惊愕地放下手提包，抬起头来，只见一把黑色的雨伞撑在了她的头顶上方。

　　回到基地，天色已经不早了，回到宿舍后，苏云已经把饭做好了，今天除了白米饭居然还有两道菜。
<br/>　　一道是凉拌皮蛋，一道是凉拌西红柿。
<br/>　　都是凉拌菜，做起了简单，也不需要烹饪，直接切好了就行。
<br/>　　“哇塞，这么丰盛。”
<br/>　　林峰打开屋，一群人都在等着他。
<br/>　　就好像林峰是一家之主一样，主人不会来，其余人不敢吃饭。
<br/>　　“林峰你可算回来了，我看着这些菜看的足足咽了半个小时的口水，肚子都饿的呱呱叫了，可是苏云老师就是不让我们动筷子，要等你一起回。可苦逼了。”陈勾道。
<br/>　　见到这，林峰心中就有些感动了。
<br/>　　看着屋内一群人。
<br/>　　狗子，苏云，外加一个后来投奔他的小混混李广，李广虽然是个混混，但有句话说得好。
<br/>　　仗义每多屠狗辈，负心多是读书人。
<br/>　　李广虽然出身低下了点，但人没的说，够义气，和狗子走得很近，当然可能也是因为林峰实力让他顾忌这一部分原因在其中，但不管怎么说，这群人都是他的朋友，也是他林峰要守护的人。
<br/>　　“我去，怎么忽然这么多愁善感了。”
<br/>　　林峰摇了摇头，“大家以后有饭就先吃，不用等着我，搞得我想是家里的大姥爷一样。另外这几个罐头大家也分了，都是牛肉罐头，我好几天的存货。”
<br/>　　一群人吃着饭相互闲聊着。
<br/>　　“对了，陈勾，你表哥呢，这几天没看到他了。”
<br/>　　林峰吃了口牛肉，好奇道。
<br/>　　“他啊，想开店，这不，昨天晚上还找我，让我帮他和你说说，他不好意思和你开口。”
<br/>　　陈勾道。
<br/>　　“开店？什么店。”林峰道。
<br/>　　“不知道，反正那小子心思就是只想着赚灵石，对于什么提升实力没多大想法。”陈勾道。
<br/>　　“开店不错啊，我看着基地内很多店生意很好，这样吧，他要多少，我就算是大股东入股了。”林峰道。
<br/>　　吴三之前就是千万身家，这家伙不是富二代也不是拆二代，纯粹的白手起家，说明还是有点商业头脑的。
<br/>　　而且，吴三是陈勾表哥，陈勾是自己兄弟，那么吴三也就是自己人了。
<br/>　　“也要不了多少，也就大概五六百颗晶石吧，这几天我给他送过去一百颗了，妈的，几天下来的劳动成果都被那小子拿走了，那小子要是给我亏了，我非要暴揍他。”陈勾道。
<br/>　　“这么说也就差四五百颗了。问题不大。”
<br/>　　林峰点了点头，“不过要过几天，因为我现在要存晶石买一套房子。”
<br/>　　“买房子？”
<br/>　　陈勾李广苏云等人都是诧异的看着林峰。
<br/>　　“这个地方住的毕竟条件有限，你看苏云老师晒个内裤都要躲躲藏藏的，生怕被一群变态给拿走了。”林峰道。
<br/>　　苏云无奈的翻了个白眼，不过林峰说的还真的是实话。现在基地内变态真不少，甚至之前还有人把一头母狗给上了，简直各种毫无底线了。
<br/>　　“没想到末世前我买不起房子，现在倒还可以买得起房子了，还真是讽刺啊。”
<br/>　　林峰哈哈一笑。
<br/>　　一群人吃吃喝喝闲聊着。
<br/>　　……
<br/>　　翌日。
<br/>　　空投刷新。
<br/>　　五个D级空投，林峰全部捡着丢在了物品栏里。
<br/>　　然后就去了一个地方等候一个人。
<br/>　　没多久。
<br/>　　寡妇来了。
<br/>　　这是他和寡妇约定好的碰面地方。
<br/>　　每隔三天，寡妇来交底一些情报。
<br/>　　“主人，梅姐他派我去监督骷髅会去了，另外这几天，梅姐似乎和尸傀门走动频繁，尸傀门的人已经入驻曙光基地了，看样子是要着手调查龙哥之死和战门袭击的事情了。”
<br/>　　“骷髅会居然也是你们王家的？”林峰意外。
<br/>　　“是王家在俗世的势力。”寡妇点头道。
<br/>　　“这么说，那还真的是杀的一点都不冤，这么看来，我和这个王家还真是杠上了。”
<br/>　　林峰摸着下巴道。谁能知道，因为一个小小的吴倩倩，事态会演变到这种地步。
<br/>　　“另外，张家也在调查张长风之死的事情。”寡妇道。
<br/>　　林峰头疼，这些事情都是他林峰做的。这么说来，三大家族，张家和王家都和他有仇了，剩下的一个古家也说不定如此。
<br/>　　“特么的，我招谁惹谁了。”
<br/>　　林峰无语。
<br/>　　“对了，这张家背后是什么势力在给他们撑腰？”林峰问。
<br/>　　“是大刀门。”寡妇道。
<br/>　　“实力和尸傀门比如何。”林峰问。
<br/>　　“相差不多，据说这大刀门和尸傀门门主都是十阶的高手。再更多详细的我不知道了，他们非常神秘。”寡妇道。
<br/>　　“十阶？”
<br/>　　林峰一听咋舌，这基地内最强的也就是九阶。不愧是组织中人。
<br/>　　让寡妇回去后，林峰坐着思考对策。
<br/>　　虽然看起来吓人，好像林峰下一刻就要死了。
<br/>　　但实际上林峰现在无比的安全。
<br/>　　张家那边没有人知道是谁杀了张长风，王家那边没有人知道是谁杀了龙哥，甚至看寡妇那意思，王家已经完全把目标放在了战门身上，林峰这一招祸水东引算是完美的达到了效果。
<br/>　　“对，还有个战门，他们在我身上搞的这什么追魂索命蛊也是个隐型炸弹，妈的，最好你们杀的两败俱伤，狗咬狗一嘴毛。”
<br/>　　林峰想了想，目前他实际上还是很安全的，寡妇被小妲己控制了，丝毫不担心。
<br/>　　“对了，那个古家的骚货！好像叫什么古月的！她是当初在墓地那边见到过自己脸的！不过这个女的似乎她压根不知道我是谁，不然的话，我早就暴露了。不管怎么说，实力才是根本，有了实力，才能兵来将挡水来土掩。”
<br/>　　林峰正准备出去猎杀丧尸搞些晶石。
<br/>　　忽然他就看到一群人在他战士会宿舍区面前鬼鬼祟祟的。
<br/>　　看到林峰这群人顿时眼睛一亮。
<br/>　　“林峰！”
<br/>　　其中一人立即对他举手示意。
<br/>　　是昨天救下来的那个侦查大队的副队长朱民，其余的都是他的部下。
<br/>　　林峰走了过去，“怎么，任务完成了。”
<br/>　　“是啊，托你的福，林峰，这是送你的，一点小心意。昨天没有来得及好好感谢你。”
<br/>　　朱民道。
<br/>　　一袋子橘子。
<br/>　　这可已经是厚礼了。
<br/>　　林峰欣然接受。不要白不要。这玩意现在可值钱了，每天都是限量供应的，价格高不说，而且基本上要有门路才可以买得到。
<br/>　　“你们侦查小队这几天可有侦查到什么情况。”林峰问道。
<br/>　　“这个嘛……”
<br/>　　朱民有些不怎么好开口。
<br/>　　“哦，要是涉及什么机密的话，那就算了，是我唐突了。”林峰道。
<br/>　　“那个，机密的确是一点小机密，不过林峰你是我救命恩人，你既然要问的话，那我就告诉你。实际上情况不太好，我们在基地南边一个叫做白骨山的地方发现了大规模的丧尸群。”
<br/>　　朱民认真道。

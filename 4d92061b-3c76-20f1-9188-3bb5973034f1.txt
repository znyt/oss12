　　如今楚风眠单单血脉之力，便是媲美双龙之力。
<br/>　　而楚风眠的实力，远不止如此，他的剑术，他的冰火神体，无数奇遇，几乎让楚风眠的实力，远超双龙之力。
<br/>　　如今哪怕是在九劫古帝之中，都算得上强横，实力足以媲美两尊，三尊九劫古帝的凰公子，都被楚风眠一击轰杀。
<br/>　　虽然算是有着几分巧合，却也足以证明楚风眠的实力，在九劫古帝之中，楚风眠绝对是当之无愧的强者了。
<br/>　　尤其是现在，楚风眠才是八劫古帝的境界。
<br/>　　一旦是楚风眠踏入九劫古帝，那对楚风眠的实力来说，也是一个飞跃的提升，九劫古帝跟八劫古帝之间差距无比巨大。
<br/>　　一旦是踏入九劫古帝，实力将暴涨十倍。
<br/>　　不过想要冲击九劫古帝，也谈何容易，楚风眠资质逆天，身上奇遇无数，楚风眠想要冲击境界，几乎是同境界武者难度是数千倍，数万倍。
<br/>　　所需要的资源，也是万倍来计。
<br/>　　上一次楚风眠冲击帝尊境，几乎是将积攒下来的无数财富，全部消耗一空。
<br/>　　这一次，哪怕楚风眠是在这赤红大陆之中，已经是得到了不少的机缘了，更是杀了不少的九劫古帝，得到了他们的空戒。
<br/>　　但是楚风眠想要冲击九劫古帝，单单是需要的资源，这还远远不够。
<br/>　　而且楚风眠的实力，还依然没有达到八劫古帝的巅峰，八劫古帝巅峰都没有达到，更何谈冲击九劫古帝。
<br/>　　越是逆天之人，想要冲击更高的境界，便是难上加难，而且楚风眠降下的天劫，也要比起同境界武者，强横万倍。
<br/>　　楚风眠只有在做好充足的准备，底蕴积累到极致后，才会去选择冲击境界。
<br/>　　历史之中，越是逆天的天才，越是可能死在天劫之下。
<br/>　　楚风眠可不想要步那些人的后尘。
<br/>　　“冲击九劫古帝不行，但是我的实力，依然足以提升，尤其是肉身之力，还远远没有达到极致。”
<br/>　　楚风眠缓缓的站起身来，看向那古老金门。
<br/>　　他这一次来的真正目的，可不是对付转轮和尚等人，而是为了这金身佛尊的陵墓而来。
<br/>　　现在转轮和尚跟小耀王寒夜，都被楚风眠逼走，凰公子死在了楚风眠的手中。
<br/>　　这金身佛尊陵墓之中的无数宝藏，便是只属于楚风眠一个人的了。
<br/>　　轰隆！
<br/>　　楚风眠起身，手心之中凝聚灵力，化为了一道龙爪，轰击在这古老金门之上。
<br/>　　这一道古老金门，已经是刚刚就被楚风眠打出了无数裂痕，这一下子，随着楚风眠凝聚力量的轰击。
<br/>　　古老金门，当即破碎开来，在空中解体，古老金门背后的景象，当即是显现了出来，映入到了楚风眠的双瞳之中。
<br/>　　无数宝光，显现而出。
<br/>　　这些宝光，在刚刚古老金门还没有破碎的时候，都想要钻出来，现在古老金门破碎，这一道道宝光的真正模样，楚风眠也看的一清二楚。
<br/>　　其中最为耀眼的那一道宝光，则是一块拳头大小的金红色石头。
<br/>　　就是这一块金红色石头的宝光，最为耀眼。
<br/>　　“好，是藏金精！果然这藏金精，就藏在金身佛尊的陵墓之中！拳头大小的藏金精，完美金红，这品质完美，算是藏金精的极品了！已经是接近仙物了！”
<br/>　　楚风眠看到这一块金红色石头的一刻，顿时大喜，他也认得出来，这一块金红色石头，到底是何物。
<br/>　　藏金精。
<br/>　　以纯粹厚土之力所化的诸天奇物，比起厚土之精蕴含的厚土之力，要精纯千万倍。
<br/>　　这样一小块拳头大小的藏金精，蕴含的厚土之力，比起厚土之精所化的一块大陆，甚至都要多。
<br/>　　楚风眠曾经得到过一整条山脉的厚土之境，可跟这一块藏金精相比的话，甚至不足其百分之一。
<br/>　　从这一点，就足以看得出这藏金精其中到底是蕴含这多么庞大的厚土之力。
<br/>　　而且这藏金精的珍贵之处，不止是其中蕴含庞大的厚土之力，而是其中厚土之力的精粹。
<br/>　　像是厚土之精，其实在九劫古帝之中，虽然算是宝物，可是如果想要花费大价钱的话，想要买到大量的厚土之精，都不成问题。
<br/>　　哪怕是比起这一块藏金精之中蕴含的厚土之力还要雄厚的厚土之精，楚风眠花费灵石，也可以买的来。
<br/>　　但是这藏金精，却是无价之宝，因为这其中灵力的精纯程度，不是人力可以凝聚而出的。
<br/>　　藏金精，只有在天地开辟，混沌之中，才有可能诞生，后天之中，哪怕是极端蕴含厚土之力的一些域外之地，都无法诞生出藏金精来。
<br/>　　更何况也如今楚风眠手心之中的这一枚藏金精，更是藏金精之中的极品。
<br/>　　这种极品藏金精，是凝炼肉身极致，冲击不朽真身的必备之物，对无数半步天人境的强者，甚至是天人境的强者，都是梦寐以求之物。
<br/>　　这样一块藏金精，一旦是炼化，足以是让我的肉身，可以至少增强五成！
<br/>　　楚风眠感受着藏金精其中蕴含的力量，不由的感叹道，不过楚风眠却是没有着急炼化这藏金精。
<br/>　　而是将目光，继续落在了这古老金门背后的石室之中。
<br/>　　这石室的中心，是一座石棺，不过楚风眠一眼看去，却是足以透过这石棺，发现这石棺之中，空无一人。
<br/>　　没有尸骸，只有一件古老的袈裟，这古老袈裟之上，被一位绝代强者，留下了禁制。
<br/>　　虽然经历了无数岁月，但是这一件古老袈裟，却是没有腐朽。
<br/>　　“金身佛尊的骸骨，不在其中，看来多半是可能金身佛尊，是陨落在外了。”
<br/>　　楚风眠看到这一幕开口说道。
<br/>　　在一些古籍之中，有着对金身佛尊的记载，但是也是只有寥寥数笔，至于金身佛尊的陨落，更是没有任何的记载。
<br/>　　只是眼前。
<br/>　　既然是金身佛尊的骸骨，并不在这陵墓之中，那多半是金身佛尊，也是陨落在外了。
<br/>　　后来净土宗，也没有能够寻回这金身佛尊的骸骨，只留下了这样一道衣冠冢。

        &lt;h3&gt;第三卷第八十八章  锐利的李凌菲&lt;/h3&gt;
<br/>
<br/>        西城九曦的项目开标了。
<br/>
<br/>        陈子霖和呼延寰宇一起参加了这次项目的开标，陈子霖带着呼延寰宇和李凌菲一起到达招标现场，呼延寰宇负责讲解，每个投标单位有十分钟的技术问答。
<br/>
<br/>        陈子霖是公司法人，呼延寰宇负责技术问答。
<br/>
<br/>        李凌菲负责商务，虽然在招标现场可能不会涉及到李凌菲的工作，但是李凌菲之前就是集采人员，对流程和政策把控得很到位。
<br/>
<br/>        令陈子霖没有想到的是，他在招标现场碰上了夏颜，夏颜作为专家会参加评审。
<br/>
<br/>        呼延寰宇并不知道陈子霖和夏颜有过节，他只知道李凌菲和夏颜有过不愉快。
<br/>
<br/>        当然，夏颜也是看见了陈子霖，这么一段时间，她夏颜平平安安，无风无浪，事业上也是百尺竿头更进一步，在圈子里已经是一个专家了，陈子霖能耐她何？
<br/>
<br/>        陈子昂和陈子霖再怎么厉害还不是要犯在她的手中？
<br/>
<br/>        这次她的老同学也在这一次的投标中，原本她是不来参加这样的评标的，但是有同学参加就不一样了，不过，也没有人知道他们是同学关系。
<br/>
<br/>        李凌菲也真是没有出息，好好的工作不干，跑去给人家打工，打工一年能赚多少钱？他们单位的项目就是他老同学做的，老同学为了报答她，在西城江某国际社区给她买了一套130平米的房。
<br/>
<br/>        夏颜轻蔑的盯着陈子霖他们，至于别的厂商的投标情况，她一概没有听见，等到陈子霖他们了，夏颜立马就精神了，在整个10分钟的讲解中，夏颜懵圈的听着，似乎太深奥了，她好像听不懂。
<br/>
<br/>        夏颜很想找到呼延寰宇的破绽，但是凭着夏颜的水平又挑不出来任何问题。
<br/>
<br/>        等到专家质疑的时候，夏颜问呼延寰宇：“你们公司为什么要来三个人？难道是想质疑我们这里的专家吗？”
<br/>
<br/>        “这位专家您好，招标公告和标书上并没有说不能来三个人，况且来招标现场的也不是只有我们公司一家来了三个人，您这是对我们公司抱有偏见吗？”陈子霖也不是任人捏的柿子。
<br/>
<br/>        “你胡说，我并不认识你，我对你们有什么偏见？”
<br/>
<br/>        十几个专家对夏颜提出来的非专业问题感觉不解，前面几个也有好几家都是三四个人，她不问，怎么到了这一家公司她这样问？
<br/>
<br/>        “夏总，您和我认识，我们曾经在一起工作过，而且我们关系很好，各位老总，为了给大家公平公正的原则，我请求，夏总回避我们的这个项目评审。”
<br/>
<br/>        大家面面相觑，没有想到李凌菲会这样说，看来这家公司还不错啊。
<br/>
<br/>        夏颜也没有想到李凌菲会这样说，她一时不知道如何回答。
<br/>
<br/>        “夏总，您和我关系很好，为了保证评审的公正性，您还是退出评审，无论怎么样，人的潜意识都会有一点点倾向性，无论我们公司中不中标，都和您没有关系，在业界您拥有很高的威望，我不想破坏您的威望。”李凌菲诚恳的说。
<br/>
<br/>        下面的厂商听李凌菲如此说，都开始窃窃私语。
<br/>
<br/>        夏颜气得要死，她同学的方案还没有过，她就要退出了？
<br/>
<br/>        可是现在的这种状态，她好像不退出也不行了。
<br/>
<br/>        乔小东对夏颜说：“夏总，你看？”
<br/>
<br/>        夏颜沉吟不语，乔小东接着说：“基于公平公正的原则，我们可以启用备用专家。不过我们会听大家的意见。”
<br/>
<br/>        “那就换专家吧。”大家当然不希望自己的方案被有倾向性的专家看到，一分都会拉开差距。
<br/>
<br/>        “我退出吧。”夏颜无力的回答，站起来走出去了。
<br/>
<br/>        什么叫做，偷鸡不成蚀把米！夏颜深深的理解了几个字的含义，李凌菲还真的不可小觑呢。
<br/>
<br/>        李凌菲成功的解决了一个挡路的，对于他们来说，技术上的问题就不是问题，方案上的问题也不是问题。
<br/>
<br/>        一轮角逐下来，呼延寰宇信心满满。
<br/>
<br/>        在这个行业内，他的专业知识如果他说第二没有人敢说第一。
<br/>
<br/>        而且他们公司的业绩有口皆碑，商务分和价格分他们有可能会拿到满分，至于价格分，他们三个低头商讨了一下，合计了一个合适的价格上交上去，三个人说说笑笑的离开了。
<br/>
<br/>        “凌菲，看来今天你是来对了，要不然我们都不知道怎么对付夏颜。”陈子霖对李凌菲说。
<br/>
<br/>        “看着夏颜吃瘪的样子我真的感觉很爽，你说她前面那几家一点也不关心，等我们了就给我们找茬，你都已经不再那里工作了，她还那么小心眼。”呼延寰宇说。
<br/>
<br/>        “夏颜已经是这个行业的专家，后期肯定会碰到，据我对她的了解，后期肯定会给我们找事情的。”李凌菲叹了口气。
<br/>
<br/>        “陈总，真对不起，我给公司添麻烦了。”李凌菲以为夏颜是针对她的。
<br/>
<br/>        “没有的事情，你今天做得很好，以后再说以后的事情。”陈子霖说，夏颜不是针对李凌菲而是针对他陈子霖，或者说是针对他的妹妹陈子昂，只是呼延寰宇和李凌菲不知道。
<br/>
<br/>        陈子霖总是想不明白，夏颜已经交换了她妹妹的人生，为什么还要和他作对？难道她的心里就没有一点愧疚？陈子昂通过努力有了现在的一切，如果是换了另外一个人又会是什么样的状况呢？
<br/>
<br/>        想到夏颜，他就自然而然的想到了他的二嫂，冯睿。
<br/>
<br/>        冯睿，当初是怎么想的？
<br/>
<br/>        那天哥哥动手打了冯睿，他是看到了，他差一点就说出来冯睿对子昂所做的事情了，但是他还是忍住了，说出来，难过的是很多人，有二哥、子昂，还有大家。
<br/>
<br/>        虽然冯睿将手镯送人了，但是如果有可能说明情况说不定还能要回来，哪怕要回来，找到了姑姑再给人也行，但是将冯睿对子昂所做的事情说出去，怕是没有回旋的余地了。
<br/>
<br/>        想到夏颜那趾高气扬的样子，陈子霖心里的愤怒像长了草。
<br/>
<br/>        他也不能去责问冯睿，光凭夏颜的语言谁也不会承认这个事情，包括她夏颜。
<br/>
<br/>        事情过去了这么多年，当初的一切都烟消云散了，想要找到证据怕是难上加难吧？
<br/>
<br/>        无论怎么样，他都得先受着。
<br/>
<br/>        虽然现在看着他们的生意不错，但是他们陈家在西城真的是没有任何背景，以后还是低调点，夹着尾巴做人吧。
<br/>
<br/>        为了公司那几十口人，陈子霖有时候怕也得忍气吞声。

　　第17章
<br/>　　从来富贵险中求，这句话的确是真理。当这场危机度过后，等待蛤蟆寨的便不是富贵，而是——暴富。
<br/>　　一匹匹游散在山林中的马，都是移动的财富，徐神机高兴得胡子都快要竖起来。
<br/>　　徐灵儿会养马自然也会驯马，她口中发出一阵阵哨声，将群马赶在一起。秦蛮本身是猎人，对于马匹也不是外行，再有徐神机帮忙，很快便将山林中的战马聚成一个圈。
<br/>　　仔细数过一遍，除开跑散在山林中的，被乱石砸死的，围成一个圈的足有八十一匹。
<br/>　　这个数字对于穷鬼徐神机来说，瞬间便令他觉得人生虚幻起来。秦蛮也没见过什么世面，挠着脑袋呵呵傻笑：“这……这……真是……像做梦一样……”
<br/>　　徐神机，徐灵儿，秦蛮……总的来说，其实就是土包子。山寨如今唯一见过点世面的，大概就是林少羽。毕竟人家出场装备强，有枪有马有衣服全套蓝装。不过，纵然是林少羽看到这一幕，也只能勉强保持镇定。这就像在一个二代面前摆下八十一辆豪车，怎能不让人心潮澎拜，这直接就是一个豪车后宫呐。
<br/>　　当然这场战斗还留下好东西，比如盔甲，兵器，可蛤蟆寨人手太少，实在搬不动。只好取大舍小，先把最珍贵的马匹占有，其他的……也只能强忍肉疼。
<br/>　　“灵儿，把马赶好喽，咱们回山寨！”徐神机把最后一柄刀搁在马车上，如今这辆马车已经被他装满，放不下任何东西。
<br/>　　秦蛮手里提着两张铁弓，背上背着箭筒，憨憨的笑着。
<br/>　　“徐老头，这么多马，你骑得完么，借二爷一匹呗！”
<br/>　　一道戏谑的声音，从山林中缓缓行出一队人来，徐神机一惊，他们正是飞虎寨的高飞虎，高飞豹两兄弟。
<br/>　　“呦，二当家，罕见呐，您今天也起这么早！”
<br/>　　“老头，别废话！”飞豹不屑道。
<br/>　　高飞虎冷哼一声：“徐老头，你们蛤蟆寨可是发了财，你们吃肉竟让我们啃骨头，说吧，这批马怎么分！”
<br/>　　徐神机一激灵，目光环视一圈，只见从周围的山林中，走出一拨拨人。来的不仅仅是飞虎寨一家，杏花岭小白狼，狗头岭的千丈蛇，秃头山的熊大熊二……青牛山盘踞的山贼来了一多半，他们带的人都不多，大多都是二三十个，但也远胜蛤蟆寨的人数。
<br/>　　现在他们都盯着这八十几匹马，露出贪婪的目光。
<br/>　　“我们凭本事抢来的马，为啥要和你们分！”徐灵儿。
<br/>　　“灵儿，闭嘴。”徐神机立刻喝止自己女儿，然后一脸谄笑的冲众人道：“各位当家辛苦，大家今天都发财了吧。”
<br/>　　“没你发的财多。”高飞虎道：“你们蛤蟆寨占了大头，我们也就拣到些骨头渣。可这么多好马，你们蛤蟆寨吞得下么！”
<br/>　　今天只要来到这里的山贼，有一个算一个都抢到些东西，兵器，盔甲都能让手下喽啰的战斗力提升一个等级。可是，一和蛤蟆寨抢到的比，这就不算什么了。先不说这些战马，就是徐神机装的那一马车兵器盔甲，都能惹的人流口水。
<br/>　　“我看也别分了，大家凭本事抢吧，我要那匹白马。”高飞豹大叫一声，率先拍马冲向马群。
<br/>　　各家山贼也是蠢蠢欲动，谁也不想落下风。
<br/>　　嗖！
<br/>　　一支羽箭凭空而来，直接射中马的右眼，灰马哀鸣一声，噗通倒地，连带着上面的高飞豹也给摔了下来。
<br/>　　“谁，谁！”高飞豹站起来提着大刀乱叫。
<br/>　　秦蛮手中张着弓，又从身后箭筒抽出一支羽箭，搭在弓弦上，羽箭随着他目光，在群贼身上转。
<br/>　　此人好俊的箭法！
<br/>　　众贼心中都暗暗赞了一声，一时倒是将马勒住。
<br/>　　高飞虎冷笑一声：“看蛤蟆寨的意思，却是要和弟兄们过过手？”
<br/>　　秦蛮铁青着一张脸，不发一言，只是将弓箭瞄准高飞虎。
<br/>　　事实上，徐灵儿和徐神机都不能打，蛤蟆寨唯一的战斗力仅仅秦蛮一人。而他刚刚才和一百骑兵激战，此时力气并不剩多少。
<br/>　　双方僵持下来，高飞虎提着手中的大刀，没拿准要不要和秦蛮动手。
<br/>　　“这么大的货，蛤蟆寨难道想要独吞，分一杯羹给众兄弟也不肯么？”高飞虎握紧刀柄。
<br/>　　“高当家的意思，应该怎么分？”
<br/>　　正在这个时候，一道慵懒的声音响起，众山贼扭过头去，只见山谷之中缓缓走出来一匹瘦马，上面坐着一个年轻人，肩膀上扛着一只大斧。
<br/>　　“大当家！”徐神机三人齐声道。
<br/>　　众山贼窃窃私语，心道这就是当初蛤蟆寨老寨主的儿子，听说他是个酒囊饭袋，看今天的样子，不像呐。
<br/>　　程大雷扛着大斧来到众人包围圈的中央，高飞虎仔细观察，发现在这种情况下，这人竟然依旧保持着镇定。
<br/>　　“不如我出个主意？”程大雷目光环视一圈，道：“把你们都杀了，岂不是就不用分了。”
<br/>　　哗！
<br/>　　一石激起千层浪，程大雷的声音在人群中引起轩然大波。
<br/>　　“小子，你吃了熊心豹子胆嘛！”
<br/>　　“你爹活着的时候，也不敢和我们说这种话！”
<br/>　　“你他娘的才几个人，也敢放着狗屁。”
<br/>　　“别废话，杀他娘的！”
<br/>　　群情激奋，吵吵嚷嚷乱作一团。
<br/>　　“都住了吧。”程大雷忽然大喝一声，盖住所有人的声音，然后道：“我就问你们，够不够一千人？”
<br/>　　高飞虎皱起眉头：“程大家问这个做什么，我们有一千人又如何，没有一千又如何！”
<br/>　　“没有一千，我怕不够我杀。”程大雷淡淡道：“我已经杀了一千，还在乎再杀一千嘛？”
<br/>　　唰！
<br/>　　像是秋风扫过，人群雅雀无声，众人面面相觑，都从彼此眼中看到困惑。
<br/>　　是呐，今天蛤蟆寨不过四五个人，怎么杀得一千人落荒而逃。这件事，到现在大家都没有想透。
<br/>　　高飞虎目光闪烁，放轻声音道：“程当家暂息雷霆之怒，休发虎狼之威，还未请教今天的事情您是如何做到的？”
<br/>　　“哈哈哈哈！”
<br/>　　程大雷扛着斧头，放声大笑，笑得群贼心中都发毛了，心道这神经病究竟笑什么。
<br/>　　忽然，程大雷猛地止住笑声，口中爆喝：“我有神法，能请天雷！”
<br/>　　
<br/>(本章完)

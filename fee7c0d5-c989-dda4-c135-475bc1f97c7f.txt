　　“诸位道友请进！”
<br/>　　混沌之中，新生的世界中，女娲那淡淡的声音在众人耳边响起。
<br/>　　接下来，自然是一番庆祝，女娲讲道一百年。
<br/>　　三清、接引、准提、镇元子、帝俊等人都各有明悟，但并不多。毕竟这是到了他们这个境界，每前进一步就是天大的喜事了。想要通过听一次讲道便突飞猛进，那怎么可能？
<br/>　　最后，当众人问起女娲如何成圣？女娲却只是淡淡的回了一句：
<br/>　　“造化人族，功德成圣！”
<br/>　　“哼！女娲，你不就是成圣了吗，有什么了不起的。向你询问，你却如此搪塞，真是岂有此理？”
<br/>　　老子与通天通过元始的解说，却是知道女娲成圣经过，但其他众人不知道啊，他们此刻听到女娲如此回答，还以为女娲在随便敷衍他们。一个个不由得脸色变得阴沉无比，看了一眼女娲，便告辞离开。
<br/>　　当众大神通者离开后，妖帝帝俊、东皇太一恳求女娲灭杀巫族，女娲却摇头说道：
<br/>　　“天数已定，圣人不可轻出手，否则会遭天谴。”
<br/>　　帝俊、太一闻言，脸色一阵难看，冷哼一声，大袖一甩，便踏步离开了。
<br/>　　在他们离开后，女娲一脸担忧的望着远去的伏羲，心中却想着以后该如何帮助自己哥哥伏羲。却是在她成圣那一刻，看到了一些巫、妖两族未来的结局，心中对自己的哥哥担忧不已。
<br/>　　“不管怎么样，哥哥，我一定不会让你受伤的。”
<br/>　　……
<br/>　　不周山山谷之中，在女娲走后，初生的人族，智慧极低，他们目带茫然之色，打量着周围的环境。他们刚刚初生，什么么不知道，不知道自己应该如何去生存。
<br/>　　十几万人族，占了整个山谷，他们相互对视，不知道接下来会有什么命运等着他们。
<br/>　　“诸位，母亲创造了我们，并给我们起名为‘人’，今后，我们便是人族了。我人族初生，身体羸弱，更应该团结起来，共抗灾难，这样才能更好的生存下去！”
<br/>　　陡然，有一人举臂高呼，对着山谷中茫然的人族大声喊道。
<br/>　　“不错，他说得对，我们如今需要先去找一个安全的地方居住下来，再去寻找食物，之后再做其他打算。”立刻就又有人应声叫了出来。
<br/>　　这四人，都是女娲最先创造出来的四人。
<br/>　　“我看这山谷就不错，我们便在此处暂时居住下来，等以后找到更好的地方，再做打算，如何？”四人开始商量对策，其中为首一个人说道。
<br/>　　“可以。”
<br/>　　其余三人纷纷点头同意。
<br/>　　接下来，在这四人的带领组织下，人族选择这个山谷作为暂时居住地，定居下来。
<br/>　　之后，人族分为十个部落，在周围找到一些山谷，聚集而住，他们以捕猎和采摘野果为生，生存分外艰难。
<br/>　　日复一日，年复一年，不知不觉中，洪荒已经过去了百年时间。
<br/>　　人族通过自己的不断摸索进步，学会了许多生存方式。他们学会用一些简单的树枝、石头搭建简单的居所，他们学会用一些动物的皮毛保护身体，他们也学会一些简单的狩猎技巧，以树枝、兽骨做兵器……
<br/>　　这些都不是他们一开始就会的，而是经过长时间的摸索，牺牲了不少人，才学到的一些简单经验。他们虽然都已经开启了灵智，但好像他们的智慧极低，这还只是一些最简单的经验，稍微复杂一些的东西，他们直接学不会。
<br/>　　渐渐地，人族在慢慢壮大，部落变得更多了。有时天上会降下各种灾劫，如火灾，雷灾，水灾、洪灾等，人族在这些灾劫中死去不知多少人。但，谁让人族天生羸弱呢，这是他们必须经历的考验。只有经历过这些考验，人族才能变得更加坚强。
<br/>　　他们天生畏惧火焰，因为死在火灾之中的人太多了。但也有胆子大的人族，他们等火灾过后，会去探索降临火灾的地方。之后，他们捡到了许多烤熟的动物尸体。吃过后，他们发现烤熟的食物要比生的食物要好吃多。于是，他们开始在每次火灾后，收集火种。
<br/>　　这些火种，不仅能够烤熟食物，还能帮他们驱赶野兽，照亮取暖。但这火种却会随时熄灭，很不方便。于是，他们都在想办法，怎样才能自己生火。
<br/>　　这天，不周山下起了倾盆大雨，大雨瓢泼而下，伴随着大雨的还有大风，山谷中有好多五六人合抱的大树，都被连根拔起，在风雨中随处翻滚。
<br/>　　原本，人族居住的许多简单房屋，都随着这场大雨而毁灭。无奈之下，人族只好躲进一些山洞之中躲避这场大雨。
<br/>　　初生的人族，带着对上天的无穷的敬畏之情，他们躲在山洞深处，全身不住的颤抖，好像有什么危险在附近，身体会下意识感觉到危险。
<br/>　　这场雨下了三天三夜，带给人族无尽的苦难。多少人族在这场灾难之中丧失了生命。
<br/>　　雨过天晴后，他们走出山洞，发现山谷间到处都是东倒西歪的大树，还有被摔碎的果实布满了整个大地，树木下，还有大片的动物尸体，还有时不时传来几声哀鸣声。
<br/>　　几乎所有人都站在洞外，静静地看着眼前这一切，没有发出任何声音，眼中充满恐惧，那是对未知力量的恐惧。同时，也有少数人变得更加坚强了。他们的眼中，充满坚定。那是一种要与自然对抗的坚定精神。
<br/>　　人族开始将地上的动物尸体收了起来，重新搭建简单的屋子，准备开始新生活。
<br/>　　这天，一个人族部落首领出外探查周边情况，意外发现了一种树木，名叫燧木。在一个偶然的机会下，他看到一只鸟在不停地用嘴啄燧木，那燧木竟然发出火星。
<br/>　　那人见此，颇为好奇，好像是发现了什么不得了的东西。他连忙捡了一些燧木，之后放在干枯的草木上，不断用木头像那鸟啄木一般钻那燧木。
<br/>　　随着他不断的钻木，在失败多次后，那燧木竟然燃起火焰来。他十分惊喜，举起燃火的燧木，如同一个火炬，高高的，直指苍穹，口中大声喊叫起来：
<br/>　　“火！火！火！燧！燧！燧！——”
<br/>　　巨大的动静惊动了周围的其他人，其他人见此，纷纷跟着举手高呼起来：
<br/>　　“燧——燧——”
<br/>　　呼喊声此起彼伏，渐渐汇聚成一声，惊天动地，渐渐地，响彻整个人族，响彻整个不周山山脚。
<br/>　　与此同时，九天之上，毫无征兆的汇聚起一片金光灿灿的祥云，紧接着，一道粗大的金色光柱从天而降，直奔那手举火焰的男子。
<br/>　　璀璨耀眼的功德，刹那间，便惊动了洪荒无数大神通者，如此庞大的功德，自洪荒开辟以来，可不多见啊。
<br/>　　燧人钻木取火，开启洪荒万族文明进化！这不仅对人族有利，对于其他万族，一样有利。
<br/>　　这是洪荒第一把薪火，人都说薪火相传，这火可是承载着洪荒万族的文明开端。所以，它降下的功德也最多。
<br/>　　自此，人族告别茹毛饮血的生活，进入文明时代。
<br/>　　人族灵智自此大开，普遍人族智慧大增。
<br/>　　不久，又有人为了躲避妖兽的侵袭，在树上搭起房屋，这房屋可比之前的简单屋子好多了，既可以遮风挡雨，又可以躲避妖兽，人们欣喜无比，纷纷效仿。天道有感，再次将下功德。
<br/>　　人族常年****全身，天寒地冻时，不知有多少人死于寒冰中。有人以兽皮制衣，用来御寒。天道有感文明进步，一样降下功德。
<br/>　　自此，人族出现三大始祖，领导这初生的人族，开启了人族的文明之路，他们分别为燧人氏、有巢氏、缁衣氏。

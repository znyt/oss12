　　李承乾的脑袋被门弓子抽了么？当然没有！
<br/>　　他只是不想让自己手里有太多的钱而已，尽管到了现在这个时候他的太子之位已经稳了，等到老头子百年之后继位也没有问题了。
<br/>　　但行百里而半九十这句话却时时刻刻在提醒着他，事情并不会那么简单，老头子的虽然看上去已经对他没什么好办法了，可谁知道这小老头儿有没有什么后手？若是万一让老头子起了什么疑心，又或者觉得他这个儿子有点不靠谱，那特么可就悲剧了。
<br/>　　当然，还有另外一个问题就是不赚有数的钱！
<br/>　　毕竟电风扇也好，电灯也罢，这些东西眼下都不是普通百姓能够用的起的，也就是说就算能卖出去，销量也十分有限。
<br/>　　而且只能供应长安城城里的一个角落，再远的地方……不好意思，给钱也不卖，原因只有一个——没电！
<br/>　　而说到电，这个才是李承乾真正的来钱之路，举一个最简单的例子，后世的现代家用电器都不怎么贵吧？可是你再看看电费一个月要多少？一年又要多少？人活这一辈子还要多少？
<br/>　　而且不但是电费，水费又何偿一是如此？我们日常开的汽车又何偿不是如此？
<br/>　　所以说，不要被眼前的利益所蒙蔽了双眼，那种百十万贯就会封顶的买卖李承乾这个从后世过来的宅男就算再傻也不会稀罕，完全可以送给老头儿、老太太乐呵乐呵。
<br/>　　而真正赚钱的买卖，他才不会放手呢！
<br/>　　坐在东宫明亮的白炽灯泡下面，李承乾得意的想着。
<br/>　　不过话说回来，虽着那些电器越来越多，发电机功率不够的问题又摆在了他的眼前。
<br/>　　学过物理的朋友们都会知道一个常识，那就是能量守恒，一台发电量为100千瓦时的发电机绝对不可能为102千瓦时用电量的电器供电是谁都不法改变的真理。
<br/>　　所以如何提高发电机的功率又成他需要解决的大问题，而且供电距离也是下一步李承乾需要面对的第二个问题。
<br/>　　想要在电上赚钱，那就需要提高供电范围，而供电是有损耗的，所以后世才会有高压电这个名词出现。
<br/>　　提升电压的用来延长输电距离的升压变压器，将高压电转为可以普通电器能够使用的降压变压器，还有什么电流互感器，电压互感器，滤波器……，等等等等十几种与电相关的设备名称闪过李承乾的脑海。
<br/>　　而最重要的一个设备就是：电表！
<br/>　　想收钱没有电表怎么行？没有这东西你知道跟人家收多少钱？按天收？这不是扯么！
<br/>　　于是乎，李承乾又拉着小唐开始了新一轮的折腾。
<br/>　　……
<br/>　　而与此同时，大明宫的变化也引起了一些长安贵族的注意。
<br/>　　尤其是一些经常跑去找长孙皇后谈心的贵妇。
<br/>　　当这些妇人看到长孙皇后房间中那盏漂亮的不像话的吊灯之时，瞬间便被征服了。
<br/>　　可以不用再点那该死的蜡烛，可以不用再闻那乱七八糟的烟味，最重要的是这东西实在是太亮了，亮到耀眼的那种亮。
<br/>　　所以短短三天之后，长孙皇后便收到了数量几乎近万的灯泡订单，把个李二陛下乐的是眉开眼笑。
<br/>　　要知道，大唐的灯泡可不便宜，一个至少也要五百文，上万个灯泡那就是五千贯，去掉两千贯左右的成本，至少可以剩下三千贯。
<br/>　　而且这还只是刚刚开始，以后说不定还会有更多的订单。
<br/>　　什么？你说上万的灯泡用不了？
<br/>　　嘿嘿，不怕跟大家说，上万个灯泡还真算不上多。
<br/>　　毕竟眼下用这东西的都是贵族，书房要装十来个吧？卧房要装十来个吧？老婆孩子屋里要装百十个吧？而且为了不被别人瞧不起，仆役丫鬟的房间也要装几个吧？
<br/>　　连仆役的房间都装了灯，院子里也要装几个吧？
<br/>　　长安的贵族哪个不是家大业大？谁家的院子没有万把个平方？这么一算一家两三千个灯泡几乎还不怎么够用。
<br/>　　当然，这个时代的灯泡并不怎么结实，不可能让你用一辈子，所以你还要买一些备用。
<br/>　　这么一算，近万灯泡简直就是毛毛雨，如果放开了供应李二估计百把十万个灯一点问题都没有。
<br/>　　……
<br/>　　美，卖出了灯炮之后，老头子心情大好，借着八月十五大宴群臣。
<br/>　　不过，虽然名义是赏月，可是……可是四下里无数大灯泡点的明晃晃的，照的人眼都睁不开算怎么一回事儿？这种环境里一抬头就是满眼的光斑，别说是赏月，就是想分清哪个是月亮都不容易好吧。
<br/>　　但老李却不管那些，尽管被灯晃的睁不开眼睛，但却依旧十分高兴，听着一群大臣时不时打听着灯泡的价格，心中那个高兴就别提了。
<br/>　　……
<br/>　　“什么？你说什么？不行？为什么？”手头有了十万订单之后，老头子再将把李承乾叫到了书房，不过当他让某人去装灯的时候却遭到了拒绝。
<br/>　　“父皇，您先别急，听我说成么？”李承乾尴尬的搓着手，苦着脸说道。
<br/>　　“你说，今天如果你不给朕说出个子午寅卯来，休怪朕打折你的腿。”灯卖出去了，却没办法安装，这让老李觉得很没有面子。
<br/>　　“父皇，您也知道，电灯这东西是用电的对吧？”李承乾想了想，对老头子问道。
<br/>　　“这还用你说？”老头子瞪了李承乾一眼。
<br/>　　“可是父皇，咱电不够用啊，眼下能保证宫里用电就已经很不错了，根本不可能再给长安城供电了。”
<br/>　　“怎么能不够，这不是用着挺好的么。”老头子显然不怎么相信李承乾的话。
<br/>　　“父皇，儿臣给您举个例子吧。其实这发电机就像人一样，人喝多少水它就尿多少尿，可是这也是有极限的，你不能让人一次喝下两石水，然后尿出两石尿吧？”
<br/>　　恶心，真特么恶心，李二坚定的认为李承乾举的这个例子就是故意在恶心人。
<br/>　　不过对此老头子早就已经习惯了，听完他举的例子之后冷冷哼了一声说道：“那你就多叫几个人来喝，那不就尿出来来了！”

四目相对，陆寒霆的眸色有些冷冽幽然，安静看着人不动，让人心慌。
<br/>
<br/>        夏夕绾避开了他的目光，“今晚，谢谢你了。”
<br/>
<br/>        看着她躲闪的样子，陆寒霆勾唇，俊脸上有笑意，但笑意不达眼底，“除了谢谢，没什么要跟我说的了？”
<br/>
<br/>        夏夕绾轻咬了一下红唇。
<br/>
<br/>        这时陆寒霆抬手，修长的手指突然落在了她领口的衣扣上。
<br/>
<br/>        夏夕绾瞳仁一缩，迅速按住了他的大手，警惕道，“你想干什么？”
<br/>
<br/>        陆寒霆看着她，几分讥讽的嗤笑了一声，然后将她领口散落开的那两颗纽扣给扣上了，“你说我想干什么，跟你车震？”
<br/>
<br/>        夏夕绾每一次都说不过他，看得出来他现在心情很不好，整个人也变得咄咄逼人了起来，他不是不会羞辱一个人，他羞辱一个人起来简直是扒皮抽筋的。
<br/>
<br/>        夏夕绾被逼的节节后退，有些难堪，雪白的耳垂更落了一层薄红，“我想去医院看林婶。”
<br/>
<br/>        “恩，等一会儿就到。”
<br/>
<br/>        夏夕绾想起身，离开他的怀抱。
<br/>
<br/>        但是搂在她纤柔腰肢上的那条手臂如铁箍般强硬有力，不容她离开，她不安的动了两下，引来了男人不悦的蹙眉，他的手掌在她凹下去的腰窝上捏了一把。
<br/>
<br/>        夏夕绾疼的拧眉，不敢再乱动。
<br/>
<br/>        此时的商务豪车驶上了高速，外面璀璨的霓虹灯透过蹭亮的车窗流泻了进来，将陆寒霆英俊精致的五官镀的无比绚烂，好看的一塌糊涂。
<br/>
<br/>        他觉得心口闷闷的，喘不上气，他伸手扯了一下脖间的领带，“记得我跟你说了什么？”
<br/>
<br/>        什么？
<br/>
<br/>        他说如果有什么解决不了的问题打电话给他么？
<br/>
<br/>        夏夕绾拽着身上的衣衫，将衣衫拽成了一团褶皱，有些话必须说清楚了，她不想再玩暧昧，“陆先生，我记得你说过的话，我承认今天晚上我有些莽撞，如果你没有及时赶来的话，接下来发生的事情无法预料，但是，我不想什么事情都麻烦你，毕竟我们只是和平协议的关系。”
<br/>
<br/>        只是和平协议的关系…
<br/>
<br/>        陆寒霆眸色沉了下去，“你真是这样想的？”
<br/>
<br/>        夏夕绾点头，“恩。”
<br/>
<br/>        陆寒霆收回了自己的手臂，“没良心的东西，现在就从我的大腿上滚下去！”
<br/>
<br/>        夏夕绾没见过他发脾气的样子，但他现在已经很可怕了，夏夕绾连滚带爬的下去了，自己坐在了一边，尽量不去占地方。
<br/>
<br/>        陆寒霆见她如此乖顺的模样都要气笑了，他让她滚下去她就滚下去么，什么时候她这么听话了？
<br/>
<br/>        ……
<br/>
<br/>        两个人都没有再开口说话，半个小时后到达了医院。
<br/>
<br/>        林婶已经进了VIP病房，打了点滴，加了心跳监测仪，还有专业的看护照顾着了。
<br/>
<br/>        夏夕绾知道这些都是陆寒霆为她做的，他是一个思虑周全的男人，成熟而睿智，让人忍不住的心安，想要去依赖他。
<br/>
<br/>        林婶昏迷着，还没有醒，情况并不是太好。
<br/>
<br/>        夏夕绾坐在床边握住了林婶冰冷而年迈的手，心里十分的难过。
<br/>
<br/>        这时头顶响起了一道低沉磁性的嗓音，“先吃点东西。”
